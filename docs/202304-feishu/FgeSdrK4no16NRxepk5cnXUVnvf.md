# 二、我的困惑与思考

加入生财的第一天，我其实是没有焦虑的，根据@IDO老徐给的建议，制定了初步的学习策略：

![](img/50bdd995212cf489a64edb889af8f9a0.png)

但实在是低估了生财精华贴的深度和数量，刷了几十条我便感觉不对劲，星球的优质内容太多了，每一篇精华贴都是前辈们的倾囊相授，内容全面、细致又深入。我这么没头没脑地刷帖，还是像只无头苍蝇一样。此时我突然想起了《生财有术｜第七期新人指南》，当即决定还是老老实实的看完新人指南再说！

新人指南确实贴心，把新手探索生财的过程分成四个阶段，每个阶段又有相应的推荐阅读，但是在阅读“推荐阅读”的过程中，我踩了个大坑：

推荐阅读里的每篇帖子都让我受益匪浅，但每篇精华帖的结尾又“附赠”了新的推荐阅读。那些标题十分具有诱惑力，想必内容一定非常有用。作为一个新手，我很难抵住诱惑不去点开。就这样，我被层层“套娃”，巨大的内容焦虑再次向我袭来。

插播小建议：在阅读新人指南的时候，先快速阅读一遍一级页面的内容，对生财有个大致的了解。读完后，再阅读第二遍，此时可以点开推荐阅读的帖子进行阅读，每篇推荐贴读完就收手，回到一级页面阅读其他推荐贴。

当我意识到自己被套娃，我强迫自己停下来，开始思考我应该如何使用生财。同时在@IDO 老徐的社群里，我观察到，不止我一个人有这样的感受：

# 三、新手期生财高效探索指南

进入生财第一件事，读完新手指南，。建议阅读不要过分深入，仅做大概了解即可，以免迷失在精华帖里徒生焦虑。

对生财小白来说，最重要的事，是先搞明白副业赚钱的底层道理，快速提升认知。这篇贴子对我帮助很大，也推荐你阅读，重点推荐阅读新手篇和人物篇。

对小白来说，与其迷失在精华帖里，不如先了解生财有哪些项目可以做。

在航海实战里，往期的实战活动已经为小白们分门别类整理好。可以通过阅读对应的航海手册，找到自己喜欢的、适合自己的项目，再按图索骥，使用 生财有术 精华贴索引工具 2.0 搜索与自己问题相关的精华帖，更加高效的探索生财宝库。
