# 1.赚钱的本质是什么？

解读一下：在整个社会或市场内，存在许许多多的需求方，他们在不同平台不同场景下遇到了多种问题或产生了多样诉求，这时候如果我们能够找到他们，为他们提供解决问题的方案，无论这个方案是实体产品还是虚拟服务，用户愿意为这套解决方案买单，我们就能赚钱。

因此，里面存在几个非常重要的关键词，它们直接决定了我们赚钱的整体思路
