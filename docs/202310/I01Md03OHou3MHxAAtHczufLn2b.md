# 为什么要做社群表情包？

之前小马宋老师预测未来5年出现更多超级个体，个人IP化已形成趋势。

未来每个人都可能拥有自己的IP和社群，强烈建议每个自媒体人可以在自己的社群设计一套社群表情包。

千万不要小看表情包，它传递的是一种文化，就像是企业文化。

第一：它能够凝聚社群人心，当大家都自发地用社群表情包进行发言都时候，一定是对社群对IP产生了归属感和认同感。

第二：表情包可以在群内进行传播，也很容易破圈，带来更多的影响力。

##### 在网页的左侧选择【素材】-【形状】-【查看全部】-【方形和三角形】组成气泡对话框

![](img/b044a590fe6077468af498b5ee1bfb4f.png)

![](img/aa14429a1c6242f7a566dc3da1eb167a.png)

![](img/a94174004d474506b68ce070830cfeff.png)

##### 添加文字，建议是群内经常出现的，符合【高频+高需】

比如【老马生财陪伴群】高频出现的有以下几种情况：

新人入群的场景：便可以把“欢迎加入老马生财陪伴群”制作成表情包

老马日常分享的场景：觉得老马说的很有道理，便可以把“满满的赞，给老马举大旗”做成表情包

嘉宾日常分享的场景：觉得嘉宾说的对你太有帮助了，便可以把“感谢分享，真的很有收获”做成表情包

群主要给大家发红包/福利的场景：想提前预告，便可以把“准备，要发红包啦～”做成表情包

跟着老马赚到钱的时候：觉得老马的分享激励人心，跟着老马有钱赚，便可以把“跟着老马，天天爆单”做成表情包

我给老马设计的“十年之约，一起成长”便是使用字由点字简宋，字体在画布左上角可以进行替换，多种免费字体可以去自由选择

# 注意事项

### 表情包设计需要简约、言简意赅

*   文案尽量金句化、清晰传达含义

### 表情包设计需要考虑高频场景

*   提前设想，不同的场景下，用户可能会答什么话

*   将大概率会答的内容做成表情包

### 表情包设计需要考虑排版配色

*   考虑品牌色，如果你平时沿用某个色系，可以就某个色系作为底色，比如老马选择的是红色

*   配色尽量不超过3种颜色，以免花哨

*   排版上可以文字+插画素材结合、也可以是文字+IP头像结合，粥左罗老师便是后者

![](img/7025a4310d105d6f40c23247d14765d7.png)

### 表情包设计适当口语化、日常聊天的口吻

*   口语化可以使聊天的过程变得亲切、拉近大家的距离

### 表情包设计需要整体统一

*   配色统一；底色不要换多种

*   排版统一；比如左侧文字右侧素材

*   字数统一；尽量不超过10个字，不然会显得表情包非常赘余
