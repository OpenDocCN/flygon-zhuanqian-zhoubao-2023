# All in小红书6个月，扭亏为盈变现70w+

> 来源：[https://f6uz5wrpjh.feishu.cn/docx/V06XdxQBxoAJQ8x9bPxcOVQnnud](https://f6uz5wrpjh.feishu.cn/docx/V06XdxQBxoAJQ8x9bPxcOVQnnud)

大家好，我是牧歌。

今年3月份的时候，我放弃了视频号赛道（那几个月官方政策朝令夕改，对矩阵店铺不是很友好，容易成为平台发展的炮灰。）经过一番平台之间的优劣势对比，最终决定all in 小红书。

# 01 持续亏损，团队崩散

初入小红书，花了不少学费，报了很多课程，学了很多方法，然后信心满满出发了。不出意外，准备的太多，碰了一鼻子灰。

3月份和4月份，都在持续亏钱，新组建的小红书团队也失去了最初的热情，对于测赛道，测品，都开始应付了事。那段时间，能睡觉都是一件奢侈的事情，满脑子都是小红书，根本就睡不着，就算做梦都是小红书。

魔怔了一般，就憋着一口气再干，我就很确信，我不可能一直失败下去，只要我学的够多，测的够多，挖掘的够深，坚持的够久，就一定会成功的。

就这样一直在坚持，团队的人员在一直变更，这点令我很难受，老成员留下的摊子，新成员的培训，这些琐事占用了我一部分研究的时间，我不想因为这些琐事耽误进度，干脆把所有成员召集起来，给出了两个选择，一个提前领一个月工资，然后走人，另外一个，之前谈好的工资+提成模式作废，我多加1000元，拿固定工资。

后面招募成员的时候，我都按照这样的方法，只谈工资，不谈提成，全亏算我的，大赚也跟员工也没有关系，他们只需要做好自己的事情。团队的事情就这样解决了，虽然我觉得这样的团队可能不会太持久，但是在项目成功之前，我只能这样快速解决了。

# 02 越努力越幸运

项目迎来转机，说起来可能都是运气。

一天晚上我在下载一个软件的时候，灵光一闪，应用商店里，那么多软件，开发者都是做了市场调研的啊，而且他们还有专业的团队，美工在运营。那我为什么不去搬运这些软件里的内容呢？越想越兴奋，仿佛挖掘到了宝藏，连夜开始挖掘测试。

## 开启2.0测试

网站知识整合，精准变现

app那么多，起号基本都是100%，获得的利润也很快让我的团队进入盈利。

但我比较喜欢继续挖掘，app是知识载体，那数量庞大的网站，又蕴含多少知识和机会呢？

我在业余的时间，会去看一些别的行业领域的故事或者分享。我不希望自己的思维被行业所束缚。而且很多偶尔看到的东西，某天可能就能给你带来一份意外的想法和财富。

我以前在星球看到过圈友分享的一篇文章，建立电脑的搜索和文件管理系统的。在我的2.0测试中，目前给我带来一笔可观的收入。

很多女孩子，对电脑问题，会束手无策。

我在网站上找了很多关于电脑会出现的问题，然后整理分享笔记，目前导流了3w+私域流量。

目前直接变现的业务就是：重装系统，故障指导还有新机设置，建立电脑搜索和文件管理系统。目前我单独拉了一个4人的小团队，负责这个业务，每天的利润都在500+。

这3w私域流量，后续还可以持续跨行业变现。

2.0测试中还挖掘到一个新的亮点：外国网站。

来自于国外网站的内容，“好奇心”自带流量。目前也在深入挖掘中。很多行业，都可以用同样的玩法，复制出成绩。

# 03 越分享越幸运

选定了一个正确的方向，就不要动摇。

坚定的走下去，熬过了黑夜，收获都是巨大的。目前我们还有29家店铺（之前37家，上次封号之后，8个没有解开），现在都进入流水线运营，7月的净利润17w+，8月净利润44w+.

当你的长线布局，都开花结果的时候，收益就会持续倍增。

这套走小路的起号变现方法，目前没有听到有人分享过，我们也还可以闷声赚一两年。但我想市场那么大，分享出来会更好，大家都进入这里面卷，3.0测试，4.0测试或许会更快出来，所获得的成果也会更多。

我的分享只是一个引子，从这个引子出发，找到自己的另外一条路，或许你会有更大的收入，就比如这两位生财圈友。我也不知道他们具体怎么做的，但他们肯定不是按我分享的操作照搬的，市场对于先知先觉者的回报是最为丰厚的。
