# 一、为给女儿看病，开启母婴零售

## 1、原生家庭打压，女儿早产确诊先天性甲减

我在农村离异家庭长大，爸爸因重男轻女，在我很小时就离家出轨，不知所踪。妈妈、奶奶都觉得因为我，爸爸才不回家，所以对我只有无止境的打压，日常威胁遗弃。

14岁那年，唯一疼我的爷爷去世，我走投无路想到自杀，从3楼一跃而下，结果却只是骨折。

当时，家人都把我当拖油瓶，也不愿意供我继续读书，就把我扔到救助站，在派出所的帮助下，我才找到爸爸。

当时为了能赚钱养活自己，我在暑假就打工赚钱，早6晚6十二个小时连班倒，干了2个月才攒够上大专的学费。

后来借着不服输的性格，我自考了本科学市场营销做销售，因为我知道，想赚大钱只有做销售才行。

在我看来，女人还是要活出个人价值，保持终身学习，赚足够多的钱，才能有自立的能力，才能不用看任何人眼色讨生活。

这样的日子，在婚后仍旧如此，婆婆家里条件好，总是看不起我，我当时只有一个想法，要赚更多，有钱才有脸。

这也是为什么我嫁人后衣食无忧，却仍不甘于现状，想要努力赚钱的原因。

正是这份不服输的倔强，我孕期也没有停止工作，劳累过度导致女儿早产。

原本3月9日的预产期，但在2月8日就因高位破水紧急住院，打了三天催产针转剖腹产。

![](img/4b859853fe454e2afba06c036d23ef6c.png)

但最揪心的不只是早产，在新生儿检查时，女儿被发现是先天性甲减，医生说可能会影响孩子的智力发育。

当时，初为人母的喜悦瞬间被击溃，因为我怀孕的时候就是甲减，一直有在按时服药，没想到还是遗传到了女儿身上，真的是又自责又心疼。

好在医生告诉我，后期可以做专业治疗，有康复的可能。也正是医生这句话，让我恨不得拼了命的赚钱，比之前的信念更坚定。

当时，我暗下决心，一定要努力，赚更多的钱，给我女儿一个健康的身体。

因为我知道，作为学历不高、0背景、0资源的普通人，能救我和女儿的只有我自己。

## 3、拓宽销路，意外开启社群团购大门

当时为了测品赚钱，我还专门找了当地一家知名品牌可心柔纸巾，试图带货，却意外收获了不贴标的边角料同款。

当时，我老公怕大家不认，但我想着纸巾是消耗品，量比较大，帮宝妈们省点钱总归是好的。更何况同城送货，不满意就退，试一下总没错。

就是这次尝试，帮我开启了社群团购的大门。

从2022年5月8号一直到5.20号，我们总共卖了1400箱，这也是我母婴行业的第一笔高收入，一次性挣了7000块。

![](img/ee97216a2aeef680595a74d8f60195a2.png)

当时厂里都没有货了，还在欠单，我老公一趟一趟的帮我拉货，挨家挨户送货上门，五县两区，都有家里小五菱的踪迹。

![](img/0a04779d089cfe5723af75c8ebf5645e.png)

【火羽白经验小结】——团购选品

在团购【选品】时，有这几个小技巧，给大家分享一下：

1、最好选择消耗品，这样出单率、复购率都会比较高。

2、在条件允许的情况下，价格才是最大的优势，所以推荐大家尽可能选择物美价廉的消耗品来带。

3、具体方法：选择同行卖的比较火爆的产品来带。

筛选标准：

1）去淘宝搜同款看看销量，优先选择销量大的产品。

2）再去咸鱼搜索同款产品，看看出二手的人有多少。

如果淘宝销量大，咸鱼出货少，则说明这款产品需求高，且质量过关；

如果淘宝销量大，咸鱼出货也多，说明它可能不太好用，大家买回家都出了，就不建议选。

本以为我会一直做团购，谁曾想女儿4个多月时的一次生病，让我再次燃起要赚更多的决心。

当时她连烧4天，挂完针在家洗澡时，突然小嘴发紫，身上有鱼鳞纹。我当时已经快吓死，赶紧喊我妈出来，直接打了120。

【火羽白经验小结】——引流裂变，及客户维护

2、维护老客户也很重要。逢年过节送惊喜好礼，推出各类活动锁客、留客等。

我采用的是充值储蓄会员卡，就能享受优惠价及惊喜礼品。这样一能帮大家养成在我这儿消费的好习惯，二也能不断积累信任度，拉进用户关系。

# 二、如何卖爆母婴产品？在母婴赛道掘金？经验分享

我将从流量裂变、科学选品、矩阵卖货、售前售后、拓宽货品5个方面，为大家分享详细的经验。

## 1、启动流量，3招搞定第一波流量，做好流量裂变

众所周知，做带货，流量一定是最重要的核心。我变现7位数的搞流量经验，可以简单总结为以下3点：

1）线上，混宝妈群

可以尽可能多加一些宝妈群，我常用的来源有美柚、妈妈网育儿等官方的母婴app群聊。

当然，也可以在日常带孩子去医院、游乐园等地方时，多与宝妈搭讪，互相邀请进宝妈群，进而获得流量。

这里需要提醒大家一点，我们在混宝妈群时，最好能够有一定话术，不要暴露太明显的“成交”目的，容易引起反感。

这里，我常用的话术有：

1.有没有离的近的姐妹加个好友以后一起溜娃呀？

2.本地有什么比较好的产康推荐吗？

3.****母婴店有团购活动，我们要不要一起凑单？

2）线下，小区遛娃搭讪

在线下，我们也可以去自己小区、周围的小区，在遛娃高峰期带上孩子去遛娃。

比如夏天，就可以选择傍晚的时候，去遛娃区。最近是冬天，就可以中午太阳大的时候去。

遇到同样遛娃的宝妈，可以随后搭讪，问问对方宝宝在吃的奶粉是什么牌子，在用的尿不湿是哪家的等。

利用宝宝的共同话题，深度交流沟通，进而借势加微信，以互换宝妈交流群的形式，混进新的群聊，也能邀请她进我们自己的社群。

3）线下，母婴店附近拉客

当然，可以的话，你也可以选择在母婴店停车场蹲守拉客，因为这样的客人更精准。

通常，我会以母婴平台官方做活动为由，用送礼品的方式，引导宝妈们添加微信，邀请进群，进而进行进一步转化。

这里也给大家分享几句我当时常用的一些话术：
