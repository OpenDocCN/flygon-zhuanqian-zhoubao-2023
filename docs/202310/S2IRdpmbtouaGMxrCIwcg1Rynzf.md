## 1，获取操作rpa步骤

第一步问gpt，提示词如下：

```
我是一个代码小白，完全零基础，也不会/想安装python等环境，但是我想使用爬虫爬取weixin.sogou.com的微信文章，我应该如何使用影刀rpa从https://weixin.sogou.com/weixin?ie=utf8&s_from=input&_sug_=y&_sug_type_=&type=2&query=aigc&w=01019900&sut=1028&sst0=1695214126073&lkt=1%2C1695214125965%2C1695214125965获取到标题，时间，来源等数据呢
```

## 2，根据步骤逐步操作

建议可以安装一个snipaste，可以直接把截图贴到屏幕上，这样我们可以看着步骤来

### 1，安装影刀rpa

### 2，打开影刀rpa

先注册，登录，进入主页面，创建新任务

![](img/2fd3cf961c2f57e8d00da8c0218fe0a3.png)

### 3，打开浏览器动作

一开始我就直接点击右上方的浏览器，后面发现没用之后，原来是在左边有一个添加指令

搜索打开，有个打开网页，直接拖到右边

![](img/13f995655bee2baac61ed835cc823e85.png)

然后让你设置参数，咋们把要爬取的链接放在这里，浏览器我就默认影刀浏览器了，因为其他浏览器好像有额外设置，大家根据自己需要

![](img/c6fa16c01df309a73fd81c5cbf5963d5.png)

### 4，设置等待时间

直接左边搜索等待，第一个拖过来，设置3s差不多就行，如果网页加载较慢，可以调整

![](img/c3233b107a309ec0cbb681eb199677aa.png)

![](img/37e71d13caf390936df08de61085dcb3.png)

### 5，获取元素内容

我在左边没有搜索到，后来发现在上面有个数据抓取，不过他会打开默认的浏览器

![](img/8ee61231a349c63af0fff27d2620168a.png)

我们先点击右边的运行，打开一个窗口，我们不关闭，直接切回来，再点击数据抓取默认窗口在右边，点击开始抓取，按住ctrl点击标题，发现它默认把内部元素都给选中了，真可谓意外之喜

![](img/0f4729c848125b3ad7555c49312b019b.png)

点击完成，但是我们要爬肯定是爬取多页啊，勾选爬取多页，再，勾选其下方的去元素库选择，捕获新元素，翻到最下面，按住ctrl，点击下一页，他就会识别到下一页按钮，然后设置一下爬取页数，设置一下数据保存到哪儿，该获取数流程就完成了

![](img/345f5342c1c5b4cc574021f4a8824f18.png)

![](img/1aa35ecdc827ae741ead986adc5dae1e.png)

记得拖动一下等待的位置，它默认弹了下去

### 6，保存数据

我一开始直接搜索保存，找到保存excel，然后拖到右边，发现缺少了一个excel对象，我寻思默认来一个呗，结果没有，它报错（左边有一个小叉叉），我于是点击了右上方的使用说明，发现需要先启动ecel

![](img/99163c9abb3255f1ae4f01b5f231d0eb.png)

![](img/3effbc9908734f0f43aed0b255068561.png)

![](img/327e9397a358744ed004171f8b457bdb.png)

搜索启动，有一个打开/新建excel，然后配置内容【新建excel，预览一个地址（我选择桌面），设置保存对象】

![](img/24e569d4c6a8fabe4529c65a1d948824.png)

![](img/f9e5eed6bf59f5ba1873620dd059b760.png)

![](img/1b6faa0cc0c3b2db4624ccb0d0a53c08.png)

这个时候再把保存excel拖过去，他默认设置咋们之前设定的内容，勾选保存后关闭

![](img/a136bea2747437594f0e48b4901b6632.png)

最后一步，将数据写入表格，搜索写入，配置行号1，列名A，写入内容选择我们再批量数据抓取的部分保存的变量

![](img/948990665e797d91ac955c9f64c6f529.png)

![](img/5492dee3e151d486f1a65cf578caaccd.png)

![](img/07f85c2e19223381a73f2d5caa8dca0c.png)

这样整个流程就完成了

![](img/c842a6596e31694ae7839ebd81e2fa07.png)

### 7，执行任务

这一步发现，保存excel是多余的

![](img/97b5f0a94551eac275d6fce1029de2df.png)
