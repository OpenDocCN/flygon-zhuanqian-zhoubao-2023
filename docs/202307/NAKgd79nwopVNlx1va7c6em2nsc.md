# 1、为什么需要合伙人？

现在我们个体创业者，虽然基于互联网创业的成本没有以前传统创业成本那么高，但是一个人要想把流量、转化、产品这商业的核心三个环节都打通，难度还是不小的。

都说超级个体，真正你去创业落地去执行的时候才发现，超级赛亚人也没办法一个人干那么多吧。所以不管什么样的项目，能不能赚到钱，最终还是拼执行力，执行力便是最好的竞争力。

一个企业，最有竞争力的并不是项目赛道，并不是产品，这些都是虚的。最有竞争力的便是团队，人才才是企业最核心的资源。

所以在我们创业过程中，如果能找到志同道合的合伙人一起做，如果合伙人又是能互补自己某方面的能力，那你做这事情成功的可能性便大大提升。

# 2、如何吸引合伙人？

首先，清晰的愿景和价值观，合伙人希望加入一个有明确愿景和积极的价值观的团队。确保你能够清楚地传达你的业务愿景、目标和核心价值观。这将帮助吸引与你的使命和价值观相契合的合伙人。

其次，要展示自己的专业知识和经验。在创业领域，合伙人通常会被吸引到那些展示出深厚专业知识和经验的创业者身上。通过分享你的专业知识、演讲或撰写相关文章，展示你在行业中的专业能力，这将增加你的吸引力。

再其次，就是共同成长的机会，创业合伙人通常会被吸引到能够一同成长的机会。强调你的创业项目具有成长潜力，以及合作伙伴在其中可以发挥重要作用。展示你的业务发展计划和增长预期，让潜在合作伙伴看到与你合作的机会。

然后就是利益了，提供公平的激励机制，为合伙人提供公平和有竞争力的激励机制是吸引合伙人的重要因素。这可以包括股权分配、利润分享、绩效奖励等。确保激励机制公正透明，并与合伙人的贡献和业绩相匹配。

再然后构建良好的工作文化，合伙人关心的不仅仅是工作内容，也关注工作环境和团队文化。建立积极、开放和合作的工作文化，以吸引那些与你的价值观相契合并且希望与积极团队合作的人。

最后就是创造成功案例和口碑，成功的案例和良好的口碑是吸引合伙人的有力证明。如果你已经有一些成功的项目或合作伙伴，将这些成果展示出来，这将增加你的信誉和吸引力。

总体来说，就是在价值观吻合、觉得我靠谱、觉得项目前景不错、觉得利益分配不错、觉得团队氛围不错（归属感）、然后也有对应的成功案例和口碑，就可以吸引到合适的合伙人。

其实这是一个很难的过程，合适且靠谱的合伙人，本质跟找老婆、找老公一样道理，可遇不可求。而且合伙人的选择，尤其在价值观、在人品态度方面跟你合拍是最重要的（否则后面内斗的风险会非常大）。再加上能力上也有一定要求，更是难上加上。

# 3、内容链接是最好招募方式

很多人说合伙人最好是身边认识的朋友、同学、或者同事，因为大家知根知底，对人品有保障，不至于后面搞些幺蛾子。

但我们大多数人，其实身边很难找到这样熟悉且合适的合伙人。主要自己的人脉有限，认识的人大多数都不是从事一个行业的人。

即使有熟悉、有知根知底的人，但是大家认知基本上不在一个维度层面，你跟他说自己的创业项目的时候，反正各种质疑、各种觉得不靠谱的，很难聊到一块，何况还要谈一起共事呢。

另外，合伙人的招募，千万不要从一些招聘网站，招聘渠道去招募。这些招聘网站，都是打工者求职的地方。你想招一个有主人翁意识的合伙人，就是典型的去错了地方。

所以面试结果下来，基本都是认知不足，聊不到一块，别人就是想找个能学习、能拿不错工资，且稳定靠谱、且能摸鱼上班的地方而已。

我是从今年2月份开始对外写文章，把文章发布一些内部创业社群圈子里，开始物色合适的合伙人。在一些内部圈子，这些人员质量、认知、价值观都普遍比较高，算是比较符合的人群。

然后在这些圈子，通过自己的一些干货文章内容来吸引志同道合的人，无疑是性价比最高的方式。同时也能跟自己的IP聚集人气和势能，可谓一举多得。

三四个月时间，我在圈内发了差不多十多篇文章，微信链接加了三四千人。期间遇到了很多交流者、合作者，让我在圈子里的人脉得到了极大补充。

比如我最近在弄AIGC的项目，然后上周五在内部圈子发了一篇《AIGC三剑客赚钱机会的实践与思考》，仅一篇文章就链接了很多做这方面的志同道合的人。

有些时候，你链接的人脉和机会、见的项目案例足够多的时候，你抗风险能力就足够强了，这时候你就很难不成功了。

# 4、几个痛点和解决办法

在微信链接的三四千人中，有合伙意向的，我先是微信和电话做基本了解和筛选，觉得合适的话，再安排见面进一步确认。

差不多见面聊了10个左右想要深度合作的合伙，筛选了2个线下一起办公，一个合作了差不多一个多月，另外一个差不多合作了三个多月。最后觉得不合适，就分道扬镳了。

所以对合伙人的招募也算是试错了许多，几个感受比较深的痛点问题和解决办法，列举如下：

痛点1：认知问题，都喜欢看确定的东西

其实我们选择一个项目，核心就两个因素，一个是这项目赛道前景如何，其次就是做这项目的负责人是不是靠谱。而且对于投资者来说，他们愿意投资一个项目，核心因素是看人，其次是再看项目。

但很多人没经历过创业，可能自身对项目、对人、对创业没有很深的了解。特别是一个项目还处于前期的时候，没粉丝、没影响力、没产品等，通过一些项目方向、计划、和创始人的背景等，让他们确定下场一起创业干，这对想要一起合伙的人来说太难了。

大多人喜欢确定的东西，比如你账号粉丝多少、你课程内容是什么等，都是希望通过这些具体的运营数据来确定。

不过话说回来了，我运营数据都能确定赚钱，我还要合伙人干啥？对一个确定的结果，找合伙人来分钱？这时候根本不需要合伙人，招打工的员工就行了。

创业的魅力和合伙人的价值就是，觉得一个不错的项目方向，我们一起把不确定性的问题都解决，让其结果确定，合伙人才能一起享受胜利的果实。

痛点2：抗风险能力

所以正如上面所说，创业是一个不确定、不稳定的过程，我们要凭借强大信念，不断去试错，不断去解决问题，让其最终结果确定。

这期间会面临收入不稳定、甚至没收入、甚至负债累累的情况，80%创业成功的人，都会经历这些不稳定、很高风险的过程。

但很多人对这个抗风险的能力不足，想追求自由、不想被上班的环境束缚，那你就得能承受创业跟你带来的压力和风险，高风险才有高收益，最终你才有资格在创业这条路上得到结果。

所以面对这些风险，合伙人也是需要具备强大的心理素质和抗风险能力。对我们年轻人来说，也就半年到一年时间就能知道这项目生死，从长远角度考虑，哪怕失败也没啥，大不了再回去上班。但对很多人来说，他们更加关注当下眼前的利益。

痛点3：自身问题

招募合伙人其实也是一个双向选择的过程，我也反思过，没能吸引更多志同道合的合伙人，也许是自己还不够牛，也许项目还不够好，加上项目当前还没拿到一个好的结果。

解决办法1：继续进行内容链接

正如前面所说，内容链接始终是性价比最好的招募方式。在内部圈子、在公众号、在自己课程内容里、在直播里分享自己的价值干货，分享自己的心得经验，利他便是利己，花香蝶自来，自然会链接到合适的人。

解决办法2：求人不如求己

其实每个企业都是面临招人难的痛点问题，这是一个普遍的常识问题。有时候招不到合适的人，招不到优秀的人，很多时候，也许是公司平台、是项目、是老板不够优秀。

一直期待通过招人去解决问题，还不如自己亲自下场一点点去改善，一点一点去进步，不断去积累自己的影响力，强者自然不缺跟随者。

其实等自己影响力和粉丝有一定基数，招合伙人、招助理，都不是跟他们发工资。他们想加人，甚至都是需要缴纳几万的门槛费（避免白嫖党一些问题）。

当有了确定性的结果，你想加入，门槛就很高了。基本现在做IP、做知识付费，招合伙人、招合作的，都是我说的这种模式玩法。

解决办法3：合作方式多样化

我现在招的合伙人，都是需要来杭州一起办公。对很多不在杭州这边的，还有一些想做些兼职副业的人来说也是一个门槛。

其实我之前通过线上合作的方式做过一些事情，虽然线上灵活性很高，但线上方式不好管理，最终很多事情就不可控。

主要还是自己在线上团队这块的组建经验不足吧，后面还是可以通过线上 + 合作 + 资源整合的方式，来拓展这块合作的多样性。以结果为目的，而不仅仅只是局限于线下合伙创业方式。

总之，路漫漫其修远兮，任重而道远。创业也是一个修行过程，希望我们创业者一起共勉、一起进步！
