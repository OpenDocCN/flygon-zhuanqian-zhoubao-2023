# 缘起

去年的时候，机缘巧合看到很多网红明星在放直播切片授权，比较具有代表性的是小杨哥这种，不少剪辑人员能够月入几十万。

当时就在思考能不能爬到这个事的上游，去做个渠道，靠分发这些授权去赚钱；

最开始没做之前觉得这事可能会比较困难，毕竟我就一个平民百姓，上哪找去？人家又凭啥给我？

秉着老子谁也不服的精神，真给我找到了，还是圈友帮忙链接的…

放授权的找不到剪辑人员，做剪辑的不知道哪去拿授权。

因为当时切片这个事才出来没多久，达人那边正好需要大量的剪辑/渠道去分发，也就没给我设置过高的门槛。所以回过头来看，这事其实就是个信息差

# 授权分发

最开始给我的只有孤零零一个链接，从那个链接申请的用户就算我这边渠道的，每天会有腾讯文档的表格，给我进行数据同步，那个谁谁谁申请了XXX的授权，需要对接跟进，帮忙授权。

我就把拿到的授权渠道做成申请链接，往社群和抖音发，帖子类似这种：

《今天xxx授权开放了，点下方免费申请》

有的明星切片授权我发的比较早，像广东夫妇、郝劭文、金星这些。有不少副业圈kol看到了，有的拍成短视频《做XXX切片很赚钱》，有的包装成培训项目《明星切片日入上万，只要998》，加上我的授权渠道还不收费，这个渠道链接链接很快就传开了，记得当时短短一个月后台申请了几千人…可把我惊呆了

![](img/40524c9ca57cc9e367c1d8c75edf8fe5.png)

## 爆发式增长后上自动化系统

当时也没有一个自动化系统，授权是很麻烦的，需要绑定MCN，扫码验证码这些流程很繁琐，而且每天有几百个人往我微信小号上加，就想着能不能用系统来自动化解决？

然后就用wetool搭建了一套自动化系统，批量自动通过好友后，有个简单问询，让人选ABC，用户必须回答“是”才进行下一步授权，后续就是自动化的授权拉群...

因为机器人都是秒回，这一下子效率就上去了，而且24销售都能授权，客户体验比同行领先了一个维度（“你看那个谁谁谁，半夜都给授权”）

# 凭什么我能赚钱

授权是免费的，但是不妨碍渠道有返佣的收益啊~

虽然剪辑人员拿商品佣金的50%我们只有5%，可是架不住人多，后台不少账号一个礼拜就能给你干出个百万GMV

![](img/ab51747435d3f380abcf0fbd6b3d5b7a.png)

一看半年过去了，后台渠道收益也不少了，因为业绩出色（说实话真没花多少精力），上个月还额外给了我几个渠道的名额；回顾了下这大半年，并不是自己有多牛逼，而是吃到了这些明星网红的红利。

切到了他们的一个痛点需求：

原来网红/明星卖货这个事，主要是靠直播赚钱，直播停了收入就停了

现在有了直播切片授权，下播了也有一堆人在帮忙卖

而每个明星网红刚开放切片授权的时候，他的粉丝是没有被切片洗过的，抖音平台对应明星的切片素材也是几乎没有的，这个时候如果去剪辑的话，素材不容易重复就容易跑量。因此是有不少人需要这个切片授权，去剪辑赚第一波钱的。

刚开放授权的时候， 不少网红明星会把佣金比例拉的很高，比如我当时发的这条风向标：

美妆达人「美希」切片开放授权了，高达85%分成
