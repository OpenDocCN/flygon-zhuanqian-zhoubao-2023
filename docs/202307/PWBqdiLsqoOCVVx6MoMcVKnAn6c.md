项目准备：

1、不分新老 千粉抖音号，可买可刷（渠道自找，刷千粉价格在100多一点）

2、账号实名需自己或家人涉及到提现问题

3、有一台闲着也是闲着的win电脑因为抖音直播伴侣不支持Mac，不过双系统或者虚拟机也可以。

# 开干！！！

1，在抖音刷到同类型直播间，进去看他是否流量还不错，还不错的直接右滑清屏截图，把他的素材图直接截取下来，上美图秀秀修改一下即可。

2、然后在微信搜“纯色背景图” 有很多做壁纸内公众号的文章，直接保存和你素材匹配的颜色背景。然后打开表格把你的保存的背景图放进去。

3、录音（编辑好直播间的话术在word中，带入你正在直播的感觉，读起来就行了“家人们…….” 大概录10-15分钟即可，要求口齿清晰，不要像读课文一样，要有点激情的感情在里面。直播时循环播放。

以下附上录音话术，录一次大概五分钟，然后调整一下顺序，在录一次，也可以自己加一些自己想说的话，先抄在创新，两段五分钟录音倒入剪映 凑个十来分钟，循环播放。就可以了

![](img/843bb9a51e68b2d466425aab666b06f4.png)

4、音频使用QQ音乐“杨戬擒悟空”循环播放

5、下载抖音直播伴侣登陆已有的千粉号。

6、添加素材窗口选择你的表格占满直播间当作背景

7、添加你制作的找不同图片 调整直播合适位置

8、打开直播伴侣右下角游戏推广，添加“全民来找茬”

9、开播——设置讲解卡片。这就搞定了

1-9操作步骤不提供截图，点一下就能看见的东西，希望不要问，问就是百度！！

注意：录音时情绪饱满一些，不要死乞白赖的像没吃饱。这关系账号流量停留问题。

使用的找不同素材，挑选别人直播间已经热门的，因为市场检验过的东西，就一定是没问题的。

录音一次大概可使用一周以上，找不同素材不要换太频繁，纯色背景图两三天一换，但是有大流量进来的素材就先不要换了，一直使用，直到不行再说。

新号开播有流量扶持，有的账号当天可起飞，有的需坚持3-5天，打开电脑使用的只不过是你的电费，成本极低。 开播稍微有点流量后就会有游戏厂商私信你，给你补贴，大概也有中介，但是也无妨了，多一块钱是一块。会给你开通游戏广告分成80%（抖音内是60%）线下补贴是0.8每个（私人转给你）平均下来推一个游戏下载量是一块多。
