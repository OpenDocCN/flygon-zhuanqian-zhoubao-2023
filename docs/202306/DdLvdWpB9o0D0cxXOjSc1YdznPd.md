# 数据查看与分析

打开数据文件，是这样子的：

![](img/bc9bea2804d0afb40b80c1830377f91b.png)

将无用字段1和字段2删除。

可以看到，采集到的数据有9.9w，带一个w，表示单位是万，但是有些是不带w的，这个w字母会影响我们进行排序，所以我们需要用一个公式，将带有w的数据中的w去掉，前面数字乘以10000，转换成直接可以计算的数字。

方法为：

在E列的第一行，填入如下公式：=IF(ISNUMBER(SEARCH("w",C1)),LEFT(C1,LEN(C1)-1)*10000,C1)

![](img/a1b1dcb7f6bd21dc510040bd79984b00.png)

填好后如下：

![](img/fffadfa8bb90540970b27fbaf43aa9a1.png)

可以看到9.9w被转换为了99000。

同理，在F列第一行填入如下公式：=IF(ISNUMBER(SEARCH("w",D1)),LEFT(D1,LEN(D1)-1)*10000,D1)

![](img/cfb7a0a6bd8917d0e2f3fe3363559111.png)

可以看到1107.4w被转换成了 11074000。这样，我们就可以用E列和F列进行数据分析了。

然后把E列和F列第一行，复制粘贴到所有2400条数据。

![](img/ff295a9d3c7552092ae7b4266f1e8a3e.png)

然后我们在I列，加一个公式，用每个品类的互动数量（F列）除以笔记数量（E列），得到一个比值：

![](img/a28883d8578bcc31360e13c8870584e5.png)

这个比值代表着该品类的竞争程度。一个品类笔记越多，等于做的人越多。互动数量越多，代表用户兴趣越大。

如果互动数量除以笔记数量的结果，越大，等同于做的人少，需求大。这个值越小说明竞争越激烈。

对I列数据按从大到小排序：

![](img/eb98be1f691b56b392e9db12a633e1fb.png)

可以看到婴儿油，课桌椅，悠悠球，这几个品类，在过去30天，是用户兴趣够大，但是笔记数量较少的。可以考虑这样的品类。

# 常见问题

## 导出时，进度条走到10.001%就不走了

我有2台电脑都出现这个问题。换一台电脑好了。可能是电脑中安装的初始软件的不兼容。遇到这个问题，换个电脑试试。
