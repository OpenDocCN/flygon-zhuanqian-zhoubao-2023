# 同城实体店，只需要反复拍三类视频：

人设视频、流量视频、客户服务评价视频。

为什么要拍三类视频？

人设视频，增加信任力。

流量视频，打开同城播放，迎合算法，打同城标签。

客户服务评价视频，让客户看到你是怎么服务的，客户评价。（服务专业，性价比高，效果好）

一般是四条客户服务评价视频+一条人设+一条流量视频。

这样循环下去发。

想要把线上内容平台（抖音，小红书）做好，你的内容产出效率一定要高，且持续。

「因为我们做生意，需要源源不断的流量，绝大部分情况下，不是一条视频，一篇好内容就能吃几个月，甚至几年的」
