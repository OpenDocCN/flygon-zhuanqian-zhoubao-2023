# MJ批量定制手机壳

手机壳也是一个极大的市场，就像老罗打趣说的“卖手机的不如卖手机壳赚钱”

之前的手机壳要么是私人来图定制，或者厂商自带的有趣图片

而AI绘画可以做到：有趣照片+私人定制

这里也许有一定的市场空间

①直接作为产品售卖，可以结合之前的头像壁纸号私人定制图片

②衍生服务产品，例如：高单价头像定制就送手机壳

③做设计师帮厂商外包手机壳图片设计，赚分成收益

④除了手机壳还有其他定制产品：如衣服，鼠标垫，帆布包，笔记本

# 小孩涂鸦定制
