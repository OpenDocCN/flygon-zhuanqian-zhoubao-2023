# 小白如何少交智商税，避免成为一颗韭菜

各位生财的圈友大家好，我是大闵，今年生财的新圈友，3月加入生财有术。3月18日链接到老瞿（生财有术最懂二手书的男人），还有一周正好3个月，在二手书行业小有成绩。

这次5月的航海有幸做二手书项目的答疑官，接触了一些人、解答了一些问题、做了一次分享，最重要的是，有了一些感悟：关于如何避免成为一颗悲催的韭菜的感悟。

很多人在分享完以后私下链接到我，说喜欢我的风格和方式，想跟我一起成长。在这里首先感谢大家的信任，大闵何德何能？成长这种事也不是我能负责的，太重了。

但为了不辜负大家的信任，我简单的把如何避免成为韭菜的心得分享给大家，虽然不能助你生财，希望对你不破财有帮助。（说话犀利实属个人风格，不针对任何人，良药苦口。）

# 韭菜的定义：

韭菜最早是金融圈的一个名词，指那些基层群众，也是不长记性的代名词。通常，韭菜会被其他所谓的专家忽悠买股票、理财等金融产品，并最终以赔钱为结局。

看！最重要的一个词，叫做不长记性。

这个词衍生到个人创业圈，指那些创业小白，在进阶的路上，交了很多无用的学费、培训费，最后没有什么收获，悲催的成为了一颗“韭菜”。

# 韭菜的特征：

韭菜其实有很多的特征，不管是人性的弱点，还是自身的不足。

通过当这次二手书航海的答疑官，我深刻的感悟到韭菜常见的一个特征：严重缺乏基本的阅读能力。为什么这么说？因为很多问题都是航海手册里明明白白的写了。这些人可能是那种买一辈子东西都没有读过产品说明书的人，是那种无论拿到什么都要问别人怎么用的人……怎么样？特别常见吧。

另外，韭菜们不学习，不研究，鼠目寸光，怨天尤人，具体如下：

1、一切都是外界的因素：抱怨社会环境不好、领导不好、同事不好、国际形势不好……总之，就不是自己不行。

2、无用的时间精力资源过剩：人会把自己的精力、资源、成本投入到对自己最有利的地方，韭菜也投入，但是韭菜投入到低级欲望的满足中：刷剧、刷视频、游戏、八卦等等，总之就是不读书。

3、大部分不想或者不能吃苦：总想着如何“捞偏门”、“一劳永逸”，不会静下心来刻苦钻研和克制欲望来积累资本。买彩票、赌博、考证、买课程（一般只看前两集）是韭菜们热衷的。

4、救世主思维：这个是最要命，就像前两天在生财看到的一篇帖子，大致意思是生财是割韭菜集中营，小白被割怪生财没有做好筛选，要求生财收这些项目主办人的保证金，对小白负责。这可能是我入生财以来看到的最无奈的一个帖子，同时也很心痛，对被割的小白也非常的同情，这也是我今天写这一篇的初衷，希望能够帮助更多的小白深度思考，摆脱韭菜的命运。

首先，对正确信息的筛选和辨别是一个人一生都要修炼的能力，做选择题比解题更重要。大到选择什么样的学校、去哪个城市发展、选择什么样的行业、选择什么样的人生伴侣……小到选择是不是每天睡懒觉、睡前选择读书还是刷剧打游戏，处处都是选择。

有时候我神游的时候总是做思维旅行，经常会想，要是真的有平行宇宙的存在，那我的每一个选择都会造就不同时空不同的我，哪一个是我最想成为的，哪一个是最优的或者说哪一个是当下我觉得最优的。

在做什么事之前不要想着“靠”别人，对自己要有责任意识：“只有我对我自己负责”，不要总想着“抱大腿”，这世界上能靠得住的人只有自己不是么？。

当然，我也不是说遇到什么问题都不要寻求别人帮助，我的意思是这个思维，不要把让别人帮忙当作首要思维，可以把别人当作你解决问题的辅助。

改变自己的思考方式，思考带来决策，决策带来行动，行动改变命运。

当然还有很多很多的特征，欢迎交流补充。

# 韭菜宿命的开始：

很多人之所以成为自媒体的韭菜，一个共同的原因就是：他们看到类似 “0基础宝妈一台手机，也能轻松月入过万”这样的标题就跃跃欲试。注意看以上几个词：0基础、宝妈、一台手机、轻松、月入过万。 仔细想想，没有一个词是符合常理和基本认知的，可是就是这样明明没有一点道理的词组成的一句话，却有了魔法，激起了你的贪婪和懒惰。

几乎所有的韭菜宿命开始的时候，或多或少的都伴随着人性的七宗罪：傲慢,嫉妒,暴怒,懒惰,贪婪,暴食,色欲。

而成为韭菜最多的两个因素，就是懒惰+贪婪。

大家有没有发现一个气人的定律：当你看好一个行业，看好一个项目，你交钱入场后，这个项目的市场就会出现极速下滑。听起来似乎没有什么逻辑支撑，你可以暂时不信，不过早早晚晚你会体会到这个定律的魔力，好像是专门针对你一样。

# 摆脱韭菜宿命应该掌握的能力：

个人觉得，摆脱韭菜的命运最重要的能力就是 学习钻研的能力和解决问题的能力。

几乎一切成功的项目，最终都可以溯源到学习能力上。几乎所有厉害的人，都是学习和钻研的高手，也是解决问题的高手。

1、韭菜拒绝学习，成功者善于学习。

入局一个项目前，认真研究，花心思学习，分析市场，分析成功案例和失败案例，列出所有的卡点（包括市场的卡点以及个人能力的卡点），能不能解决，解决的代价大不大，等等……

入局之后，放下杂念，踏踏实实心无杂念的去做，克服焦虑，小步快跑，强大的执行力去验证闭环。无论成没成，归纳总结复盘，修正自己。在我看来只有这样才能把自己的成功概率提升。

2、对自己的遭遇“讨个说法”

不是向别人讨个说法，是向自己讨个说法，对自己的遭遇能够解释的清楚。

上学的时候，班里有些面貌丑陋的女生，对没人追的解释是这样的：“我可不是那样的人！不像那些狐狸精……”虽然旁观者清，但她们自己却对自己的解释深信不疑。然而，这样的解释真的除了能让自己舒服以外，还有很多的副作用，例如，除了让自己不真实之外，甚至可能为了让这样扭曲的解释更合理，她不惜用各种手段去折磨、诬陷那些“狐狸精”，虽然那些“狐狸精”除了长得更漂亮之外一点错都没有……当然，更惨的，并且她们自己可能永远不知道的是，正因为她们实际上的扭曲，所以，她们将来找到另一半，也必然是扭曲的。

所以，小白被割了，声讨镰刀者、声讨平台，除了让自己好受一点，真的是没有其他的好处了。你是想要一个正确的解释还是想要一个让自己感觉舒服的解释？

正确的解释，会引发你下一步正确的选择和行动。让自己舒服却肯定不正确的解释，除了让你短暂舒服之外，必然只能带来各种“出人意料”的副作用……这些副作用包括但不限于：做任何判断的时候先想到风险而不是收益，有个机会摆在你面前的时候也会先往坏的方面想，最终变成一个充满怨念的人，你到底要哪一个解释？

有一位心理学家说：这世界没有什么坏人，只有好人、笨蛋和病人。在我看来，很多病根本就是笨出来的。

3、不为表象所动，探究表象之下的实质。

这个世界上人人都在学习！其实韭菜也在学习，即便是笨蛋，也在不断学习，只不过，学习和学习并不一定相同。由于学习方向、方法、方式导致最后学到的东西完全不同。

一个项目，“研究透价值之后决定是不是参与”，和“不管三七二十一先下手，之后发现不对再去研究价值”有天壤之别，最后的结果也是迥异至极。

这真是个很有意思的现象，因为聪明和愚蠢，竟然不是被天生因素所决定，决定它们的竟然只不过是“顺序”

甚至有的小白，比如我在闲鱼二手书航海中遇到的，有人连航海手册都没有仔细的看3遍以上，总结出自己的操作方法，就不管三七二十一的急于下场，没有证就去上架书籍，不看违禁书单去上书，航海刚过一天就给自己搞了几个违规。那没有证的人怎么办，很简单：等待时机，太过简单的答案往往被忽略。

4、积累自己的实力。

缺乏耐心其实是表象，本质是缺乏实力。

如果开始时“那必然被割的部分”，仅占所有资产的一小部分，那么你会“失去耐心”吗？会因此失去冷静吗？会焦灼不堪吗？会羞愤难当吗？都不会。

更可能的是，会很冷静，会无焦虑，会羞愧自己判断错了，但不会因为羞愧而做出更不合理的行为。

我也经常和身边想自己做个体的人说，年轻的时候，不要把钱花在消费上，多花在提升自己和学习上，消费上能省就省，学习上尽量大方一点，最好的投资就是投资自己的能力，当然要会筛选和甄别。

5、理智的冒险

想要摆脱韭菜的宿命，一个重要的观念就是摒弃“富贵险中求”的这种理念，这是一个信息高度透明的时代，人们有各种各样的渠道去验证信息的真伪，实在不行还可以在生财向大咖们提问，再结合自己收集到的信息做自己的判断，做最终的选择。

获得经验的最直接方法是通过自己的实践获得。然而，在冒险这件事儿上，一定要尽早学会观察他人的冒险实践，而不是只通过自己的实践。

另外有一点切记，尤其是学生，不要借钱去冒险。有一个观点是：稀缺会造成短视。你稀缺的时候，尤其是金钱上稀缺的时候，目光只能看到眼前的那点钱，所以才会有轻松月入过万这种割韭菜的标题乐此不疲的吸引你的注意力。

6、警惕高回报的陷阱

举一个简单的例子，一个人向你借100块钱说第二天还你110，另一个人向你借100块钱第二天还你200。假设这两个和你的熟络程度、关系等外在因素都一样的情况下，你会把钱借给谁？别着急回答，请思考一分钟。

我肯定不会借给第二个人，为什么？这种条件一般是赌徒才会提出的。

可是在做项目的时候，大家总是喜欢低投入高回报的项目，喜欢立竿见影的项目，恨不得今天做明天就能把学费挣回来的项目，偏偏这样的项目都是割韭菜的项目，在生活中你懂的这个道理，在挣钱这件事上却不懂。

这次二手书的航海很多人在航海开始的时候还没有出版物经营许可证，每天在群里焦虑，有的跃跃欲想着不做书我做点别的吧，于是到处去找除了航海本身以外的项目，这看看，那学学，没准儿还交了不少学费。

在群里分享的时候我告诉大家，把项目的收益时常拉长来看，生财的航海项目一定是生财认可的吧？教练也都是生财认可的吧？航海手册是各种取得成绩的老师们一遍遍的修正最终发布的吧？那为什么着急呢？既然是可以拿到结果的，把时间线拉长，不急于这不到一个月的时间一定要取得成绩，而是把手册研究明白，能办证的时候办证，能实操的时候操作。

人啊，其实不用和任何人比，和自己比就行了，别一圈下来自己一点长进都没有，别人厉害和你一点关系都没有。你可以有目标，可以有崇拜的大佬，但是大佬也不是天生的，是用努力、经验、时间等等积累出来的。

这里其实有一个底层的可以克服焦虑提升自信心的思维：have do be 和be do have。拿小孩子走路举例子：人不是天生下来就会走的，那是牛啊羊啊这样的动物。每一个小孩子学走路的时候他不会想着我要有一双好的舒适的鞋、我要先迈左脚还是右脚、更没有小孩会思考邻居家的那小子10个月就会走路了我12个月了还不会走呢我是不是天生残疾啊？等等…… 小孩子什么都不想，99.99%的孩子天生一定有一天会走路的，去走就好啦，然后走路就成了自然而然的事情了。

7、克服贪欲和懒惰

大家可以看看贪这个字它是怎么写的？今贝。什么意思呢？当下就想获得、付出一点就想得到。贪一定不得，因为你付出一点就能得到的一定是小东西甚至什么也得不到。

你看看市面上所有的智商税，是不是都是给贪心又懒惰的人准备的？对新事物懒得研究它的本质又想快速拿结果的，3天玩转抖音、5天教你短视频带货过万等等，听着就很荒唐吧？还有前几年那些金融项目等等。有油的地方都路滑，你去摔跟头那是必然的。还是那句话，任何事情，把时间线拉的长一点，看的远一点，看项目看的持久一点，大方向上考虑的长远一点。

怎么克服懒惰？

其实这个本身是一个反人性的东西，人性本来就是懒惰的。

但是从另一个方面来看，其实没有真正意义上的懒人，说白了还是动力不足，或者说看不到可以确定的结果。假如给你100w让你每天都跑步坚持一个月，每天跑完就给你3W，你看看能不能坚持下来。

我看过一个概念觉得很有意思，亲测也有效，分享给大家：

懒惰的本质是大脑对身体的失控，懒惰起源于大脑，源于大脑的无所作为。什么意思呢？

人类的大脑会消耗身体一半以上的能量，如果大脑不积极运转，贡献相应的价值，其他器官就会觉得不平衡，于是罢工，整个身体呈现出一种懒惰。

首先要做让大脑勤快起来。那些坐着重复工作的流水线上的工作人员，下班之后会感到非常疲惫，因为他们几乎不用脑，他们的身体属于被迫劳作。而那些每天睡六个小时的企业家之所以精力旺盛，正是因为他们顺应了谁消耗更多能量，谁就贡献更多价值的原则，极大的调动了大脑的作用，使得身体其他器官跟随大脑一起贡献价值。

其实大脑这个消耗能量最多的器官同样有它的惰性，那就是它更喜欢去运算自己熟悉的事情。

你发现打工没前途，想利用业余时间考个资格证，大脑说，我都这么辛苦了，你就让我舒服舒服吧，王者峡谷的地图很熟，咱们去那儿转转；你发现自己认知浅薄，想学一学哲学，大脑说，我都多累了，干点熟悉的吧，来点精酿，配上足球赛，我看得懂所有球队的打法……

大脑对于熟悉的追求高于一切，因为越熟悉，运算越简单，消耗的能量也就越低，这就是舒适区很难逃离的原因，因为大脑熟悉。

其实，现代人越来越不爱动脑，背后有一个更深刻的原因，那就是经过几亿年的自然选择，那些最爱动脑的人因为大脑发育过快，消耗能量过多，而身体无法提供那些能量，他们的基因已经消失在历史长河之中。

而我们这些依旧存在的人类，一定不是最喜欢用脑子人，所以才有了那个说法：人类的文明史，尤其是科技史，根本就是一部懒惰史，而所谓的进化也可以视为另一个层级的退化。

可以想象，未来人工智能的普及会让被迫进行体力劳动变得越来越少，而人类社会更多行业和岗位都会指向大脑的主动作用。

给大家的方法建议是，不要试图克服懒惰，更不要轻易用意志力逼迫自己，先让大脑运作起来，努力寻找一个能够和现实利益嫁接的领域。当大脑开始主动思考，主动创造价值的时候，你就会自然而然的离懒惰越来越远。

# 韭菜的进化：

从“听大多数人的话”到“参考少数人的意见”，最终“自己做决定”。

一开始，你是小白的时候，你什么都不懂，你听了很多大佬的分享，报了很多训练营，加了很多社群，最终你会发现，大多数的声音都很吵、听着也很累、很没有新意，翻来覆去的就是那些车轱辘话，一遍又一遍，走一波来一波。在我们电商行业有个很有意思的现象，大多数的讲师其实都是运营的门外汉，当厉害的操盘手已经出去讲课卖课的时候，这个行业基本已经在走下坡路了。

这里不得不再夸一夸生财的航海手册，对于小白来讲，交付已经足够了，真的足够了。不夸张的说，如果你的表达总结能力够好，有短视频的制作能力，每一个航海手册其实你都可以录成一套视频课程，内容秒杀割韭菜的。

随着你踩过的坑越来越多，见过的项目越来越多，你慢慢的能分辨谁是大佬、谁是镰刀。你也知道如何吸取精髓就行了。到了这个阶段，你应该已经做成了一些项目，不管在财力上还是认知上已经有了很多的积累，慢慢的拜托了韭菜的命运，虽然什么能做你不一定知道，但是什么不能做你大概率是能够判断的。

最后随着你的思考越来越深入，你发现少数人也越来越少了，你已经变成了那个少数人，你有了自己的方法论。

韭菜进化完成后是孤独的。

事实上在任何领域，想要把事情做到极致，都要学会独处。一定要善待自己的“孤独”

# 结语：
