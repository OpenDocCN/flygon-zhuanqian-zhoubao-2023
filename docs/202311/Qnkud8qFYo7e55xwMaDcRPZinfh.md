# 一，产品

为什么要先说产品？通常情况下，在做一个项目之前，只有先了解产品，才能做好流量。

几个争议比较大的我就不写进去了

### 预测

相对常见的术的种类：小六壬，大六壬，四柱八字，紫微斗数，梅花易数，奇门遁甲，六爻等

主要功能：根据你所提供之信息，分析预测未来的发展趋势，或基于某个具体的问题，给出相应的判断

卖点：趋利避害，情绪价值，意见参考，信息咨询，正向的心理安慰，在某些情况下，道长算出他未来可能会有非常严重的事情发生（岁运并临），但是不会直接跟他说，而是引导他往阳光的未来方向去说，给予其希望

提供此项交付的人群：道长，各类居士，盲派高手，家传人士......等

姻缘 事业 财运，子女缘，学业等

PS:绝大部分线上交付的都是预测，收费根据模型的不同而不同，99-2k起不等...

### 风水堪舆

这个一般就是阴阳宅堪舆，一般引流遇到的还是比较少，通常需要强背书的情况下，会比较适用这个产品。

强背书不一定要指你的出身强，风水师的专业能力出众也属于强背书。

风水布局之类的。单次一般收费在5-7位数不等。

### 符箓

这个就不展开讲了，小到手机壳贴，大到天师符，这些祈福类的。懂的都懂

### 代上香，点灯祈福

代上香，以及点灯还是相对正规的，因为很多香客会求一个心安。然后是需要跟道观强合作的情况下，可以做这个产品。一般收费情况是按周、月收费。

### 图片+口播（图1-3）

这个类型的直播模式，是用的手游直播，提前先制作好素材图片，然后用iPhone相册的幻灯片，循环播放+画外音口播的方式。

#### 1.模式

重点：素材需要足够吸引人，能够一眼命中观众的心理。

优点：起号飞快，三天就能起来，0成本

为什么能起来？：图文作为直播间的主要内容，观众更能一眼看到你想要表达的东西，更吸引眼球，以及可以更好的过滤掉我们不需要的客户，留下目标客户，以我们的账号举例子，初始切入点是姻缘，感情的用户基数足够大，绝大部分客户需要的也是问感情的居多，能够更好的兜住系统推的绝大部分流量。我们给抖音的用户产生了价值，所以你获得了流量。

这种方式也可以用在情感，等其它类目....

我们这种类型的直播号，用的都是企业员工号，找朋友帮忙实名刷脸的，有关于这一块不懂的，可以在生财里搜索相关操作方式。一个蓝V主体可以开50个员工号（前提是要先做任务到黄金等级），这里就不再过多论述

进线路径：Feed推荐--看到你直播间----停留----进粉丝群----卡片/微信号进私域

### 真人出境（图3，6）

真人直播的类型一般有四种，1.写名字 2.有一定功底的付费连麦直接解盘 3.颜值演员出境纯导流 4.A+B模式导流

这里以写名字举例（因为我们只做过写名字）

其实做直播做的久的同学，看一眼直播间，脑子里立马就能把他拆的干干净净了

场景：我们这边用的是桌子加镜子增加场景的纵深，其实随便找个角落简单布置一下就行

1.场景需要突出主题

比如说你的直播主题是猜你心里想的是什么？

那么你在直播间的视觉设计里面就需要突出你的主题

2.主播只需要做两件事

写字，随便写，公屏有什么写什么

引导，写几个字，引导一波进群解惑

ps：这种形式的转化率我们只做到10% 单粉7块钱，本质上是属于泛粉直播

做的好的同行大佬单粉能做到14块，他的量级是一天进线2000-3000

这种写字的形式，需要搭配精心设计的营销模式才能发挥出最大的效益。

后来意识到这个问题的时候，我们就放弃了这个模式

# 四，伪IP录播模式

即使你不是做国学的，也可以看看这个板块，我觉得他可以应用上很多的内容类目，而且我看星球里写这一块的文章也比较少，希望对你有帮助，这个是我们实打实去实践出来的，可行性还是很高的

事情的起因是在6月中旬，有一天我偶然刷到一个24小时录播的直播间，直播内容是情感+国学，当时觉得很有意思，于是就跟朋友在那里反推他的业务模型

数据库由短视频+录播素材组成

在经历了反推论证之后，我们开始了大量的数据调研

### 小结

录播可以应用到其它类目上面，他是可行的，不过要做好封号的觉悟

1.更新素材的速度保持在3天一换，可以减少封号的概率

2.演员的表现力决定了账号的流量层级

3.所有问题的发生，都要先做内部审视，99%的问题都是内部的细节没做到位

三思而后行，方向选对了以后就让时间去发挥作用，比如IP就需要长周期的去孵化，没有数据支撑的事情不要干
