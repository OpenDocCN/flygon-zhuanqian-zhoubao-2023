# 二、视频号实名和提现

视频号想要带货，我们先要区分视频号的橱窗和视频带货权限，视频号的视频带货（视频挂车）有些门槛（本文暂不探讨这块内容）。

刚开始带货我们先开启橱窗，然后通过各种引导方式，让顾客通过主页“商品橱窗”进行购买。

# 三、视频号换绑

听说视频号是可以交易的，是不是需要把对方的微信号一起买过来？微信的实名信息怎么办？提现又该怎么办？

注销视频号的时间太长，我还想在这个微信上用视频号，有什么办法吗？

……

我们可以这样理解，视频号对于微信是独立存在的，视频号是可以随意换到其他微信账号上的，前提是接受视频号的第二个微信上还没有注册视频号

假如我们还想用微信A 来运营视频号，但是微信A 的视频号a 被限流了，想要注销视频号a 需要等很久。这时候我们可以把视频号a 换绑到微信B 上，然后在微信A 上重新注册视频号即可

注意：先结清视频号a 的带货佣金，然后再操作，换绑中需要填写微信A 的密码，如果忘记密码，找回密码需要等待14天后才能继续操作……

接受视频号a 的微信B 必须没有注册视频号，先用微信B 关注视频号a ，然后即可进行换绑

换绑很快就可以完成，至此，我们在微信B 上就可以看到视频号a 了！

以上的例子可以举一反三，如果不是视频号要注销，只是视频号要调整和微信号的对应关系都可以使用，当然也包括了视频号交易的情况。

换绑后商家资质认证不变，如果交了100元橱窗保证金，100元也在（保证金提现还是到之前填写的商家资质认证账户下）。开通联盟账户不变，还是原来填写的信息。

# 四、橱窗号突破5个限制的方法

通常情况下，我们把视频号换绑到微信上，该微信都是实名的，这样我们一个SFZ只能供应5个视频号的实名，那有没有办法增加呢？

如果我们一个SFZ已经实名了5个微信号，想要更换微信A 的实名信息，通常是在微信的钱包里——支付设置——注销微信支付。然后在新的微信B上重新认证实名信息，那么始终只有5个微信有实名。注销微信A 的实名后，如下图状态

![](img/4fbd509eda51508823b38ffdd4a48638.png)

如果是这种状态下（未实名状态），我们换绑一个视频号到微信A 上，那是没办法进行橱窗上架的，在带货需验证微信支付实名信息这步会失败

![](img/4edf0ea6f3ed6e7a0752b6739c9eedfd.png)

这里我们来看解决办法！首先要了解一个概念，微信顶实名（冻结微信支付状态）

![](img/9299faeee24acd7241697cd8a3886b29.png)

如上图，它的钱包里还有钱，但是转账的时候提示需要实名认证，其实这个微信号的实名状态已经没了（实名被顶，半实名状态）

但是这种状态下是可以正常橱窗带货，并且提现到开通联盟账户

![](img/f60a9bf648f7be1b25b62812f596689e.png)

![](img/5a9829b1fcd7a9873cca032b596ef97d.png)

![](img/7f65212a8f2eef621c2d5d3a624655dd.png)

那么这种状态微信顶实名（冻结微信支付状态）如何实现呢？

首先需要在已经认证实名信息的微信号（通常是微信大号）上操作

钱包——帮助中心——实名问题——查询名下账户——前往查询——验证账户信息——可以看到名下绑卡账户——清除即可

![](img/cd09a8c836507e67fc6a856673a404e9.png)

![](img/c451225d92aaec8b6fa539ed3f2fb6bb.png)

![](img/04b441bdd6ea1a3eadd84a4d2afe158b.png)

![](img/7259124a9693b4e9d211e11dbf434cde.png)

![](img/3fe4887f8f0a51560b4d3d0be742b736.png)

清除掉的那个微信A ，它的实名状态就是微信顶实名（冻结微信支付状态）。注意：冻结微信支付状态之前，要在钱包里放几元钱，防止被直接注销微信支付。

如果微信A 上本来就有视频号，我们去试试橱窗上架和提现是正常的，不受影响的。如果微信A 上本来没有视频号，那我们把视频号换绑到微信A 上就可以正常带货和提现了！

那么理论上说清除微信B C D E 的实名信息也可以按以上操作进行，空出的微信实名也可以绑定在微信F G H I 上，这样就突破了5个橱窗号的限制。不过有朋友反馈，操作太多可能会引起风控，导致后面实名认证视频号的时候有限制。所以建议实名认证满5个视频号的时候再开始操作，操作的时候适可而止。
