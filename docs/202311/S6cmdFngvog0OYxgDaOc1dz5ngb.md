# 1、为自己编写产品说明书，首先要认清自己

用什么来撒网捕鱼，即我们靠什么来吸引人？本质就是产品内容，我们找对象的本质，就是把自己当成产品，营销给卖出去。

你如果没内容（矮胖丑、性格差、生活无聊），没料（收入不行、学历不行、家境不行等），对别人产生不了价值吸引，那谁对你感兴趣？

当然人无完人，每个人不光只有缺点，也有属于自己的优点。营销自己的本质就是找到自己的优点，适当的放大和宣传，让别人看到你的闪光点，并实现交换的一门艺术。

但往往最难营销的是自己，只是因为很多人不了解自己，因而进一步导致自己不知道找个什么样的，以及什么样的才适合自己。

所以，了解认清自己，为自己编写产品说明，并发现自己这个产品的优点，是我们找对象的第一步。

自己的产品功能说明书：

|  | 外在：身高、体重、年龄、高矮胖瘦、穿着打扮、气质等。 |
|  | 谈吐：风趣、温文儒雅、侃侃而谈、满嘴跑火车、油嘴滑舌，以及说话声音、说话表情、说话习惯等。 |
|  | 爱好：宠物、美食、电影、逛街、旅游等日常兴趣爱好，当然也有吸烟、喝酒、赌博不良嗜好等。 |
|  | 性格：如开朗、刚强、懦弱、粗暴、孤僻、自信、积极、幽默、温柔、可爱、体贴、脾气火爆、善解人意、换位思考等。 |
|  | 三观：对待事情的认知看法，生活观、爱情观、工作观等。每个人的家庭环境，教育经历等，导致每个人的经历千奇百怪，经历的不同是导致价值观不同的核心。中国人讲究门当户对，以及教育学历相差太高，都很大可能导致不合适。这里面本质原因都是价值观不同。 |
|  | 人品：正直，善良，勇敢，诚实，守信，宽容，守法，才智，勤劳等。 |
|  | 性方面：夫妻生活是否和谐，这也算一方面，很多婚姻因为这方面离婚等。 |
| 经济基础 | 物质：收入、存款、房产、车辆等。 |

找对象的底层逻辑，就是价值互换的双向选择，你在选择别人的同时，别人同样在选择你。所以你的产品有什么功能及优点，这是你选择别人的筹码，同时也是别人选择你的筹码。

认清自己很重要，很多人要么就是很自卑，感觉自己没什么优势。要么就是过度觉得自己很优秀，一定要找个很优秀的另外个TA，典型的存在普信男、普信女一大堆。

切勿眼高手低，这就是对自己有个错误的认知，没有把自己摆放在正确的位置，导致找对象的时候错误的预估，导致别人反感和看笑话等。

运营操盘手视角：

认清产品功能特点，确定能提供的差异化价值，然后通过标签化，进而打造IP，提高影响力。有了IP影响力，流量就有了信任背书，流量的粘性、转化就变得很容易了。

海王视角：

就更加简单粗暴，直接通过过度的美化包装自己的外在、内在、经济，建立高大上、高逼格的人设，形成对你的高价值吸引力，这同时也是一双无形的手在掌控着你。

而真正优秀的人大多都是谦虚的，低调的，是不屑于过度的美化包装自己，更加不屑于去做海王这种低级趣味、败坏人品的事情。

# 2、定位你自己这个产品的用户画像

用户画像可以使产品的服务对象更加聚焦，更加的专注。在行业里，我们经常看到这样一种现象：做一个产品，期望目标用户能涵盖所有人，男人女人、老人小孩、专家小白、文青屌丝...... 通常这样的产品会走向消亡，因为每一个产品都是为特定目标群的共同标准而服务的。

在对自己这个产品的功能、优点有足够的了解前提下，你就要想着适合你产品的用户群体的功能特点和优点。用户群体的特点和优点，同样也是指在外在条件、内在能力、经济基础上的功能特点和优点。

我们在为自己定位用户画像，为自己选择的过程，也是在为TA选择的过程。

大家在找对象的大多数情况下，你看的上的人，基本上都是优秀的，但往往大多都看不上你。这多半是因为你对你自己这个产品的用户画像定位不合理导致，所以科学的定位是我们找对象的基础，找对象的用户画像定位的以下几方面经验供大家参考。

## 2.1、旗鼓相当定位模型

从结婚对象角度来看，很多人谈对象分手、离异，80%的原因是因为不合适，10%是因为出轨、赌博等原则性问题，10%是因为不喜欢、不爱了。

喜欢是两个人在一起的基础，但基本上都是因为不合适而分开。所以在喜欢基础上，合适才是最重要。

所以一般情况下，大家都在自身的外在条件、内在能力、经济基础上寻找相匹配、相合适的对象，这是普遍靠谱的找对象模型。

而合适的关键因素，在于性格、三观等内在原因。当然每个人都不同，也有因为经济情况、外在条件因素而分开等。

所以寻找跟自身的外在条件、内在能力、经济基础上相匹配、相合适的对象，寻找旗鼓相当的是常见基本原则。中国人讲究门当户对，在一定程度上还是很有道理的。

但由于每个人的条件和经历不同，导致在找对象的过程中要求偏重也都不一样。从心理学角度，人往往越欠缺什么，往往就越想去找个什么样的弥补。

比如身高很矮的，想找个很高的。没钱的，想找个特别有钱的等。这种条件相差太高不匹配，很可能导致找不到。

但如果你还是坚持，想通过找到另外一半，来弥补自己某方面很缺失的地方。那就要你在某一方面有过人之处的优点。

比如你想找有钱的，除非你非常漂亮。你想找很高的，除非你特别有才等等。正好你的过人之处的优点，正好是对方特别需要的，你们之间才可能配对成功（核心就是前面所说的价值互换）。

## 2.2、向上/向下定位模型

在家庭的角色中，一般男人作为家里的顶梁柱，承担的能力和角色更多些。 而女人一般希望得到安全感和保护，一般作为一个成功男人背后的顶梁柱，互相依靠、互相支持。男主外女主内（并不是极端的让女人做家庭主妇），也是大家所倾向的。

所以男生在择偶选择，很多人选择条件比自己差1个层级的。反之女生在择偶选择，很多人选择条件比自己优秀1个层级阶级的。

一般男人好面子，总不能找个比自己更优秀的女人，感觉hold不住。而女人好安全感，找个比自己优秀的，能够得到保护，这也都是大众化普遍的择偶要求。

很多时候你追不到别人，可能是因为你们层次阶级差距太大。你看得上的人，往往都很优秀，也不只是你一个人在惦记。

当你追求不到的时候，更多时候要思考的是自己，只是自己的能力、自己的价值不够罢了，其实这是很简单的底层逻辑。

如果你想从众多人脱颖而出，唯有励精图治、磨砺自己，让自己变得更优秀，才能配得上那个TA，得到你想要的。

那这个层级阶级是怎么计算的呢，可以拿：

*   外在（外貌和身材打分）

*   家庭（比如官二代、富二代、学二代等，当然大多数都是普通家庭）

*   事业（收入层次、职业岗位、创业老板等）

*   学历（初中、高中、大专、本科、研究生、博士等）

以上这四个维度层次来考量。也可以拿无产阶级、小资产阶级、中产阶级、上层阶级来划分等。

## 2.3、不顾一切定位模型

当你们层次阶级相差很大时，可能因为学业经历，或者参加某活动，让你们跨越阶层认识，并且相恋走到一起。你们互相也喜欢爱着彼此（如果单相思，那就是癞哈蟆想吃天鹅肉了哈），你为了TA可以放弃所有，为了TA可以克服所有困难，为了TA忽视所有条件差距等。

一般这样的感情存在电视剧电影中，存在灰姑娘和白马王子的童话中。哪怕这样的感情存在现实，也一般存在在年轻的时候缘分偶遇。当面对婚姻现实过日子的时候，性格，三观，家庭等很多不合适现实问题，不得不让这段感情夭折了。

## 2.4、总结

运营操盘手视角：

产品适合国内国外用户？适合一线二线城市，还是三四线城市？适合什么年龄段？适合什么男用户还是女用户？什么职业？等等。

海王视角：

海王对自身猎物往往有着清晰的画像定位，一般傻白甜、单纯的都容易上钩。正所谓姜太公钓鱼愿者上钩嘛。

为什么很多人容易吸渣？海王不去伤别人，而偏偏来伤你？这是因为但凡家境、教育、工作等成长环境优秀的人，海王的这些花里胡哨的套路在他们看来都是小儿科，自动屏蔽过滤掉海王的骚扰了。

而你，人情世故欠缺，认知不足，又不懂得分辨人，自然被套路了。

# 3、缘分需要自己创造，为自己寻找适合的渠道

对自己这款产品有了用户画像定位后，接着就是要选择合适的渠道，把自己营销推广出去。很多人找不到对象，并不是对自己认知不够，客观原因是因为自己平时工作圈子窄，而且身边都是同性，遇不到单身、合适的异性去接触、去了解，就更别谈找对象了。

不是缘分没来，而是你认识的人有限，又何来缘分呢？找不到对象，更多时候不是因为要求高，而是你流量基数不够大，认识的人不够多而已。

所以我们选择靠谱的资源渠道，打开自己的交际圈，这是我们找对象很关键的一步。常见找对象的渠道：

## 3.1、亲朋好友介绍

通过朋友、同事介绍，并且有一定信任背书，是一个很好的方式。但一般老家介绍的，由于工作地域，异地的问题导致两人很难相处走到一起。

另外由于每个人圈子不同，单身的自己身边要不就是一些都已结婚的，要么都是同样单身的同性同事和朋友，还同样指望着你帮忙介绍。

运营操盘手视角：

亲朋好友介绍，对运营操盘手而言，就是裂变模式。但由于很多人由于圈子原因，认识的流量基数有限，一般裂变不起来。

海王视角：

而海王狐朋狗友一般很多，资源圈子也很多，平时相互推荐资源渠道，也是海王用来扩展自己鱼塘比较常用方式。从运营专业角度，这也叫流量互推/互换，在公众号运营中非常常见。

## 3.2、线下活动

去线下参加一些兴趣活动，比如宠物、车友会、狼人杀、飞盘活动、话剧等等，都是扩展社交圈，找对象很好的方式。

不过在线下面对陌生人，需要上前搭讪，积极表现自己，这对我们这些社恐，喜欢宅的人来说，有点难为情了。

运营操盘手视角：

线下活动这种渠道方式，在运营角度，就是属于地推模式了。去线下参加活动，就如同去宣传自己，跟自己发传单了。

海王视角：

海王一般游走酒吧、夜总会、KTV等夜场寻找着自己的猎物，通过喝酒的一套互动流程，制造氛围、关系快速进阶，对一夜情套路非常拿手。

## 3.3、相亲交友（推荐）

相亲分线下和线上渠道方式，比如珍爱、marryu一些线下相亲平台，一般三个月收取2万块左右红娘服务费，红娘跟你介绍相亲对象在线下面谈。

我之前在上海工作的时候，也曾是珍爱线下会员，线下红娘安排的，一般都帮你做过筛选，一般都是靠谱的，但合不合适就另外说了，基本上大多都是互相看不上吧。但由于线下见面的时间成本较高，后来由于工作忙等原因，我也没去过了。

找对象的渠道，我最推荐的就是相亲软件。随着互联网发展，线上的购物、社交、娱乐，成为年轻一代主要的生活方式。通过相亲交友软件去找对象，是如今非常主流的方式。

主要是每个人在平台上填写的资料、个人介绍、相册等，一般都能大概了解外在条件、内在能力、经济基础方面，通过在线沟通交流的方式，主要效率高，避免什么都不了解的情况下，盲目的见面导致做无用社交。

推荐的相亲软件如下，而这些相亲软件的渠道，就是我们的公域流量池：

*   珍爱：国内比较大的婚恋相亲平台了，主打线下门店相亲服务，APP软件的用户基数也比较大，平台用户运营质量还不错。

*   Marryu：也有线下门店相亲服务，主打APP线上视频相亲服务。APP用户基数比珍爱弱点，品牌算是后起之秀。

*   伊对：主打APP线上视频相亲服务，貌似没线下门店。门槛比珍爱和marryu低些（会员费要低很多），相比珍爱和Marryu，很多用户的资料不完善，用户质量稍微低些。

*   她说：品牌并不是主打婚恋相亲，但软件上大多都是找对象的。每天推荐21个异性，采用双向匹配，并且要回答女方设置的问题后，女方同意才匹配成功。整体有一定流程门槛，但同时保障了彼此互选的满意度质量。

*   青藤之恋：从小程序开始运营做起来的小众相亲软件，vip每天推荐15人，采用推荐双相匹配模式。整体用户质量、用户基数还凑合。

其他一些小众相亲交友软件，就不推荐了。值得注意的是，找对象是一个既花时间、又花成本的事情，需要我们去对应资源渠道营销推广自己。

如今酒好也怕巷子深，很多人很佛系的说找不到对象的原因，是因为缘分没到。你工作太忙，又不愿意花时间，又不愿意花成本，那活该你单身了。所以把以上六款软件都下载充值下开始用起来，迈出这找一步。

## 3.4、总结

运营操盘手视角：

根据用户画像，选择合适的流量渠道来引流。抖音（综合性流量大平台），小红书（偏向女性用户），知乎（偏向白领高质量用户），B站（偏向二次元年轻用户），快手（偏向三四线城市）等都是常见公域引流平台。当然也能通过地推、招商引资等线下活动进行公域引流。

海王视角：

海王游走在各种娱乐场所和各种社交APP上，手里的异性资源渠道也是很多，也是能通过线下 + 线上 + 流量互推等渠道撒网捕鱼。

# 4、我为自己代言，我为自己引流

有了资源渠道后，我们为自己代言，为自己引流。根据前期我们对自己认清了解，以及定位产品的用户画像，然后在对应平台里面大浪淘金筛选出符合自己匹配的人。

值得注意的是，我们不是跟海王一样真的要去批量撒网捕鱼，而是在这些资源渠道，在茫茫人海中寻找那个唯一。

大体要做的核心流程如下：

首先，我们为自己编写的产品说明书，关于外在条件、内在能力和经济基础方面的内容，要在各个渠道平台上的资料里体现，即要详细描述并写到这些相亲软件平台中。

这类相亲软件平台，在软件功能上，一般体现在个人资料、头像、相册，然后我们可以在个人独白介绍里尽量完善自己关于爱好、工作、性格、家庭、感情经历等内容，也可以进一步描述对另外一半的择偶要求和期待。

但很多人有误区，甚至不想填写个人资料，头像，相册，个人资料信息都不填，哪怕填了，也只是填些空白信息。主要两个原因：

误区一：觉得资料公开不安全，特别一些认证功能，担心自己隐私泄露

误区二：担心遇到同事和朋友等熟人不好意思

相亲软件上的资料信息，就是你自己这个产品的功能说明书。头像、相册，就是你自己这个产品的图片展示。资料认证（真人认证、学历认证、房子认证），就是你产品的3C认证。

找对象的底层逻辑就是价值互换，而找对象的过程就是营销推广自己。谁人愿意跟一个没有产品功能说明书、没有产品图片展示、没有安全、质量和靠谱认证的人接触？你这是为自己找对象添堵，更是为别人找对象添堵。而上面两个误区担心，也是多余的担心。

然后，我们可以利用平台的择偶条件，再结合推荐的机制，把自己营销推广出去，去筛选匹配合适的人。值得注意的是，这一步你会遇到大量形形色色的人，这需要你擦亮眼睛，具备较好的辨别能力：

*   有群发撒网鬼混的

*   有资料信息不真实的

*   有做广告引流的（推广产品的）

*   也有一些骗钱的骗子等

你需要从大量的繁杂信息中筛选出真诚靠谱、且觉得合适的人，这也是对你识人辨物能力的考验。这个筛选过程，通过在软件上通过对方的个人资料信息、认证信息，然后跟对方进行沟通交流，来判断对方的真实性、靠谱性，以及是否合适等。

运营操盘手视角：

运营操盘手，一般运用产品故事，产品价值等内容，去产出适合各个平台的图文、音视频等内容，然后通过设置资料包、礼品福利等诱饵在各个公域渠道平台引流捕鱼。

海王视角：

海王一般用软件的群发功能，预先用设置好一些自我介绍的话术，去群发撒网捕鱼。海王是最懂营销推广自己，所以海王的资料毋容置疑都是写的很完善的。营销的核心就是价值吸引，海王通过营造的高大上、高逼格的人设，迅速吸引你，进而通过一系列的SOP流程掌控你。

所以我奉劝那些为了追一个人，把自己当工具人任劳任怨，感动了天、感动了地，感动了你我TA，但就是追不到。因为一个人不会因为感动而在一起，只会因为喜欢而在一起，而喜欢的本质和底层逻辑就是价值吸引（宇宙万有引力法则）。

这时候你还不如把自己积极、努力上进等有价值的一面展现给TA，让TA看到你的有吸引力的地方。如果这时候TA还不喜欢你，那强扭的瓜不甜，那就换个目标，不要在一棵树上吊死。

往往得不到的是最好的，但得到了也就那样，有些时候并不是你有多喜欢，更多只是你的执念罢了。其实这就是心态，让自己的心态乐观积极，成为一个有魅力、有吸引的人，自然不愁找对象了。

# 5、微信是了解一个人很重要的方式

在适合的渠道，通过个人资料信息、认证信息，以及跟对方的沟通交流，筛选出觉得合适的候选人，然后加微信进一步交流沟通。

## 5.1、为什么要加微信？

有些人在相亲软件上聊两句后，直接跳过加微信这流程步骤，就说见面聊聊看看合不合适。这种没有提前了解清楚情况下，冒然见面，很可能导致见面尬聊、互相看不顺眼等情况，很大几率在做无用社交浪费时间。

再比如亲戚朋友介绍、红娘介绍等，事先两个人都没接触沟通过，就冒然见面，很大的情况就是见面尬聊，彼此也没啥感觉，都是互相看不对眼。结果也变成应付，及浪费彼此时间等。

从流量漏洞模型来看，提前沟通了解彼此个人信息，工作情况，兴趣爱好，性格，感情经历，家庭情况，未来期望和打算等等，这必要的沟通，就是为了精准流量（合适且对的人）筛选的必要条件和流程。

微信作为生活的一种方式，几乎每个人人手一个，是了解一个人的必要的一种方式。而且任何社交软件，都迈不过微信这座大山，都必然会加微信。这已经是用户习惯，已经是生活习惯。

所以加微信，从流量漏洞模型来看，为了增加相处了解的流程，淘汰掉不合适的人，筛选出对的人。从生活角度来看，加微信也是走进彼此生活，进一步了解彼此。

## 5.2、什么时候加微信合适？

值得注意的是，在相亲软件上彼此最好做些适当的沟通了解，觉得合适，再接下来添加微信进一步了解。很多群发及不靠谱的，没聊两句上来加微信，这种未免让人有点反感，甚至有些骚扰的感觉。

在相亲软件上，彼此基本信息了解，聊天感觉氛围觉得合适，然后这人感觉也是自己想找的类型，双方觉得合适，就互留微信。避免过早加微信，彼此微信上也不怎么聊，以及话不投机半句多的觉得不合适，又互删的浪费时间。

## 5.3、朋友圈是自我营销最好方式

前面说过，微信是生活的一种方式，成为了解一个人的一种方式。而通过微信怎么样了解一个人？朋友圈是非常重要的自我营销的方式，通过朋友圈能很好的侧面了解一个人的工作，事情，兴趣爱好等很多方面，是对一个人印象评价的很好途径。

要说私域转换，朋友圈是非常重要的途径。做过私域的应该都知道朋友圈展示，朋友圈营销的重要性。但很多人，几乎不怎么发朋友圈，也不爱发朋友圈，这相当于放弃对别人做自我介绍、自我展示的机会，自然就就错过价值吸引的重要方式。

当然，朋友圈照片、视频，很多信息也经过美化包装等，自己也要一定认知和眼光去分辨判断。

## 5.4、必要的语音和视频

文字信息有时候也感受不到彼此的情绪状态，通过语音和视频更加真实立体，是最能接近彼此真实情况的方式，也是对朋友圈、个人资料等很多信息真实性判断很有效的方式。

当然有些人可能由于习惯原因，不太好意思视频等，也想保留点神秘感，想留到直接见面等。但必要的语音和视频，其实也是为了进一步让见面的成功性增加。

## 5.5、总结

运营操盘手视角：

在个人IP营销、品牌营销中，私域运营几乎成为如今运营操盘手的必修课，除了朋友圈营销外，群发私聊、微信群、视频号、视频号直播等，都是微信私域非常好的营销方式。

海王视角：

朋友圈是海王最直接、最有效营造高大上、高逼格的人设方式，海王养鱼无非就是经常发发朋友圈，用高价值的内容来吸引着你。

而这些所谓高价值的内容，男人无非就是晒车、晒表、晒房、晒名牌，让人觉得你很有钱。女人无非就是秀自拍（美貌）、秀美腿、秀身材、秀生活气质，让人觉得你很美。

虽然从心理学角度来说，越是缺乏什么的，越是去晒、去秀的表现些什么。但往往这种直接晒、秀的方式，却是营销自己最直接、最有用、最有效的方式，百试不爽。

# 6、线下见面、相处最终确定TA

正所谓百聊不如一见，通过微信的了解（营销自己）的结果就是为了见面相处，进而确认TA（转换成交）。

如果在见面阶段，你发现对方现实跟网上反差特别大。很大程度上，是你在认知眼光上出了问题，即你在相亲软件上和微信上的了解判断过程中出了很大问题。

不要别人写什么你就信什么，不要朋友圈里面有啥你就觉得是什么，更多需要去认清这个人本来真实情况。但正所谓人心隔肚皮，两个人的相处是复杂的，但也是简单的。你为人处世有经验的话，你就很容易分辨一个人。

相反你没啥概念的话，很容易被花言巧语这些花里胡哨的哄骗的话，这就会导致你感情经历很容易走弯路。这是找对象最基本的认知。要不然你就一次又一次在错误的人身上浪费时间，要不然就一次又一次的感情失败。

虽然对方是通过开朗性格、儒雅的谈吐、高价值朋友圈等很多外在条件，有价值的内容吸引着你（这就相当于简历的面试的敲门砖本质一样）。

但这仅仅只是别人让你看到了TA想要让你看到的一面，你需要通过现实中实际的相处，通过TA的言行、表情、眼神，及实际相处过程中面临的一些事情具体的反应等很多细节地方来判断这人的内在真实情况。

对于没有经验的来说，你适当放慢脚步制造门槛，不要听对方怎么说的，而是要看别对方怎么做的。这是非常有效方式，能帮你过滤淘汰掉那些图谋不轨的人。

一般我们可以通过吃饭、看电影、参加一些活动、或者一同旅行等，这都是比较好的判断对方的方式（约会的目的就是为了进一步深入了解对方）。

前期一些矛盾和争吵点，我觉得这是两个人了解彼此很好的方式，彼此对待问题的想法（价值观）和方式（性格），这就直接能看出来彼此是否合适。

甚至可以故意制造些矛盾，也算是对彼此的考验。不过一些人还是比较介意前期相处过程中就出现问题，觉得以后在一起了会不会出现更多问题，所以这也是因人而异。

总归来说，遇到合适且对的人，其实彼此在线上的了解，及现实的相处都是融洽的，在对待事情及问题的处理默契、氛围等，基本上就能确认彼此。

运营操盘手视角：

通过朋友圈营造的人设和信任背书后，运营操盘手一般通过活动方式，然后在视频号直播间直接转换成交。

海王视角：

通过朋友圈营造的人设和高逼格价值展现后，海王见面也都有一套SOP流程。海王一般通过营造氛围、挑逗、肢体进阶、私密转场等流程搞定你，整体也都是一套约炮的流程。

所以适当控制节奏，不要感情来的快，去的也快，玩玩的对象也没多大意义。寻找一辈子的人，为自己负责，更是为他人负责吧。

# 7、相爱容易，相处难

正所谓相爱容易，相处难。大多数的分开，不是因为不够爱，不是因为不够喜欢。而是因为在相处过程中，发现性格三观，以及很多地方不合适，两个人在一起一直内耗着，累了、乏了也就分了。所以我们发现一切又回到了起点，为自己编写产品说明书，我们首先要认清自己。

很多时候，在错误的时候遇到了对的人，很多缘分错过了，是因为当初还是一个懵懂少年。感情也许经历过了，才会成熟，才会懂得珍惜，认清自己，懂自己，知道什么样的才是对的人，才是寻找另外一半的核心。这样在遇见时，才会懂得真正去抓住。

如果你遇到了，应该好好珍惜努力。一辈子很长，一份美满的爱情，需要双方共同经营与呵护，需要一辈子的坚持。
