# 7天跑通日利润100的小项目：闲鱼二手书

> 来源：[https://xawbk0544jn.feishu.cn/docx/YEmndx9UXowq0XxalM5cBP8tngd](https://xawbk0544jn.feishu.cn/docx/YEmndx9UXowq0XxalM5cBP8tngd)

大家好，我是“朔愽”，今天分享一下我是如何快速跑通二手书这个项目的。

一，为什么选择这个项目

其实生财最不缺的就是项目，你能在里面找到月入3000的小项目，也能找到年入千万的大项目，但是这些项目不一定都适合正在找项目的你。

就像俞敏洪当时做东方甄选，他为什么在那么多项目里找到直播带货这个赛道，其实就是当时正好助农话题比较火，老俞在此之前也经常在抖音直播，所以他知道这个赛道可以做。

就是：这个项目刚好有前景，我刚好懂一点，所以就去做了。

假如他之前一直做的是教培业务，从来没有做过直播，说不定就不会入局直播带货这个赛道，因为这个赛道在他们当时公司的第二曲线中完全没有话语权。

所以我为什么我选择闲鱼二手书？

因为我正好手上做过几个月的营业执照代办业务，知道国内资质和执照代办行业的基本情况，所以出版物对我来说基本没有任何难度。

然后自己也运营过淘宝店铺和多多店铺，做的都是虚拟和服务类目，对电商平台有一定的了解。

所以就决定做二手书这个项目，当然，做决定的时候我也纠结过是做小红书还是闲鱼，最终决定做闲鱼是因为小红书还多了一个环节，就是选品，你到底是做文具类目还是服装类目，这些都需要测试，搞定之后还要找供应链，后端的这些环节都要花时间去研究。

但是闲鱼的后端基本不用管，你卖什么？卖的是书，供应链在哪里？孔夫子！所以后端就不用再花时间去研究了。

其实自己还有一个长远的计划，就是因为闲鱼项目周期足够短，小白上手足够简单，所以只要跑通了，自己接下来打算去抖音直播卖课，教大家做闲鱼，其实这个才是真正赚钱的板块，（因为自己创业的第一个项目就是做抖音知识主播，当时日入3000+）

这个板块大家有兴趣可以留言，后面展开讲！

所以这就是我决定做闲鱼的全部决策思考！

确定好项目后：马上开始找代办，利用手头的资源，找了一个能7天下证的代办，当时花了1150出的证。

下证当天把三个闲鱼注册好，对应好支付宝和手机号，（我是两个手机登3个号）

开始配置华少的闲管家，测试自动上品第一个成功

第二天开始上架30个品，当时一算账，如果上架满500本的话要17天，时间太长，等不了。

于是第三天开始每天上架50，后来还是觉得太慢

后来每天上架100本，就这样每天坚持

操作的时候我会把前一天曝光最低的全都下架。

就这样坚持了6天，第7天的时候有客户咨询了，但是咨询的都是软件自动上架后那些价格标的非常便宜的，有错误的，基本都是亏本的单子。

于是前5单我都是亏本贴钱出的单，因为这至少证明这个项目普通了，于是就这样第二天，第三天，。。。

第一天出单后接下来的一周，每天都有5-10单，。。

所以这个项目要说有什么门槛，其实基本没有，比的就是执行力。

这里有几个核心可以和大家分享一下

核心一。

这个项目就是通过批量上架书籍来测试平台流量，从而找到高曝光的产品进行优化然后出单的项目。

所以一开始我的计划就是粗暴上书，批量处理，所以在上架的时候图片和介绍这些东西我们都没有精细化的去调整。因为即使你调整了，说不定这个链接也出不了单，你要做的就是用数量去对抗不确定性，从而筛选出值得你精细化调整的链接。

核心二。

关于出版物：你无论找任何敢给你保证使用几年或者年审，或者有异常帮你处理的人，基本都不可信。

大家想一下你是在哪里找的代办？无非通过群友介绍，或者淘宝这样的电商平台找到的，这些代办的源头都是当地有一定渠道的人，这些人接单就是批量几百几千个的去出证，根本没时间做客服，一个人的单子他们也看不上。

所以我们大部分接触到的代办都是中间商，他们给你的保证都是为了能让你在他那里下单，至于后面有什么问题大家都不敢保证。

所以出版物就相当于一个消耗品，办下来说不定哪天就有问题了。

核心三。

闲鱼账号三个号之间基本不会相互抢流量，这就意味着你可以批量账号操作，我现在刚做一周，平均每天80-100多的利润，如果我再开3个号，再好好运营，是不是接下来做到月入过万是非常简单的。

而且前面的瞿大佬已经用团队化的运作模式证明过这个是可以批量操作的，这也是我接下来的方向。如果卖书做的好的话就多开店卖书，如果利润实在上不去了，我就去抖音直播教别人做闲鱼。

所以总体来说，闲鱼这个项目是普通人实操相对简单的了，觉得合适的人都可以去尝试一下。
