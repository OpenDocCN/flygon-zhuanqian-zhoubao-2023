# 一、如何月入1000

在这个阶段，你只要运营一个头条号或一个公众号就能达到

假如你一天写5篇文章，一个月就是150篇文章，每篇文章的收益是6.7元。

目前头条文章的收益是1万阅读20块左右，公众号文章的收益也差不多。也就是说，平均每篇文章的阅读到3500左右。

要达到这个数字，新人需要至少一个月的训练，当然有一定写作基础的会快很多。看到这里，很多人或许开始吐槽了，就赚这么点钱，还要训练一个月！

其实很多项目刚开始都是不赚钱的，做过的人都知道

在头条上或者公众号的创作者有几千万，怎么可能这么简单就赚钱？80%都是在月入1000以下的水平，大家一定要认清这个现实。

需要训练1个月，那训练什么呢？

## 1、训练网感

什么是网感？网感就好像老司机，开车时间久了，听听引擎的声音，就知道车况如何。

写作也是一样，每天拆解爆文，学习别人的内容、结构、观点、语言表达方式，文章中的关键词等，这些就是最简单的训练网感的方式，这方面没有捷径！

## 2、训练敏感度

什么又是敏感度？以前你看新闻，新闻与你没有关系，现在你看新闻，新闻就是钱，你能发现怎么写新闻，什么时候写更合适，阅读量会更高。

举个例子：不管什么新闻，一般的时间周期都是6天，从你看到这则新闻到新闻消失在大众视野中只有6天！

敏感度高的人，在这6天里就能通过写作赚几千！记住一句话：热点就是流量，流量就是钱

## 3、训练边界感

什么又是边界感？我们以情感故事为例，请问什么样的情感故事，最受中老年男性喜欢？

我想你心里已经有答案了，但是我们能不能赤裸裸的表达？绝对不行，一旦出现违规词汇，你的文章就会被下线。

可是，太正经的文章又没有人看，那怎么办？

这就是边界感的价值，既能够让大家感兴趣，又不会违规。

以上三条，试问一下，哪一条是能通过简简单单的复制粘贴就能掌握的？

所以，不要轻信网上那些说简简单单就能赚钱的项目了，一旦你相信，他们会有一万种方法去收割你。

# 二、如何月入1w

达到月入1万的程度，有两种方法，一是提高阅读量和单价，二是提高写作的数量。

## 1、提高阅读量和单价

在写作这个领域，估计你听的最多的一句话就是：把读者喜欢的内容送到他的眼前！那我们该如何提高阅读量和单价？

现在中国最大的市场，就是中老年市场。到2035年，我国60岁以上年龄的人占比40%，80后的人群到了快退休的时候

那么这群人最感兴趣的是什么内容？比如：养生养老，

只要是目标人群找对了，我们的阅读量就会很！

再来说另外一个：单价！

我们的钱是谁给的？当然是平台给的，相当于我们把内容卖给了平台。

那平台的钱又是从哪来？肯定是从广告商来，也就是说广告商越多，他们越愿意投广告，我们文章的收益越高。

既然如此，请问大家，保健品是不是很难做广告？

利润空间大，广告商又非要投放广告不可，但保健品广告限制多，这个时候，软文广告是不错的选择。

除了上述所说以外，还有的技巧是：公众号SEO技术，头条爆文技术等。

## 2、提高写作的数量

说白了就是每天能多写几篇文章，一个人能多运营几个头条账号或公众号。

有人说：我每天就几个小时时间，平时又要照顾孩子，又要上班，哪有这么多时间去写文章？

这时候，如果我们有AI来帮忙呢？几分钟就能写一篇文章，而我们只需要润色文章，排版发布，是不是效率更高了？

但是，我在这里要给大家泼冷水了，AI只是工具，工具好不好用，取决于使用的人。

如果你没有写作基础，AI工具对你来说就是半个废品，所以打好写作基础是必须经历的过程。

# 三、如何月入5w

这个时候就不能靠自己了，而是要靠工作室，我们社群目前运营了32个工作室，也就是批量生产文章，批量运营头条、公众号的小公司。

一般小型的工作室，运营的账号50个到200个左右。

大家不用感到惊讶，因为所有人都是从运营好1个账号开始的，你1个账号能稳定收入1000元，运营200个账号，其实就是分钱和分工的问题。

有的人会问：“天哪，我去哪找这么多账号？

我来问问你，假如现在我跟你合作，你提供头条账号，我提供文章，我负责写，你就负责发送文章，你愿意合作吗？

所以账号不缺，缺的是能批量生产高质量内容的人！

然后，可以找专门排版的人，账号找人合作，AI工具让人定制，自己就负责最核心的工作：“打造AI提示词，保证内容的稳定性！”
