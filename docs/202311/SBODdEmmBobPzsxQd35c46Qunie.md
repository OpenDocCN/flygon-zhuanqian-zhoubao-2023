# 📝第一个创业项目

毕业后呆过公安局，后来又进入了家族企业，负责企业规划，日子稳定且一眼到头。因为亲戚想做个甜品店，我也想脱离打工人身份，于是合伙，赶紧冲~一头热的创业，总是死的很快，店铺大概支撑了几个月，就关门收场了。

复盘：

一、优点

1.开业前期，我通过当地最大的自媒体（微信公众号）投放了软文广告，虽然不知道效果如何，但结合仅有的工作经验，觉得宣传一定是有用的。因为做的冰淇淋，又是应景的夏天，加上公众号老板当时给我们做了个噱头，让我们将好多冰淇淋叠起来拍照，名曰“宝塔镇河妖”，开业前期还是小火了把。又加上将“宝塔镇河妖”拿到麦当劳打卡拍照，将免费获得冰淇淋的策划活动，开业人流量还是不错的。

2.当时的还没有私域这个概念，但是我觉得每个进店的顾客留下微信总还是有用的。于是我通过买饮品扫微信，免费送“眼镜吸管”（吸管能从饮料穿过眼睛到嘴巴）的活动，积累了私域客户。每次上新会发朋友圈，也会有老顾客来光顾。

二、缺点

1.创业大忌，和亲戚合伙，我先犯了。虽然我两没有太大的矛盾点，但是后期天气转冷了，我提出需要增加其他品类，他觉得对标的大城市冰淇淋店，人家一年四季只做冰淇淋，也能活的很好。后期天气凉了，小城市吃冰淇淋的还是少了，幸好，我自作主张增加了“果蔬芋圆”勉强度过。

2.减少不必要的人工开支。因为亲戚还有主业，从开业开始，我们一直有请人看店，现在看来没必要的人工开支，可以免去。毕竟规模也不大，奶茶店规模，两人也能勉强兼顾。

3.人说女孩创业三件套：奶茶店、花店、甜品店。哈哈哈，我又中奖了。刚从象牙塔出来，没有什么商业思维，觉得正好自己也想吃，赚不到大不了自己吃，报着这种想法，现在看来你不关门谁关门。我完全没有调研过冰淇淋这种季节性的甜品在小城市能不能生存下去？有没有其他同行在做？冷门小众的产品，可能是别人走过了，并没有走通，才会导致冷门。没有市场调研，没有商业思维，一头热的创业，只能为爱发电，感动自己啊。

# 💎搞钱的开始

又回去老老实实打工了，四线城市几千元的月薪，吃不饱，也饿不死。2018年状态不是很好，看到了长投的9.9元理财入门课程，秉着买不着吃亏，上当的心理付费了，也许命运的齿轮在此刻转动了。我已经忘了里面讲了什么内容，但的确是我理财意识的启蒙。你不理财，财不理你，不是有了钱才去理财，而是有小钱就该理财，鹅才会越来越大。加入生财以后，发现很多人都是长投出来的，以前身边根本没人知道长投。后面我报了线下见面会、基金、股票课。在线下认识了一波谈钱不伤感情的人。被“忽悠”着买了基金、股票，所谓的“白马股”，多年后我又去看了还在原点，如果我一直在大A死磕就没有后来了，庆幸。

# 📝贵人相助

以前上基金课，班班会加餐，分享他觉得不错理财东西，有次他分享了“港股打新”，我对任何事都充满了好奇心，便顺着PDF上的水印找到了那个公众号，发现了另一个天地。后来他把我们转成了他的私域，告诉我们不如去人少的地方攒点本金，再来大A。我当时真的盲信，因为自己也不是金融出身，对数字特别讨厌，也不会看趋势图，就我这资质在大A赚钱太难了，我不如去人少的地方试试。理财课说理财要趁早，因为开始的越早，鹅越小，所以我想大不了工资亏了，爸妈还能给我口饭吃。最近燕楠大佬的一句话，要么有”慧根“，要么”会跟“，我就跟着比我懂金融的人。于是，我当天就给江苏一些省行打电话想办港卡，吃了闭门羹，后来听说杭州比较开放，第二天就冲到了杭州银行，各种给我设卡，让我买金项链，我知道不值，但是为了港卡忍了，总比飞香港便宜。有了港卡，心里就吃了定心丸，那就冲吧。

# 🔍抓到了搞钱机会

大家应该都知道19年是港股打新的鼎盛时期，我刚开始胆小，也只开了一个户，后来运气好吃了几个肉股，我就知道这东西能搞。后来多户呀，还能撸券商羊毛，有开户奖励。后来我更好奇了，这么容易赚钱的怎么没人知道呀，疯狂互联网搜索，找到了雪球，看到那时很多大V在推，都加了一遍。认识了个朋友，券商开户的奖励给我全返，这羊毛撸的也挺开心。那时候我都会算每年的收益率，最好的时候年化在150%左右，可能玩A股的那些人觉得我夸大其词，但那时候是真疯狂，上市就翻倍涨。现在回头看，那时候还是胆子小，很多人抵押了房产，在搞港股，我身边就有，他们也获得了不错的回报。

复盘：

1.对任何事务都充满了好奇心，什么都想了解下，好奇心促使我打开了新世界的大门。

2.盲信。要么有”慧根“，要么”会跟“，自己不擅长的领域，愿意知识付费，用钱换时间，相信该领域大V。

3.执行力强。只要认定这件事，执行力特别强，立马行动，不拖延。赚钱都不勤快，没事能让我勤快了。

4.大钱赚，小钱我也看得上。很多人只想赚大钱，对小钱都看不上。学了理财后，撸港股券商羊毛，我撸的可勤快了，那时候港股证券的羊毛有的会达到800多元，积少成多，也是不错的收益。

5.进入不懂的领域，混圈子混人脉。因为对港股了解的少，我也总有不踏实的感觉，需要身边有一群志同道合的人。于是我也链接了很多港股打新的大佬，进入了他们的粉丝群，水群，也认识了很多没见过面的搞钱小能手，大开眼界。

6.不懂，我就多学多看。每次上新股要认购时，我会去参考各个大V的意见，看群里的情绪，综合下来，形成自己的认购理由，然后总结复盘，验证自己的想法是不是正确。

7.现在回头来看，我最大的优点是胆子大，一个A股都亏的人，在信息相对闭塞的小城市，敢自己冲到港股去，好在结果都是好的，我也顺利积累了第一桶金，在此之前，真是吃光用光。

8.抓到风口，猪都会飞。我也的确是因为风口，才能积累第一桶金。

9.理财越早越好，小钱也该理财。

10.闲钱去投资。听过一个女的把老公看病钱拿去港股打新了，想着能改善目前窘迫的环境，后来老公看病要钱，她也没港卡，钱回不来。闲钱投资，记住了吗！

# 🚀重大转机

后来跟的大佬说现在也到了港股鱼尾行情，大家转战比特币吧。那时候比特币对我来说就是骗子，泡沫。后来也是出于对大佬的信任，我想定投小玩玩也不碍事。我进场时大概6000多，大佬说你若干年回来看会觉得很便宜，我当时半信半疑。当时定投了一会，很快就有了正反馈，我又像发现新大陆一样了，这玩意真能赚钱啊。后来把我港股赚到的第一桶金转到了比特币。那时候bnb才几十，后来到了几百。除了长期主义，也玩打新，毕竟咱也时吃过打新红利的人，打新也小小赚，还有什么山寨币，也赚了小小波。后来的事大家也知道了，可以说我的第一桶金，通过币圈放大了,也是我第一个100w。

复盘

1.还是胆子大，盲信，大佬指哪我都冲，也获得了不错的收益。人家说不懂的你别碰，我什么也不懂，先冲了再说。

2.运气有时候比努力更重要。币圈老说 “钱是大风刮来的，靠努力是没用的”，我感受到了。

3.选择比努力更重要。如果我不是选择了投资理财这条路，要不是对金钱的渴望，还是拿着死工资，每月吃光用光。

4.耐得住了寂寞，沉得住气。以前玩港股打新的时候，朋友说要是我赚了，立马就花了，我说那这样你的鹅不是越来越小了。如果我在港股赚的第一桶金，都先享乐，那我遇到比特币这个机会，我抓住了，也没本金。

# 💎跨境电商创业尝试

当时在小红书上看到了跨境电商赛道，因为之前都是去人少的地方掘了金，于是冲入了跨境电商赛道。在几个平台的选择后，最后选择了coupang平台，毕竟比亚马逊人少嘛。我又风风火火注册了公司，顺便把心路历程分享在了小红书上。每天泡在上架上，海量上架，一天搞10几个小时，从早到晚。半个月后出了第一单。同时我发现我的小红书好像火了，无心插柳柳成荫。100多的粉丝量，浏览量冲到了4000，8000，粉丝量一个月冲到了1000.越来越多人回复，私信我，问我用什么erp，怎么注册公司,于是我把他们都拉了微信群，无私的奉献自己踩过的坑。可惜的是，我那时候哪懂什么转化成私域成交啊，一点商业意识都没有。爱惜自己的羽毛，别人割了我，我也不想去割别人。很快群里聚集了几百个对跨境有兴趣的人。慢慢的我的账号火的被我那家erp的销售他们发现了，他们开始找我合作，让他们进群”割“，我那么爱惜自己羽毛的人，肯定不答应，任何有人想商业化，我都T出去。后来他们提出，我来转化，一个给我500元。哎，那时候守着跨境电商的店铺，稀稀拉拉的订单，多久能赚500元啊，但是，我高傲（傻），拒绝了，就想是个纯净的交流平台。后期店铺也会陆续出单，但是并没有成体量，对我来说很消耗，毕竟咱们也是见过风口的人，我知道这店铺没有再次像意料之中的飞起来。回到小红书，后来对店铺关注度不高了，小红书还是有人问，我不太喜欢把我没做成功的事展现出来，把所有文章仅自己可见了。现在才知道，我没抓住小红书的风口啊！

复盘

1.听了韭菜课，选择了非标品（家居用品）进行尝试。非标品的好处是还有空间，不像一说到水，就会想到农夫山泉，如果我没记错的话（时间久远，又对失败案例记忆不深刻，说错轻拍）。

2.哪怕那些韭菜课教你怎么选品，比如开学季，多上学习用品这些，你还是选不好品。七分选品，三分运营。道理都懂，还是过不好这一生。于是我只能在大方向的指引下，盲上架，量大概率大，总能撞到吧。但是非常累，现在看来不建议这么做。精度还是很重要，盲目的选品，只能消耗自己。

3.新手想要出单，最好的方法就是跟爆款。看到别人的爆款，立马就跟，价格降到无法让人拒绝的地步，单子就来了。以前就听说赔本赚吆喝，虾皮还挺多商家以本伤人，他的售价，比国内进货还低。在我看来，出单并不难，难的是稳定有利润的出单。

4.发散思维，想自己生活中经常会用到的东西，这种就是高复购，要做有复利的品。这些总结都是后话了，早知道，哎，┭┮﹏┭┮

5.之前跨境电商的选品我基本都是从某多多上一件代发，因为不用运费。实在没有的品，才去1688上一件代发。有人会问了没体量能一件代发？经验是基本1688都能一件代发，如果显示不可以，可以私聊客服，吹嘘自己的实力，会给你发的。

6.我当时第一个爆的品是露营灯，无意上的品，和某品牌撞款了，但我找到的是广东货，价格差了百倍，我的利润还可以。后来我怕品牌纠纷，就下架了，国外品牌意识还是很强的。我后来有复盘，那时候疫情期间，大家对户外都特别向往，于是我上了很多户外用品。

7.后来陆续爆了墙面贴纸，韩国人好像还挺喜欢在墙面上进行装饰，客户里居然还有住别墅的，应该是他们对美的要求还挺高的。

8.我通过喜欢户外活动，延伸了运动产品。天天泡在韩国网站，我感觉韩国人挺喜欢高尔夫的。于是我上架了高尔夫周边产品，高尔夫球杆也爆单了。

9.争对小红书抓到机会，却没有及时转化来说，现在我看来赚钱最重要，吃饱肚子，再谈理想。再清高的人，也是要吃饭的。靠本事吃饭，不丢人。

10.其实小红书来说，刚开始也试图做理财科普，毕竟自己理财拿到过结果，但是吧发是发了，也有人看，但是不一会就违规了，投资理财都是敏感词汇，比特币更是大骗子，哈哈。

# 💎痛定思痛

选择去进修学习了自己一直以来的梦想——服装设计师。后来面试坎坎坷坷，也算进入了服装行业。我当时面试，只要是hr面，我基本没戏，但是老板面我多半成功，当时也不知道是自己的擅长助力了。在服装行业不管是跟的设计师，还是老板都还挺厉害，深圳十佳设计师，或者很早就蛇口独栋大别墅。我知道我所有的匍匐是为了创业，更了解基层，怕将来创业不懂被坑。同时我报了小红书的割韭菜课，想做穿搭博主，后面做自己品牌。老师说如果不是自己背书特别硬，不建议冲穿搭，我还是头铁。一边上班一边拍视频、剪辑、化妆、换装还挺费时间，所有都休息日都在小红书上。坚持了一个多月，浏览量一直在几十，没有什么粉丝增加，那时候疫情严重，我也阳了，把小红书也搁置了。工作的同事都很不错，但是大多时候聊的无法深入，他们好像太打工思维了，娱乐就是吐槽老板，和我想的东西不太一样。后来一个差点成为我合伙人的人说，你不适合打工，你去和他们做同事降维了，有些浪费时间了。

复盘

1.小红书定位不清晰。定位是服装设计师，可以增加工作场景，来丰富人物。增加走播展会、时装周、面料市场。与业内大佬沟通，剪成切片发布。

2.也可以连麦给素人出设计图。

3.定位可以再精准些，如给“年营收XX万女老板设计服装”。

4.提前整理一个月要发的素材，这样才能让你尽可能保证每周都有产出。

5.新手最好保证每周两三次的上新频率。

6小红书最重要的是坚持，看到浏览量低也要耐得住寂寞，话我会说，坚持好难啊，看到帖子的你们一定要坚持！！

# 📝差点有了服装品牌

接着我从公司出来，在想如何创业。也如愿遇到了一个有工厂，做了很多年的老板，正在找合伙人。老板不是技术出身，通过和我聊天，她觉得我身上有她没有的点，她说她比较柔软，很难做决定，觉得我是个杀伐果断的人，正好是互补的。她资源工厂都是现成的，想重新在小红书上建立个品牌，品牌已经走了0.5了，风格我也挺喜欢的。老板邀请我聊了很多次，我能感受到她的诚意。我是心动过。她是一个一直做生意的人，我能感受到她的精明，所有的条款都是对自己有利的。我这人不迷信，但是有个室内设计师以前给我算过，说我不适合合伙，可以找个不错的销售，我又陪朋友去黄大仙算了一卦，我求财！也说鲁班要自己干，不要合伙。我开始问我以前公司的销售主管，后来自己创业开了档口的朋友，她说所有的账目给你看的，都是对方想给你看的，想不想分享，取决于她愿不愿意和你分享。我一下就清醒了，让我全身心投入进去，没有工资，还要投资现金，她不会全身心管这摊子事，还有其他产业，很怕会吃干抹尽，浪费时间和金钱。

# 🔍遇见了生财

今年5月通过深圳线下活动认识的苏苏，进入了生财。其实进入的时候很纠结，知识付费的老韭菜了，被割的害怕。进了生财，我真是开眼了，每天就像个大海绵，疯狂吮吸知识。好久没有这样专注的感觉了，上次有这个感觉还是2018年。我开始每天刷帖子，看到了好多牛人大佬，也参加了线下。好喜欢和一帮臭味相投的人聊怎么搞钱，太让我快乐了。认识了组局大佬叶子，不主动链接，一堆人找她买保险，年纪轻轻就购置了南山房产，羡慕~还有情商很高的瓜藤大佬，和我们分享了web3的一些坑。他和我们说如果赚到些钱，记得回馈社会，他也是这么做的，捐助了贫困学生。这格局，一下子人在我心中高大了起来。后来因为一些原因来到了北京，遇到了宋大和start，都是各自领域做的很不错的人，如果不是生财，我这种小透明，也无法和他们在同一个场合出现，还对话。我老说宋大是我心中女性楷模，很希望看到更多的女性强大起来，当然我也更希望自己这样。

复盘

1.今年虽然没啥特别突出的成就，但是生财这个大宝藏，给了我太多无形资产。光线上的财富，都让我回味无穷了。

2.一定要多参加生财线下活动，如果出差党，每到一个城市也可以参加线下。你会发现每个城市的特点不一样，搞钱的风格和路子也不一样。线下讲的内容，好多线上都不会说，这些才是生财之道的重点。

3.生财的朋友都非常坦诚，如果我和圈外刚认识的朋友，不会说更多的核心，但是生财的朋友不一样，他们真的能问到重点上，问到你自己都没想到的点。同时，他们也会对我们毫无保留的公布一些细节，数字，这种坦诚，你在外面交个朋友，不深交几年人家才不会和你说呢。

4.最近痴迷于学网球，初学者很难对打，接球也难，网球比羽毛球真是难多了。我的教练会和我说：“如果你球拍向上球会是什么结果，球拍向下会是什么结果”。我说：“耳朵会了，手还是不会”。他说：“如果我不告诉你哪里出问题了，你永远都不知道问题在哪，但是我告诉你正确的姿势，你以为学不会，但是一次次在进步，一次次在修正自己的姿势，最终就会回到正轨。”最后，我真想他说的那样，每次进步一点，最后修正成了正确的出球方式。我想说的是生财这些大佬在线下分享的干货，你看着吧，好像和你关联不大，参考不大，但是潜意识会受到他们的影响，最后在自己的行为处事中，一次次修正，最后向他们靠拢。

5.初期新手去线下活动要多听，多看。刚开始参加深圳线下，我除了自我介绍，很少说话，并不是我不擅长沟通，而是深圳的大佬们都是身怀绝技，让我这个小白大开眼。我光听他们分享搞钱故事，就活力满满。大家也知道E人，是通过沟通，才会产生快乐，我第一次发现听众也很享受。

6.如果你在深圳，建议你参加工作日活动。深圳人都是除了主业，还有副业的，工作日的局基本上都不是”牛马“，深圳工作日的局也特别多。他们的生财之道五花八门，社群创始人、保险主理人、多家美团超市老板、公众号变现几十万的大佬等等，每一个都会让你开眼，原来钱还可以这么赚。

# 📝一个月赚回生财票价

我每天都在风向标里溜达，想捡个赚钱的信息，但是看的越多越焦虑，越来越不知道自己要干什么。亦仁说如果要进行下一个知识付费，你就要至少赚回10倍，那我要赶紧赚呀。焦虑啊，一直没转到好的切入点。后来瞎溜达，看到了证券撸羊毛，我前几年每年要算年化的时候，迫使自己疯狂搞主业之外的钱，做过这些，我知道稳赚的。我就想目前没有找到切入点，但是我必须要赚钱呀，一个月赚回了生财票价。也告诉了苏苏，她也赚了波，她拉我进生财，我应该反馈下她，哈哈。就证券撸毛而言，从以前到现在我应该赚了好几倍票价。有人说现金流不丰富，怎么撸呢，我当时钱都在投资上，现金流很差，朋友教我撸信用卡，一环套一环，因为这是稳定收益，所以没啥风险。其他高风险的不建议撸信用卡哈。如果你私域丰富，也可以自己赚佣金，放单子给他们做。我自己不做这些，不是广告，只是给大家说个撸鸡腿机会。

# 🚀中了风向标

和苏苏闲聊，她说你去小红书发要投资，肯定很多人来找你。当时我本来也想投项目，也想看看人家做什么。于是发了小红书做测试，粉丝量小几百的账号，浏览量冲到了1000多，不停有人回复，私聊，又有种风好像要来的感觉。因为写了在北京的缘故，大部分是电影这个赛道，还有其他汽车电池、火锅店、面馆等，原来大家都在做这些，我看了也觉得是个很有趣的测试，最后因为没有我特别感兴趣的项目，都没有深聊下去。过了几天，小红书的违规警告来了。最后为了凑3条，把这个发成了风向标，还中标了。后来在风向标群里，有人说合伙人看了觉得有启发，那种帮助到别人的感觉很不错呀~没想过自己也能无心插柳，帮助到别人。

# 🔍总结

今年我听说，之前港股打新赚几百万的人，后来去玩了A，全亏完了。靠运气赚了的钱，终将回去啊。此文不构成任何投资建议，我的原则是从来不拉任何人投资，拉人投资在我眼里和杀人放火差不多。我也算曾经抓到过小风口的人，起起伏伏，一个资质平庸，但又疯狂想搞钱的女孩，瞎折腾搞了很多，成功的很少。目前我在疯狂找项目，就是不想打工啊。测了盖洛普，我的沟通和交往靠前，这才知道为什么我面试会比较容易过。但是以往的打工经历，几乎没展示过我的优势。上周的北京线下，有圈友说我表述自己的瞎折腾经历、吐槽很有趣。那我是不是应该往自己擅长的地方去深耕赛道。播客和访谈，最近也有了解，但是吧，为爱发电对我目前来说太难了，肯定是希望正反馈快，变现快的项目（谁不想呢）。现在是迷茫期，也希望跑通项目的大佬带带我，也许我不是最聪明的那个，但是我执行力杠杠的！听话照做没问题。不白嫖，知识付费老韭菜。最后，祝看过帖子点赞的靓仔，靓妹日进斗金！
