# 围观船员视频号带货如何7天10万粉变现5w+

入生财三年了，第一次发文，也是第一次航海

很惭愧，别人都10月底上岸了，我10月底刚刚上船。

10月的航海实操报了3个项目（创业IP，共读，视频号带货）

先说结果：

创业IP-全网100多个粉丝（我花精力最多的地方）变现4k

共读—上岸

视频号带货-未上岸-10月28号开始登船4个账号到11月7号变现5万+

我的背景（项目能做起来有运气成分也有过往经历的成分）：

给冬奥会吉祥物发布会出过整场节目。

接奥组委项目到美国归化过队员。

（谷爱凌大家应该都知道，没错他是另外一个项目组归化回来的哈哈。）

我们规划的人物叫朱易，很遗憾没拿到好的成绩。

19年去了杭州，在发小@直浪那里学电商。

基于这个版块我们开始自己做抖音直播。

网上那些妖魔鬼怪的花招都学过，没做起来。

最后是参加了水松张阳的课，理清了思路。

20年自己的盘稳了以后开始接一些陪跑和代运营。

项目接不过来，风生水起。

然后21年扩张后赔个底掉。

而是因为及其不稳定，现在还能活下来的代运营公司大概率是自己有好的货盘或者背靠了愿意给你肉吃的品牌爸爸。

很遗憾之前没意识到。

今年把项目停了回到长沙。一些陪跑业务偶尔自己接接，开始找新的方向，准备重新起航。

参加生财应该有快3年了，第一次参加航海。

在前20天精力都只够投入到创业IP的，导致视频号带货航海被划为围观船员。

这篇主讲视频号航海，回到正题。

我是如何在围观船员的情况下拿到7天10万粉变现4万的结果，现在每天继续在进账。

一、视频号带货开始之前的准备

群消息从头翻到尾+航海手册+生财搜索“视频号”关键词

几乎这些资料整体阅读不下5次（盗坤文林等等大佬的文章很有感触）

查资料更是数不清多少次了。

二、通过收集信息进行剖析

几乎所有的在玩视频号项目的大佬都在说从7月分开始收入降低，搬运受打击严重。

结合上航海群里的信息点。

本质上项目是从除视频号之外的其他平台找到好的素材，同步到视频号获取流量，再进行转化。

分析（7月份开始收入降低，平台打击搬运很多账号0播）这两个问题的原因。

我的猜测：大概率入行的人多了大家都在做带货视频；以前纯靠搬运，现在平台开始有打击动作了）

三、如何从这两个问题中间找到突破点

我在的是视频号带货2群，直到登船的时候变现的船友确实不少，但几乎都是概率性的爆，或者过一段时间之后就0播了。

我想避免掉这些问题。

概率性的爆，绝大多数人做的是商品的视频，以搬运为主，用数量对抗概率，实际结果下来可行的，但是后期容易0播，搞一波流。

纯粹做商品的视频其实很难爆。之前已经有大佬入局吃到蛋糕，像盗坤说的一句印象深刻：大家都在一个鱼塘捕鱼，他们在船上用网捕鱼，我们在岸边用钓竿。并且视频号航海应该有上千人吧。

打个比方，每个人5个号，这就是5000个号了，每个账号10条商品视频，这就5万条了。

做商品类型的种草视频成功概率太小了，并且还有很多有船的大佬，现在都开始付费了，像我这样的单打独斗打不过。

大路上的人太多了，我换条小路。

四、视频号别人做商品，我做人群，伪原创。

做了四个账号。

A:20岁-30岁，刚毕业的大学生白领群体，特征：虚荣心强但穷。

B:35岁-45岁，80-90年代的怀旧情怀

C:50+的女性，亲情共情。

D:还有一个带货账号

AB账号2天破百万播放，7天破千万播放，真正变现应该是从5，6号开始到现在大概有2W多

C账号第3天破千万，第5天单天4000万播放，7天破亿播放了，6号开始直播。大爆款的人群会泛，产品可能也没选好，实际变现其实只有3万左右。

D账号更了20条没什么反馈，其他账号都起来了于是就放一边了。

（这里的变现金额为入口袋的利润）

1、如何做人群：

我把短视频内容分为“情趣用品”四个字

情-共情（举例：喜怒哀乐）喜怒哀乐里面又能延伸小朋友的喜怒哀乐，20岁的，30岁的，40岁的，50岁的，男人的，女人的，有钱男人的，没钱女人的等等一些列分支。

趣-有趣（有意思的实验，80岁老人做口播，一些反差较大的有意思的内容）

用-有用（你教会我做菜了，你在教我做PPT，你教我做电工等等）

品-新奇特的产品（北方人大多数没见过酸角糕，很多地方特色的绿植其他地区没见过-昨天看文林写了一个爆款叫彼岸花的盆栽）

这四个板块的分支的延伸可以细分到上千种。

所有短视频平台的爆款内容，都可以纳入其中。

做好定位应该是每个内容创作者最初期的想法，但是往往想要的越多越陷入困境。

这些成千上万的分支里面能扣到一个点，其实就够了。

2、如何伪原创

基于4.1如何做人群，定位好自己的风格。

抖音，快手，优酷，知乎，小红书，甚至TK，所有的短视频平台都是你的素材库。

这里分享一下安先生短视频课程我记录的Xmind

通过这个底层逻辑，你会发现很多爆款都是有迹可循，我们要做的无非是把这些素材不违规的二创成自己赛道的作品，进行发布。

通过大概几十条作品的制作之后，你大概率会发现其实有些作者没上热门的视频，其实他的画面或者他们音乐很好。

你就会开始留意那些视频的那些片段，或者音乐，脑海里面会自然给他做一个合适的拼接。

（这里建议如果多少赛道同时做的话，尽量分时间段去找素材，真的很容易脑裂）

当然关于底层逻辑这里，不是一天两天可以练成的功夫，这也是我做抖音时候积累下来的一些经验，不过不用特别担心，脚踏实地去拆解，百炼成钢是没错的。

五、知识的诅咒或思考者的退堂鼓

这几天案例做起来后，也有不少朋友来问方式方法。

其中有一部分点，大家警惕，不要让自己陷入到知识的诅咒当中。

这部分人群其实对短视频平台，内容制作有很深刻的理解，但是考虑的太多。

比如：我做情感账号应该怎么变现，变现效果万一不好怎么办，我的资源能不能尽量的都用上等等远处的困难。

反而没有顾虑的人执行起来更容易拿到结果，切勿陷入精神内耗。

这里分享两点我是如何克服精神内耗的。

第一点大多数人都会在意的点，账号如何变现，能赚多少钱？

这点其实航海教练也说过很多次了，想知道你的赛道如何变现去看你的对标。

第二点没有播放量的焦虑如何克服？

不去看他，我给自己定的目标是一天一个账号10条视频，比如今天我剪了又40条视频。

一天最多看一次后台数据，去分析视频号的人群喜欢看什么类型的内容。

（抖音的热度内容不一定在视频号爆）

六、接下来我怎么玩？

账号爆了三天，自己播了三天，真的坐下来开播就是钱。

一个小时几千块的感觉，会让人沉迷。

但我意识到不能这样，如果每时每刻都在直播，我没有精力去做内容，流量迟早被消耗完。

而且主要三个账号一起爆，我只能选择最好的账号直播。

吃了三天最高峰流量。

第四天大概只有2000万左右播放了，我把直播时间减少到3个小时。

正好这个时候遇到一个广州的直播基地，在抄我的账号（不得不佩服，我才起来几天，现在已经有长沙一波，北京一波，广州一波，云南一波在复制了）真的很卷。

我都加了这些同行的微信聊天，大家一起共创。

在账号第四天我就想过招主播了，考虑到好的主播不划算，不知道这个账号生命周期有多长，培养周期又比较高。正好想到这个基地这个，我跟他聊把账号拿去给他们播。用他们的供应链，这样供应链成本又能下来，我只需要专注做流量，拿分润。

于是第五天就签试用合同了，到10号已经播了有3天了，我每天躺着收钱。

我的思路一直是专注一条线。

如果我又做账号又直播只能是个人打法，能赚钱但有限，你也不知道什么时候平台打击你的账号，就一无所有了。

专注流量自己孵化账号，同步找伙伴加入一起做账号。做出的账号输送到基地去日不落。

帮助基地打造他们的视频号案例，然后手牵手去做商家。

大概方向是这样，之前抖音DP盲目扩张团队的惨痛经历在，不想重蹈覆辙，接下来的寒冬里，借力打力，做一个温饱的超级个体户。

希望以上能帮助还陷入困境或坚持不住的伙伴们一些希望，视频号大有可为。
