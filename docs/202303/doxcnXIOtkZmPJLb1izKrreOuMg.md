今天在拓跋勇群里，勇哥向我确认了，他差不多两三分钟发个帖子，由此我就安心大批量发布了，本来还打算等发完的帖子看情况。

# 二，基础

![](img/51f87d4d6054346ba980c59f6f84d53e.png)

AMZON POST是去年亚马逊推出的一项测试计划，仅适用于在美国市场中拥有活跃店面的品牌注册卖家。

目前是免费的，但需要完成店铺创建并上传链接的条件。

我很早就听说过，有人通过大量上传产品相关的图片，以此获取非常多的亚马逊站内免费流量。

但我一直都还没试过，一来是上班时公司没重视，店铺都没创建，也没产品素材；二来搭建团队的时候，碍于整体成本和时间精力，顾不上那边去。

现在我则有比较多的时间，也达到了基础认知，通过一些方法能够比较快速地完成大量的帖子发布。

# 五，图片

咦，你是不是觉得，我少讲了，那么多图片素材，哪里来呢？

方法很简单，你不需要完全去真正拍摄或者修图所售产品，只需要找到产品同类的场景，以及对应的受众即可。

比如我在《蓝海产品方法论》当中所讲的几大人群：教师，司机，护士，父母等等。

其实这里，顾小北的精细化运营文章之一，给了我启发，不信你可以自己查阅一下：亚马逊如何巧用post ，让新店0-1。

其实这个也是潮汕牛肉丸的抖音策略，批量发布产品相关视频，反正只要有免费流量，就出现你的产品链接就好，受众尽可能精准。

最开始我是打算设置定时发布策略的，但是后面核算了下，这个真的很麻烦，还不如超级简单的，直接延时即可，如果你仔细研究的话，会发现我是设置了两分钟，也就是200000毫秒延后。

给你个大批量获取产品相关图片的方法：

第一步，打开谷歌浏览器，输入google.com

第二步，搜索无版权网站

第三步，在这些网站里，搜索你的产品关键词

第四步，安装好浏览器插件后，右键单击会出现这个按钮

![](img/9ac9e719554529a7284ff52f482f1703.png)

这个没法截图，手机上拍了张。

第五步，点击提取本页图片

第六步，依次按照顺序，点击图中红框

![](img/6aaf05dd2941b3634c2f0800c76e049c.png)

第七步，点击全选和下载选中

![](img/d3ee7e216b7c0e500845dbc4462d906a.png)

第八步，依次点击自动下载和继续下载

![](img/9af707925a80eb6ec1c5e2f2c1bb24a6.png)

如此，整个页面上的所有图片，全部都下载完成。

# 六，注意

AMZ POST，图片标准是

*   Image size: 640 x 320 px or larger

*   Aspect ratio: Between 1:2 and 2:1

*   File format: JPG or PNG

*   Color format: RGB

不建议图片上带文字。

请仔细阅读亚马逊官方POST规则。
