什么是Prompt提示关键词？

大家有想过这个问题吗？什么是Prompt提示关键词？这么多行业成千上万个Prompt提示关键词，需要我们去挨个记忆吗？它们有没有什么类型特征？

什么是Prompt提示关键词？规范的定义是

prompt 通常指的是一个输入的文本段落或短语，作为生成模型输出的起点或引导。prompt 可以是一个问题、一段文字描述、一段对话或任何形式的文本输入，模型会基于 prompt 所提供的上下文和语义信息，生成相应的输出文本。

直白点说，就是你在输入对话框中输入的任何文本信息，都是Prompt提示关键词。去除那些无关的话语，输入的Prompt提示关键词，可以归类为4种常见的基础模式：

分别是：特定指令、指令模板、示例模式、代理模式。

四种不同的模式有着不同的特征，适用不同的场景应用，就像专业的事交给专业的人来做一样，合适的模式在合适的场景下，会提高获得想要答案的效率。

接下来，常常会以“推荐使用ChatGPT制作简历”为主题，对4种基础模式进行示例，既方便大家对于4种基础模式的理解，也在示例中为大家清晰演示，4种模式下，使用ChatGPT协助创作小红书笔记的过程以及各自展示的效果。

一、特定指令

这种模式下，我们会给模型提供一些特定信息，例如问题或者关键词，模型会根据这些信息，生成与这些信息相关的文本。这种模式通常用于生成答案、解释或者推荐等。

比如，在我们有了一个想法，想要进行一篇小红书笔记创作的时候，但是又不知道如何写，没有头绪，我们的提问“你知道小红书这个软件吗？”就是特定指令。

这种指令出现在探索一个新领域，或者一个刚刚接触ChatGPT不久的新手中，如果可以类比的话，这个时候的ChatGPT就像是供小孩子拿来查阅的百科全书，真正的价值远远没有被发挥。

二、指令模板

指令模板模式下，需要我们给模型提供一些明确的指令，模型需要根据这些指令生成文本。这种模式通常用于生成类似于技术说明书、操作手册等需要明确指令的文本。指令可以是单个句子或者多个段落。

ChatGPT最大的价值是将输入的非标数据变成标准化数据，指令模板是ChatGPT价值最直接的特征体现。

这种指令模板的形式，适用于对小红书笔记风格有了初步了解的人，这种提问会有更高效的答案反馈。

你来看，这种关键词模式下写出的内容，是不是明显要比特定指令下的内容更为具有小红书的风格？

三、示例模式

指令模板很好用，但是ChatGPT给出的结果，我相信并不能让很多追求高质量的圈友们满意，因为标准不会错，但也代表着呆板没有创新。

这个时候，我们可以试试示例模式。示例模式是最像我们人类学习行为的模式。

真正的学习，绝不是坐在那里听PPT，而是看看周围的人在做什么，然后你去模仿他。

示例模式下，就是让ChatGPT去有意识的模仿我们想要的答案，我们给模型提供一些特定信息，比如问题或者关键词，模型需要生成与这些信息相关的文本。

这种模式在小红书上直接使用方法是，我们可以直接把自己寻找到的优秀笔记，以示例的形式，展示给ChatGPT，请它模仿笔记，使用我们给定的主题“推荐使用ChatGPT制作简历”来进行创作。

这种模式需要小红书笔记创作者，对小红书的爆款笔记有一定的了解，能够找到适合自己对标的爆款笔记。在这种模式下创作，ChatGPT给出的答案，相对来说效果更好。

四、代理模式

最后说的代理模式可能是大家最为熟悉的模式，为什么说可能，因为很多人使用了，但是并不清楚这是一种模式。
