# 我的副业履历

我是从20年开始接触副业，期间做过小红书博主、闲鱼无货源、多多视频、抖音本地团购、国学书单视频号、抖音游戏号、头像号等等，最终聚焦在小红书赛道

我也迷茫过，项目太多，根本不知道该做哪个，每一个看着都很赚钱，但真正让我赚到钱的，是小红书项目。

## 1、第一次副业：小红书好物分享博主

第一次副业源自2020年，公司做的是好物分享矩阵号，这类型账号的内容在当时的小红书很吃香，非常容易出爆款。一个账号在旺季能带来一两万的利润，这类型账号公司有十几个，运营总共就4个人。（图）

内容也很简单，很容易复制，于是我自己简单学习了下ps，就开始了。刚开始运营第一个号，就在一个月左右变现200块钱（广告合作）

看到收益后，找了个兼职美工帮我批量做图片。保持日更频率后，一个月利润有2000。之后我开始复制了账号去放大，我所有手机号都注册了，而且全部跑通变现。

最多账号的时候，我有包括好物分享、零食分享和穿搭在内的20多个账号，平均一个月广告收入1w+（要是当时接触了淘客，收入能翻好几倍）

但随着小红书对广告营销的打击力度增大，我的广告合作收益日益减少，账号合作报价从500降到300再到200，我终于意识到不能再继续了。于是我把20个账号缩减到3个，收益不好的全部卖掉，出售账号赚了2w+。

小红书项目从2020年中开始，到2021年中暂停，1年时间总收益大约8～10w左右，剩下没卖的3个账号每个月大概给我带来2000～5000的收入

## 2、闲鱼无货源

2021年底，我了解到闲鱼无货源的项目，网上找了份文档来学（后来才知道，文档出自生财有术）

我做的家居、户外的产品，平均单价在100~300，利润20%左右。在实操了1个星期后，出了第一单懒人沙发，单价160，利润30，非常开心。

但很快问题出现了：

1、不是专业做电商，对这个行业比较模糊，选品这有比较大的难度，过程重复且枯燥。

2、有很多人咨询，但成交的确实不多，我是个急性子，没有耐心当客服。

3、遇到个大傻X，下了几个订单，在上家刚发货的，我正在同步单号的时候，退款了。

最后这个问题直接让我放弃这个项目，可能跟我的选品有关，我真的太讨厌闲鱼那些用户了，什么妖魔鬼怪都有。

浅尝了1个月后，放弃了闲鱼无货源，总收益30块，我实在太不适合做客服类的工作了，我会忍不住骂人

## 3、多多视频带货

2022年春节在家实在没事做，又了解到多多视频带货，于是我又开始了一个新项目

直接从抖音快手和小红书搬运视频到多多视频发布，很快也出了爆款视频。但是由于第一次做带货类没经验，犯了个致命错误：没有留意佣金比例。

那条爆款视频带来的佣金全部都是0.01元～0.09元……，支付金额很大，佣金还不够塞牙缝

发现问题后，我开始找高佣金比例的产品，也小爆了几个视频，每天佣金收入有200多

后来多多视频开始调整，我搞不懂规则，到最后视频没流量了也就放弃了，总收益1300元

多多视频这个项目让我第一次感受到，一个优秀圈子和一个优秀老师对第一次接触副业的人来说有多重要。

## 4、其他副业

2021年中到2022年中，是我的迷茫期，这期间我接触到包括闲鱼和多多视频在内的很多副业，每一个都想做，但每一个都做不好

### 视频号带货

视频号刚出的时候，我觉得跟多多视频类似，也搬运内容去做。当时做的是国学书单，视频很容易爆，但比多多视频更容易违规

出了1单，佣金36元，视频一直在违规，也就放弃了

### 资料号/头像号/红包封面/抖音游戏推广

这些副业我全部都尝试过，有些跑通过变现，因为各种原因放弃，总觉得这些都不是我真正想做的事情

在迷茫期，我做过太多的项目了，看别人做什么赚钱，我也想做。还好这时候我没在生财有术，不然我可能会做更多项目哈哈哈

所有“其他副业”总收入：36元

## 5、再次聚焦小红书

之前做那么多项目没结果我都不慌，但2022年中我小红书3个号收益降到1000左右，我终于慌了，保底项目要没了！！

于是下定决心转型，做一个小红书IP，也是一直延续到我现在做的事情。

做小红书IP的想法，源自剽悍一只猫的个人品牌增值笔记知识星球。因为我自己接触过很多副业，只有小红书坚持下来并且获得收益，所以我认为大部分跟我一样的普通人都有可能在小红书上获得突破。而且因为我工作关系，做过很多类型的小红书账号，起号效果都很不错

下定决心后，我开始打造朋友圈，建立了一个公众号输出文章，并做了“小红书资料库”和“小红书问答集”两份引流文档，这些都为我带来了粉丝和学员

强烈建议做个人IP的圈友去建立公众号和行业问答集之类的东西，效果嘎嘎好！

最新的一份课程是小红书陪跑，这份课程我卖1299，到现在卖了二十几份，课程收入2w+，完全出乎我的意料，而且反馈还很好（现在小红书改动大，课程优化中）

# 我副业的三个阶段

## 没项目

我2017年大专毕业，我很早就有做副业的想法了，但一直找不到项目。

在开始第一份副业前，心里总是吊着一个石头，总觉得没有一个自己的副业，没有安全感，工作也不得劲。毕业三年，我的工资停留在4k，没有任何长进

那时候真的很愁，想干微商，又没有微信好友（这时完全没有引流和推广的思维），甚至去保险公司培训想把这当副业，因为实在受不了早会文化而放弃

我非常渴望有一个项目，渴望到入职公司后，会拆解公司项目，看自己能不能复制下来自己搞。

现在加入生财之后，非常羡慕早加入的圈友。这里的项目，太多了

## 没目标

在第一个项目遇到瓶颈后，我又没有了目标。但因为连接到很多博主，所以通过博主圈，能了解到其他小项目。

但这些项目对我来说实在太多了，每天都能听到一天几千，一天几万，一个月几十万这种项目

闲鱼无门槛无成本，一个月几万块？那我干！

多多视频搬运就能一天躺赚几千？那我干！

视频号带货分佣能达到40%多！？？？那我干！

红包封面是风口！？那我也干！

当时的想法是，我那么多个项目同时做，总有一个跑出来的吧？我把跑出来的项目在放大不就好了？而且每一个看着都不难，自己又会，每个项目花一个小时足够了

理想很美好，现实情况是，只要你把注意力分散，那每一个你都干不好，拿不到正反馈，就没法坚持下去。

这是我第二个至暗时刻，项目太多，反而丢了自己的目标。

## 聚焦

没项目很难受，项目太多又做不好，这时候刚好看到彪悍一只猫的星球内容：一个人如果什么都抓，最终的宿命就是平庸。在打造个人品牌的路上我们一定要搞清楚，哪些事情才是真正的超级重点。

我决定，从这么多个项目中，聚焦一个项目，再打造一个个人品牌。聚焦项目我做了几点：

1、我做过，并且已经有成绩了

2、在这个项目中，已经有了自己独到的见解和方法论

3、方法论可以复制给别人

4、与自己的主业有关联

5、我喜欢这个行业/项目，或者我喜欢做这个事

这一筛选下来，就只有小红书了。确定好了之后，我开始对账号进行调整、转型，并开始打造朋友圈，建立公众号

所有时间只花在1个项目上，效果确实好很多，个人IP也慢慢建立起来了，焦虑感也消散了，问题只剩下一个：如何快速放大

# 我如何靠生财放大项目

## 为什么加入生财

今年3月份，生财放出过3天体验券，我领了，但那时候我犹豫了一下，没有直接加入。犹豫在于，我不了解生财，只当是副业项目的圈子，而我并不需要其他副业了（3月份的我，是这个看法）

3天体验到期后，我刚好需要找AI相关的帖子，我想把AI与小红书结合起来。一个看似不经意且很自然的思路，在一瞬间像闪电一样击中我：我不是可以把其他项目的思路迁移到小红书吗？还可以把小红书的思路分享出来给其他副业的圈友参考，更重要的是，如果我分享有价值的帖子被加精了，我的个人IP，不是多了一个亮点吗？

这怎么算，都不亏，甚至觉得2000块钱是不是有点划算了？于是我决定，果断加入。

## 如何放大项目？

第一步，我需要大量吸收其他副业的精华贴，了解底层思路，把能迁移使用的，都保留下来。

体验的3天期间我已经刷过很多帖子了，这次又把新的帖子刷一遍，包括中标的都看了一遍。看完之后感觉我之前做的东西实在太弱鸡了，有太多东西需要完善

第二步，我需要让圈友知道，我是干嘛的。

我发了第一篇帖子：《全行业通用：小红书引流逻辑和套路拆解》，反应还不错，很多圈友认可，也成功让很多圈友知道我是做小红书项目的

第三步，我需要利用学习来的知识，完善一份小红书运营课程。

目前我正在做的事情，我要把AI、引流、电商等一些项目的逻辑，迁移到小红书，做一份比较完善的小红书运营课程。而且一定要详细，详细到别人学完可以直接落地实操。所以我把AI加进去，还会把训练AI写小红书笔记的流程写进去。

但这一步我会遇到很多困难，这是肯定的。所以如果有大佬愿意教我的话，我非常非常热烈热烈欢迎！我知道生财有太多的小红书大佬了。

第四步，要在小红书领域做一个具体的项目，并取得成功

我目前虽然打造了小红书运营IP，但实际上我本人并没有在小红书上做副业项目，都是主业工作的内容。之前的项目被我暂停了，现今只有一个公众号。所以我决定一定要在今年做一个可以长期做的项目，并取得成绩，为我的IP再添一个亮点。

而加入生财，可以连接到很多大佬，也有很活跃的社群氛围。我要从社群中找到可以连接、合作的资源，来帮助我做成一个大号。

《纳瓦尔宝典》中提到过三种杠杆：

资本的杠杆、劳动力的杠杆和复制边际成本为零的产品，我第一次接触小红书，找兼职帮我作图，实际上就是小范围利用了劳动力杠杆

而我现在做小红书IP，是为了撬动第三种杠杆：复制边际成本为零的产品。生财有术太多这样的大佬了，所以我要吸收大佬们的成功经验，再去撬动我的第三杠杆。
