# 承载与防封

自动化公转私跳微信，转化承载体一般包含两方：抖音号和微信。

所有的防封措施必须同时针对两端考量，否则一方出问题，都会导致转化路径断裂。

## 抖音防封

结合上面的内容，抖音转化通道主要为手动转发，或接口（含官方产品及官方接口），风险最高的在前者。

1.提高账号安全，减少私信频率，群互动代替一对一人工私信

2.提高账号安全，小号为主进行互动，如手动转发抖音卡片

3.提高域名安全，白名单域名代替第三方域名，否则打开会有不定期的安全提示，影响转化

![](img/d2cb3e2ad51736c812b34d442fee30f4.png)

解决方案：这块有相应的抖链转换工具，生成抖音域名网址如https://v.douyin.com/XXX

最高级别防封：

1.认证蓝v/半蓝（推荐，因为不会影响昵称）

2.如有矩阵号，用员工号

3.通过升级玩法，生成卡片和自动回复，避免限制

## 微信防封

如果用官方接口或者官方产品跳转微信问题极少，这里主要说说其他玩法的防封

其他玩法跳转微信主要两种方式：小程序、微信客服

小程序支持的跳转形式如下：

![](img/4e569e5d7c656aed51cacca5ed58e85f.png)

先看几张封禁案例

1.被发现小程序没有实质内容，属于纯引流

![](img/71a7ee3b3357ad1f9744418bdd38aba7.png)

2.被同行举报，轻则封跳转功能，重则直接干废小程序

![](img/3efb03d47700b4d55dfc3bdf1b553c40.png)

防封建议：

1.保证小程序有一定的自然流量，别全是跳转来的

2.展示内容不建议用小程序页面二维码，改用跳转显示公众号文章文章或第三方小程序

3.让服务商开一个属于自己的独立小程序，别用公用小程序，否则一旦别人的项目出问题，你的跳转也会挂；同时用小程序批量注册工具注册多个小程序备用，一个出问题，马上换上

4.购买小程序或对应跳转服务与服务商沟通，如果小程序出问题，是否更新小程序，以及后续更新费用

最高级别防封：

1.小程序增加防截图举报机制

升级玩法采用的就是这类机制，举报小程序时截图显示不了对应内容

![](img/fe5b06a49259fa96ad7df6a6c4c574ca.png)
