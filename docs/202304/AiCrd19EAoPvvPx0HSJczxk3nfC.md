# 《赚30万的项目和赚300万的项目有什么区别？》@亦仁

> 来源：[https://shengcaiyoushu01.feishu.cn/docx/AiCrd19EAoPvvPx0HSJczxk3nfC](https://shengcaiyoushu01.feishu.cn/docx/AiCrd19EAoPvvPx0HSJczxk3nfC)

今天有个朋友问我，赚30万的项目和赚300万的项目有什么区别？

我说，没区别，都是一样的项目。

你赚30万，你以为只有你一个人做到，实际上，在你看不见的地方，大概率有十个、上百个和你差不多的人，他们也赚了30万。

如果你的复制能力足够强，那些人就应该都是你的分身，是你的矩阵，那这就是一个300万的项目了。

你开一个视频号赚1万块，一百个视频号就是100万块。

这是矩阵思维。

星球里面有时候有宝妈圈友分享，一个月通过某项目赚几千块，很多人看不上，觉得是小钱，这些人是真的傻，换个思路想想，如果基础一般的宝妈圈友能一个月赚几千块，意味着什么？意味着这个项目门槛不高，可复制性极强，这才是个大项目，好机会，才有几十倍、上百倍、上千倍的空间和机会。
