全方位拆解一个互联网医馆项目

来源：https://viuejh8h99y.feishu.cn/docx/FqNZdGrjgopNjzxPbdEcltd2n0e

前言

生财圈中大佬云集，可以说目前国内外比较主流的项目，在生财圈都能够看得到。分享今天的这篇文章，或多或少都有一些感慨，大概归结为3点：

1、我们无论是找项目还是做项目，一定要做自己擅长的事情；

2、这个时代无非是赚两种钱，一种是“资源差”的钱，一种是“信息差”的钱；

3、无论做什么项目，都一定要有规划，并且要明确设定每个环节的时间节点。

一、 概述

1. 项目来源

大鹏和团队成员都是90后的创业者，互联网、实体项目都涉猎过，有赚过钱，也有亏过钱。但是说实话，摆在90后创业者面前一个非常严峻而尖锐的问题，就是我们普遍并不具备很强的社会性资源。

何为社会性资源呢？比如小孩上个学，老人看个病，也就是诸如医疗、教育这些社会性问题，我们其实都不算说特别有强大的资源去解决。

没有资源怎么办呢？当然是整合资源！因为YQ的原因，其实很多行业都在洗牌，无论是老一辈的企业家，还是年轻一辈的创业人，或多或少都受到了冲击，这也是一个非常好的契机，让我们接触到社会阅历更加丰富的老一辈企业家。而这个项目中，这位前辈，手上又正好掌握着不少成都本地的三甲医院中医师，甚至是一些国医大师的合作资源。

其实这样的一个资源，是很难得的，举个例子，在成都春熙路的一家中医馆挂号看病，名中医的1个挂号费都要500RMB，后面还要各种检查、再开各种药，又是一笔不小的开销。

所以对于很多普通患者来说，看病难、看病贵是一个不小的问题；

而对于很多的医生来说，他们也不想单单局限于自己的“一亩三分地”，也希望通过互联网，让自己被更多人知道。

因此，基于上述所说的这些条件，这个互联网医馆的项目，也就应运而生了！

2. 项目目标

做互联网的老板们都会选择一个对标，我们同样也有对标。目前国内做医疗赛道的，要想接入到互联网，就一定要有相关的资质认证，比如在抖音上入驻的三甲医院医生，必须要认证了医师证，才能够拥有对应的黄V（当然，现在医疗赛道做短视频也越来越卷了）。

前面提到的这位前辈，他有一个很好的朋友，在广州孵化过2个医疗养生类的大号，一个叫【允斌顺时生活】、一个叫【顺时经络】，基本都是团队20-30人，一年3亿以上的产值。

找到了对标账号，我们也开始设置我们自己的项目目标——短期内，我们是希望在3-6个月起号，我们的账号主要分为3个：

①公司直播账号（以传统养生功法八段锦为直播展示，直播间会逐渐植入爆款引流产品的销售）

②三甲医师黄V账号（主要分享垂直相关领域的医疗问题，黄V的限制会更少一些）

③公司自己的医师账号（主要分享日常保健类的中医养生）

二、市场可行性分析

1. 市场分析

市场定位

①我们的市场定位就是从0起号，逐渐壮大团队，逐渐增加业务板块，并且前期以线上为主；

②前面提到的老前辈，其实在之前的微商、直销时代，也有自己的辉煌经历，当时借着风口，也是积累了蛮多的私域流量。只是现在时代在改变，他们自己的微信商城就没有怎么更新，所以之前的用户绝大部分处于“睡眠”状态；

③我们的想法，是希望在视频号上面开始直播，公域私域相互导流；

④最终线上线下相结合，因为我们线下也有至少1个医馆的场地，并且还可以资源整合更多线下医馆。

市场容量

YQ三年，有越来越多的人意识到了健康的重要性，并且中医证明了自己的价值，让越来越多人愿意了解并尝试。尤其是养生保健的市场，其实还是非常值得去做的。

行业现状

目前行业中确实是有不少头部公司，但是我们非常清楚的就是，目前国内依然是一个医疗资源缺乏的状况，如果能够通过互联网的方式，让我们的专家线上问诊，一对多的方式去看病问诊，那么这中间就有很强的可操作性！

2. 自有资源分析

①如果是以公司形式，去做互联网医馆，那么公司必须要有医疗资质。——这一点我们具备

②医疗领域，一定不能做太泛，必须要垂直，我们的资源，可以保证我们做【轻养大健康】垂直的经营。——后期会加入妇科

③自媒体团队（包括线上的直播、短视频，后台运营，公众号、小程序维护，社群运营等）。——这一点我们同样具备，但是存在一个问题，缺乏有经验的操盘手。

④医师团队，这对于普通人而言是比较难解决的一个问题，一来是很多医师都有自己的本职工作，要提前预约，抽出时间做视频、做直播；二来是医师团队，也就是不止一个医师（我们看到抖音上面的医生，基本每一位都是非常垂直的领域）。——这一点我们拥有至少1位常驻的三甲医院医生，同时我们也可以和中医药大学的教授达成合作的共识，经常性邀请他们来分享专业内容。

三、成本分析

1. 人力成本

前期团队以合伙创业为主，主要是进行了股权的设置。所以算下来，团队的几位成员，并没有日常的硬性工资成本，可以按照2000元/月/人的成本来计算。

2. 运维成本

运维成本主要是直播间的搭建，以及各种工具、道具的购买。这里暂时预估3-5万。

3. 营销成本

营销、做广告这一块，公司预留了差不多30万，来进行前面3个月左右的尝试。

四、项目风险

1. 政策风险

因为我们合作的这位老前辈，也是省人大代表，对于相关的政策，也是比较了解的。另外就是自媒体相关领域的政策风险了，比如公众号平台，各个视频平台等，包括直播、短视频的一些违禁词等等。

2. 技术风险

对于非专业的技术团队来说，需要克服的技术风险还是存在的，比如设备、调试等等。

当然，还有技术风险，是我们前端直播，操练的功法是八段锦，对于新手小白而言，必须要跟着老师的动作去操练，否则容易“走火入魔”，这应该也算是“技术风险”。

3. 法律风险

团队请了专业的法务，这一块我个人没有怎么涉及。

4. 决策风险

这一条想重点写一下，因为一个项目，一个团队，决策者真的不能太多了，尤其是大家合伙创业的公司或者团队，最高决策者只能有1个，否则会出现比较大的问题。

①举个例子，比如开会的时候，需要有一个拍板的人，但是如果是好几个人一起拍板，那么很容易让会议效率变低；

②创业团队的决策者，一定要是懂业务的人，而不是不懂业务瞎指挥的人。

五、结论

写到这里，其实我们的项目，最终胎死腹中。生财上有很多大佬都会在不同时间，分享不同项目的成功经历。那么我们愿意做那个“反面教材”。

结论这里，首先想和大家一起分享一下我们的商业逻辑：

1、前台直播引流：八段锦一类的中华传统健身操，加上爆款养生茶包；

2、后台专家讲解：买了前面说的茶包朋友，还能够免费获得我们的一套课程，是专家对于中医，中西医结合的养生保健攻略；（这个会放在小程序私域进行直播/录播讲解，省去违禁词的担忧）

3、专家讲解完，转化个人/家庭1-2年的全家不限次数免费问诊套餐（套餐价599/1899/私人订制化价格），以及我们对应价值的养生类产品；

4、结合之前微商城的私域分销机制，开始从第一批用户裂变团长，源源不断产生新的个人/家庭套餐。

为什么没有坚持做下去？

1、资金问题，经历了3年YQ，大家口袋里面都有些紧了（做这个项目的时候，也并没有说等到拉到融资再启动），大家勒紧裤腰带创业，创业的成功率确实很低；

2、任何一个大项目，都是需要有很多资源进行对接的，对于年轻创业者（我们）来说，资源调配是一个比较难解决的问题；（懂得都懂）

3、现在很多短视频、直播领域，国内其实非常卷了，哪怕看上去医疗赛道相对竞争小一些，但是要打磨一个账号出来，对于一个这个领域的初创团队来说，还是比较困难的。

关于这段“血泪史”，就分享到这里吧！因为还是投入过精力，最终未能有所成绩还是挺难过的，所以不放图了。生财圈的各位大佬，欢迎多多指教，咱们有啥可以私下聊。
