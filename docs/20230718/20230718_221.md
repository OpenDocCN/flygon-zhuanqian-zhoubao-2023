# 一、 什么是游戏租赁？ 

游戏租赁可以理解为账号出租，通过买断此账号的所有权来进行平台上或者私域上的出租，能简单的理解为，我手里有一百台低中高端的豪车，我只需解决流量问题把我手里的车子，进行对外的出租，来获取收益。 

在 2017 年间王者荣耀在国内可谓火的一塌糊涂，我也是王者迷，在某一天和女同事进行开黑，想着哪里能借到高端皮肤数量较多的账号呢？ 

借了半小时，女同事不断的催我，我就在 QQ 群搜索“王者荣耀账号出租”跳出了许许多多的群，当时抱着尝试的心里，果断加入了群聊。 

看到群管理发的价格表和一个店铺的链接，V8 皮肤数量相对于较多的账号能达到 7-9 快去一个小时，硬着头皮租了俩个小时...... 从那时期开始对游戏租赁这行业抱着很大的期望值，开始慢慢研究相关的知识体系。 

（1）游戏有那些赚钱的板块呢？ 

游戏项目可以大致的分成四大板块： 

1、账号倒卖 

2、游戏账号出租 

3、游戏搬砖 

4、游戏陪玩 

账号倒卖：行业称之为号贩子，通过回收低价账号进行高价出售，此行业需要在业内有一定的沉淀，知名度不断提高之后收入也会不断的增加。 

游戏账号出租：此篇文章讲围绕此板块不断深根细说。 

游戏搬砖：相比在自媒体如此发达的年代，各位圈友也应该会在抖音上看到许许多多，推广魔兽打金，梦幻西游打金等等....一系列 2980 5980 的项目介绍，此行业最累赚钱也是都有着周期性，给大家提个醒入局游戏搬砖需谨慎 切勿被浮夸的收益迷糊。 

游戏陪玩：此板块应该算是比较热门的板块，分为娱乐陪玩 技术陪玩 （当然女孩子在此板块是最吃香的，每小时 8-20 不等） 

# 二、什么样的人群能入局游戏租赁行业？ 

（1）对此行业有着一定的见解、不盲目入局； 

（2）自身也偶尔也接触游戏（什么游戏均可）； 

（3）缺少副业项目，兜里有闲置资金； 

（4）看好此行业，能承担相对的风险 比如说账号因开挂被封号以及业内恶意竞争。 

相对于其他行业游戏租赁的门槛较高，成效较快，只要愿意投入时间精力去维护自己的客户群体，很容易获得第一桶金，相对于较少的沉淀期，执行力至上。 

# 三、游戏租赁能赚多少钱？回报率是多少？ 

以回本周期为例 

游戏租赁项目市面上大多的游戏，回本周期都为半年至一年内。 

爆款游戏基本处于三个月内回本 （投入的资金并没浮动很多）。 

相对于游戏的其他类目板块，游戏租赁可以算是比较长久并且稳定的项目，游戏的周期一般在 3-5 年之久。 

风险是人为可控范围之内，风险相对于较低，人员设备上也无需太大的投入，配合脚本能达到 90%的全自动，业内也有着一句话，一个兵当一个团用，一个人当一百人用，无需投入太大的时间，但只限于王者荣耀能做到 90%的全自动。 

我们拿王者荣耀来说：目前的账号 V10 皮肤数量较多的账号每小时能达到 4-6 块，上下这区间 一款较好的 V10 账号均价在 3000-4000 之间。 

假设我们是十万资金进入到此行业，预计能有 30-35 个账号，单账号放到平台上进行出租的话，每天的佣金区间在 20-25 上下区间浮动，30 个账号的收益也就在 600-750 之间，月收益在一万八到俩万上下… 

所以回本周期基本控制在半年至一年内。 

# 四、游戏租赁能拆分为那些关键环节？ <ne-h4 id="9039450f" data-lake-id="9039450f"> </ne-h4> 

（1）游戏租赁大致可分为三个环节 

第一环节：买号或者起号。 

做游戏租赁的话第一个要解决的就是账号的问题，像主流单腾讯游戏或者网易游戏账号都可以在很多平台上进行购买，不需要去注册在充值购买装备道具，这样会极大的降低成本。 

第三方平台的话可以去找些口碑好的有包赔服务的平台进行账号的购买，这里推荐下盼之交易平台。 

第二环节：就是选择游戏租赁平台进行账号的上架。 

目前国内的租赁平台有很多，手游上做的比较大的平台有租号玩 u 租号等，选择平台的话可以先加入几个平台的交流群，看看里面的商户或者租客的活跃度，还有就是账号在那个平台上的收益率会比较高。 

例如王者荣耀的中低端号，同样一个账号在 u 租号上相较于租号玩就是会比较好做的，因为 u 租号的商户较少，价格能设置对更低，所以这部分类型的账号在 u 租号上就会更好对出租。 

举这个例子就是想说平台的重要性，选择对的平台也是能提高自己的收益率的。 

第三环节：日常维护和打造自己的租客私域，增加客户的游戏体验有利于提高租客多复购率。

# 五、关于新手小白游戏租赁从何积累稳定客源？ 

租赁和账号倒卖都均可从游戏本身的各个社群内获取 

（1）游戏贴吧 

关于贴吧，还是比较适合新手进行入场，因为门槛较低。 

可以通过一些游戏交流板块，通过截图游戏账号皮肤来吸引精准的人群到私域群里，相对于贴吧的账号合租模式，适用于一些老手，账号合租的模式，是无需承担任何风险的。 

很多退游的玩家绝大部分都会把自己的账号放给号商，从而获取收益，基本上收益分成都是 2/8 或者 3/7，分成模式基于你和号主商谈。

（2） 闲鱼 

闲鱼也相对于比较热门，设备闲鱼账号较多的话，可以通过此方法来积累固定的客源，相对于租游戏账号的人群来说没有手续费，比租号平台便宜很多。 

所以说还是有很多人群在闲鱼上进行租赁，在积累到一定的客源之后可以开展相对应的游戏出租群，来维护客户资源。

# 八、以王者荣耀为核心的实操流程 

（1） 观察市场动向对比账号价格 

价格控制在 2500-4000 之间 皮肤数量控制在 380-450 之间 当然数量越多出租的概率也就越高。 

我们这里用盼之包赔平台进行购买是有保障的，后续找回可以直接联系平台客服去找回，切记请勿脱离包赔平台后续发生账号找回得不偿失， 服务保障选择包赔服务 系统操作选择 安卓 QQ 版本 实名最好选择可二次实名。 

![](img/2a90aa3e60a3cf841a061cdd916d1f50.png) 

价格区间控制在 2500-3500 之间 选择皮肤数量较多 荣耀典藏数量多的即可，皮肤数和典藏数和后续的出租率挂钩，因为王者的皮肤更新的数量会比较快，所以尽量选择一些比较新的皮肤数。 

![](img/b415c149a14c5727b3dd0fa518ad0beb.png) 

（2） 账号购买玩进行对外的出租 

这里给大家介绍下账号购买完之后有几种方法进行对外的出租： 

1.私租（通过闲鱼低客单引流到私欲进行对外的出租） 

2.租号平台进行出租：这里推荐 U 租号 虚贝 租号玩 等平台 

3.合租 （ 找商户进行合租 按时拿收益分成 基本都是 4/6 或者 2/8） <ne-h3 id="6c67fe52" data-lake-id="6c67fe52">1.闲鱼私租注意方法</ne-h3> 

在闲鱼上进行低客单价上的引流到私域上进行沟通下单，此方法适用户，账号基数大，已经有点沉淀的玩家进行操作。 

如果账号少，但是收益肯定也是也会受到一定的影响，租客到时间才可以进行下一单的开始，因此我推荐账号数量在 4-10 是比较适合的。 

每天赚百来块钱还是比较轻松的，闲鱼的市场还是较好的每天单号的收益会在 15-35 之间，有的个别的玩家会开挂之类的，可以收取玩家 20-50 的租金，到期退还即可。
