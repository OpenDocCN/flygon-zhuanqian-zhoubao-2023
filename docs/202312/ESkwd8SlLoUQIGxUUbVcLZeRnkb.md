# 什么是快手磁力聚星？

再简单一点，类似直播带货，你只需要推广这个产品，就有佣金。

这个是快手的官方活动，长期可做，不要看利润很低，很多百万粉账号都在做这个项目。

不要小看，很多人培训收398-2980不等，因为很好交付。
