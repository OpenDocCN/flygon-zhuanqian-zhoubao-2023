# 一、工具能力

自动化收集短视频创意，支持 BGM 保存、视频关键词自动提取打标、自动保存至多维表格，当然，也支持去水印。

# 三、需求分析

MCN公司日常编导部门，在创作选题上刷完抖音站内热门视频后，手动保存视频，记录视频分类、数据等并在多维表格中添加相关记录。

*   耗时耗力、重复工作量较高、无法及时记录创意

*   平均记录单条创意时间最快 2min，如遇到下班在家的情况，记录时间变为明天

## 五、更多相关开发心得

防踩坑指南～

*   因抖音获取视频信息的接口为手动抓取，存在接口失效或更新导致集成流失败的风险，以上案例仅做分享～

*   提取的抖音视频大小不可＞20mb，如果＞20mb的情况下需要用到飞书集成平台的分片上传连接器，目前采用的方案是下载低画质的视频。

*   不返回challenge code：集成流先发布再填集成流内第一个节点[网址触发器]里的回调地址，开放平台机器人事件订阅里的key要清除
