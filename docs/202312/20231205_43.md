甜橙 : 1、关于 PDD 卡片问题的解决，可以看这篇文章哦
2、产品方面新人前期可以从 PDD 找，后面可以直接去对接找厂家货源，如果实在找不到的话，可以加入有一手厂家货源的闲鱼团队。（打个小广告哈，甜橙这有三十几个一手厂家货源可以给学员卖）
