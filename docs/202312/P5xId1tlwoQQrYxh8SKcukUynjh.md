# 小红书电脑开播保姆级教程

> 来源：[https://oghxok13aq.feishu.cn/docx/P5xId1tlwoQQrYxh8SKcukUynjh](https://oghxok13aq.feishu.cn/docx/P5xId1tlwoQQrYxh8SKcukUynjh)

# 第一步，打开小红书后台

## https://www.xiaohongshu.com/explore

## 登陆后，点击右上角的电脑直播助手

![](img/c95ce4d4dc9c61f2320ade88413b2801.png)

# 第二步，下载电脑直播助手

## 目前小红书电脑直播助手仅支持win8，win10系统

![](img/933c9a20ee2911ba4c8196efe7f09b02.png)

# 下载安装好以后，打开电脑直播助手，界面是这样的

## 点击左下角的管理平台，进行直播计划创建以及上架商品

![](img/25b77ef9b1093aded04870abc104356c.png)

# 点击直播计划，立即创建

![](img/68c92876dd73cf04d1fcbfe03bba83f9.png)

# 创建计划按自己的实际情况填写

![](img/146c9db09c00373bfa656d830b0a4709.png)

# 创建完计划以后，点击查看

![](img/c04d3684973cceefc2a318a2494018df.png)

# 出现下面这个界面以后，点击添加备货商品

![](img/639c9f5c05364eb5b8f067d1063e5ccd.png)

# 选择好要直播的商品以后，点击确定

![](img/f1d1ccabd41c6772df14c70d51c0e1c7.png)

# 开播后的商品选择就搞定了，备货列表会有你想要直播的商品了

![](img/e02da2640457cccceffabd1fd5491ca6.png)

# 确定以后，回到电脑直播助手

# 设置封面，设置直播标题，选择谁可以看

![](img/ddda9edca5077eec5fed24307665b1b5.png)

# 同意小红书直播协议，点击开始直播

![](img/c1bfc201784dda86aec0ddf0f56a17c2.png)

# 现在就已经在直播进行中了，但是进入直播间，你会发现没有购物车

# 下一步，回到管理平台

![](img/0945b5e6952d5a927409a9e702f218c5.png)

# 点击实时跟播，点击添加备货商品

![](img/795e44f78d494d7e6b24824d927c0b25.png)

# 选择你想要卖的产品，点击确定

![](img/f1aee155135f9749248005dfc54a2098.png)

# 最后点击上架，这个时候，直播间就可以看见购物车了

![](img/35a2519bfd0372536122c617112f0e9e.png)
