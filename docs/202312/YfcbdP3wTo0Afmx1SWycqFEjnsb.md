# 二.项目介绍

百度竞价我是从2021年7月份开始做的，当时还不知道有生财社群，好多赚钱项目都在小圈子流通根本接触不到，要不就是割韭菜的，可选的项目很少，而且我之前做引流动不动被封号，平台改规则很苦恼，一直想找个稳定的项目，百度竞价付费给百度买流量，不存在平台封号改规则的情况，而且只要百度不倒，项目可以一直做下去，综合比较下来选择的百度竞价这个项目。

百度竞价是我们在百度投放广告来卖产品，客户有需求后在百度搜关键词看到我们的广告，点击广告进入我们的网页购买我们的产品，冷门产品是我们找到相对竞争比较少的蓝海产品

目前百度竞价主要分为虚拟资料，生活服务，实物类产品。

上面选品的3个方法同样适用于实物类产品和生活服务产品。

2.选好品之后我们就要进行筛选，下面是我这边的筛选标准

1.首先要有同行在做：同行投放广告在跑的产品，因为广告是有成本的，长期投放的产品肯定是盈利的，除非是测试否则没有人会一直亏钱去投放

2.利润：大于300（比如奥数教程客单价是398，成本是包装2元，U盘20，书20，快递费18，398-2-20-20-18=338是大于300的）因为我们投放是要付给百度广告费的，出一单广告费成本有的产品就要200，搞不好要亏钱，所以我们尽量选择利润比较大的产品去做

3.淘宝销量：大于30

我们去淘宝搜这个产品的销量排行，主要是为了验证这个产品的需求量，销量越大证明需求量越高，如果销量太低或者没销量的不建议选

![](img/e1e5ce1e07ef65b3d8f927cf28407009.png)

4.关键词规划师：大于30

1.https://editor.baidu.com/去官网下载百度营销客户端，然后登录开户的账号密码

2.进入客户端之后先点击关键词，然后点击添加进入关键词规划师

![](img/3cc0d6e280b55596f2d273ae1dde7dcb.png)

3.进入关键词规划师之后为我们输入奥数教程这个关键词，然后点击搜索之后出来数据我们可以看到它的周均搜索量，然后点击从高到低排序

![](img/f26302f4dee6eddb76b56565c8a16b13.png)

4.我们可以看到奥数教程周均搜索量是30符合我们的标准（周均搜索量是指最近一周内网名搜索次数或范围，搜索量越大证明这个产品需求越高）

![](img/7829aca8cea9667f6a7106a20bd5721d.png)

5.竞争对手数量：小于4

我们去百度搜这个关键词，看有几个对应的同行，奥数视频教程搜出来是有3个同行，小于4（同行越少说明竞争越小）

![](img/cc3a042ce1ccf21cd99f57b771ff9997.png)

这5个条件除了利润+同行外，其他3个可以作为参考，不一定全部都要满足，但是如果都能满足的话，成功概率会更高。

创意是展示给客户的广告语,用来吸引用户点击的,一般由标题+描述组成

![](img/b97a33d52bb681c5b8eb56164d3421ac.png)

1.复制同行创意，把同行创意里面的词语句子抄下来，组成自己的创意词库，这是我之前整理的

![](img/650b3e39a85362d1c0cff1965a57e517.png)

2.先点击创意，然后点击添加

![](img/785a5ec7a2cd3de345229b2366fd6e0d.png)

3.填写创意标题，可以抄同行或者模仿同行写，也可以用前面整理的创意词库自己组合，标题要简单，醒目，通顺，突出我们产品的优势。

注意这里要加入通配符（通配符是创意中替换推广组内所有关键词的重要符号，可以增加和网民搜索的相关率和点击率，

使用的方法是“{XXX}”可以替代为组内所有的关键词，当我们撰写创意的时候，创意的标题和描述要插入通配符，插入通配符后，当网民搜索词与通配符里的词相近时，就会飘红。）

标题最好标明价格，这样想要免费的也不会点咱们广告浪费广告费。标题字符不能超过50个。

![](img/5279788b365d5ce951060782ea8d9f26.png)

4.填写创意描述，可以抄同行或者模仿同行写，也可以用前面整理的创意词库自己组合。

描述1和描述2展示的时候是连起来的，每个描述不能超过80字符，所以要言简意赅，语句要通顺，突出我们产品的优势和专业性。

创意建议多写几条，我这边每个产品都写10条左右，因为你不知道哪个创意点击率高，多写几条后期可以把点击率高的留下来，把点击率低的删掉。

![](img/7b6a844ee537e5a49885fcab05cfc558.png)

5.在下方移动计算机访问网址中填入开户商给的网址

![](img/89be1c1b232c72ab755cf2def9d57565.png)

## 5.找关键词

关键词是我们投放的词，比如我们投放了奥数教程这个词，客户要购买奥数教程，他搜奥数教程，那么客户搜奥数教程就会匹配到我们的关键词，系统会把我们的广告展现给这个客户。

找关键词之前我们需要先对关键词进行分析，站在客户角度分析，比如奥数教程，那么想找这款产品的客户是不是可以搜奥数教程，奥数课程，而且客户想要视频形式的，有可能搜奥数视频教程，奥数视频课程。

想买奥数教程的客户，是不是可以搜奥数教程多少钱，奥数课程多少钱，奥数教程怎么买，奥数课程怎么买，奥数教程哪里买，奥数课程哪里买。想学习的是不是可以搜奥数教程怎么学，奥数课程怎么学。

这样我们分析客户需要就可以先整理一批关键词出来，下图是我之前整理的，大家可以参考一下。

![](img/95c558aeb452acb24d8e49bba152cfaf.png)

我这边主要是通过关键词规划师来找关键词的

1.打开百度营销客户端，点击进入关键词，然后点击添加

![](img/7d2e96a5071efc5274c0b6c1c379c7f9.png)

2.进入关键词规划师，用咱们前面根据需求整理的一批关键词来搜索更多的关键词出来。

![](img/b99c30aa455d0460ebdd9e3748ede8cf.png)

3.点击添加全部

![](img/63a5c23d0f88157ebad8c0503175624a.png)

4.添加后咱们接着换关键词继续搜索找到更多的关键词

![](img/c818d6fb88766e5e3a6503f4b9389984.png)

5\. 把我们找到关键词复制出来

![](img/4b499e6ceae2e4c7005eb7d0978bed09.png)

## 6\. 筛选关键词

1.把我们刚才找到的关键词复制到表格中，然后一个个去筛选，去分析每个关键词的需求，把有购买需求的关键词留下来，比如奥数教程怎么买，奥数教程怎么卖，这种有明确购买需求的词都可以留下投放。

![](img/155fdc7e1c33dc9d6e71d9c7bdb6d778.png)

2.把没有购买需求的词删掉，比如下面的有必要学吗？有帮助吗？这种提问的一般还在怀疑阶段，购买的可能性不大，我们广告都是有成本的，这种购买需求不是很强烈的词都可以删掉。还有免费，这种一看就是想白嫖不想花钱的词都要删掉。

![](img/e27213111cb0f96621547769a1246d2e.png)

3.添加否词

否词分为短语否定和精确否定，短否是指我们否定某个词后，包含这个词的关键词就不会再搜到我们的广告。比如免费这个词，我们给他添加短否后，客户搜包含免费的奥数关键词就不会再看到我们的广告。

精否是指我们把某个关键词添加为精否后，客户再搜这个关键词就看不到我们的广告了，比如我们把奥数怎么做这个词添加为精否后，客户再搜奥数怎么做这个词是看不到我们的广告的。

搭建账户之前一般只做短否，精否是跑一段时间后再把不精准的关键词精否掉。

![](img/d14ae3a9d293d39ea2f592d2f7521e7d.png)

## 7.把关键词导入账户

1.打开关键词规划师，把刚才筛选完之后的词复制进来，然后选择计划，单元，点击保存

![](img/0bb8cab816ca321fc72e33b26a3dd88f.png)

2.设置出价，我一般设置1元，后期根据跑的情况再做调整。匹配模式是短语匹配，设置好后点击确定。

![](img/1b3f217388787db60b72a06a39405f8d.png)

3.ctrl+a全选关键词，然后在下方移动计算机访问网址中填入做好的单页网址

![](img/f33c3083e551f1739dc171e8046f29f4.png)

## 8.添加否词

点开推广计划，然后点击否定关键词，把刚才表格里面筛选的短否和精否的词填上，然后点击确定

![](img/f78ab6340b469dcde67601b3e3f80330.png)

点击上传更新，然后点击确定就可以，上传成功后我们的广告就开始投放了

# 五.账户优化

## 1.检查搜索词报告并进行否词

1.每天必须检查前一天的搜索词报告并进行否词，防止垃圾流量浪费广告费，我们点击进入搜索词报告，导出前一天的搜索词数据

![](img/98c9adbf551f0cb04908a0b5f21d288d.png)

2.打开我们导出的搜索词数据，进行否词，每天要做的就是检查前一天的搜索词报告，并进行否词。

去分析每个词背后的需求，比如咪咕视频小学奥数，这个咪咕视频网站是免费的，客户想在这个网站找资源，没有明确的购买需求，不是精准的流量，购买的可能性很小。那我们可以把咪咕给短否掉，下次有客户再搜咪咕就不会搜到我们的网站浪费我们的广告费。

瀚博初一奥数，瀚博是一个培训班的名称，大家遇到不清楚的词可以去百度，客户是找这个线下培训班的，显然也没有购买我们课程的需求，瀚博这个词我们就可以给他短否掉。

通项归纳法小学奥数，视频，这个搜索词客户是需要找奥数里面通项归纳法的方法视频，他只是找某个方法，让他买一套课程的可能性也不是很大，所以归纳也可以短否。还有小学一年级奥数辅导，辅导一般是指线下指导还有帮助，这种购买需求也不是很强，也可以短否掉。

![](img/8137eb239e77c641678cfb121665ec5b.png)

3.把我们筛选的否词添加到账户中，然后上传更新。

![](img/5b261f478917b8d575e93453b1f04014.png)

## 2.关键词调价

我们每周可以检查一下上周的产品消费情况，如果某个产品消费很低的话可以适当的在初始出价1元的基础上上调0.1-0.2这样，观察几天看消费和流量是否有增加。

如果有增加再看下转化情况如何，如果没有增加消费和流量的情况可以继续上调出价，可能有部分产品流量就那么大你上调出价也没有增加消费和流量那这种情况可以把价格调回去。每次出单都要做好数据统计，把出单的搜索词关键词统计下来，这个自己做个表格统计就可以，如果有特殊出单好的关键词和产品可以重点加价投放。

我这边是用200广告费来测试的，消耗200广告费没出单的产品停掉。每天不断的上新产品测试，这样等你上够100个产品的时候最少能有三四十个产品是出单盈利的。我上了大概有200个这类产品测试，最后留了80个在跑的。 教程类产品不可能每个每天出单的，我们主要靠怼量，上几十甚至几百个产品，这样能保证每天都有几个产品出单。

# 六.客服转化

我这边目前成交大概20%是订单系统直接下单的，还有80%是加到微信转化的，客服话术建议大家用小号多去找一些同行，把优秀同行的话术扒下来用就可以，包括朋友圈都可以直接抄同行的，同行就是最好的老师。

客服微信最好是用老号，新号加好友很容易提示风险。收款的话微信会提示风险，收款提示风险很容易丢单，建议用支付宝收，但是好多客户没有支付宝，我这边是用的有赞小程序收款的（有赞一年5000左右，可以正常收款没有风险提示）。或者大家可以让客户预付30，剩下的货到付款，只要预付的货到付款签收率能达到99%。

# 七.新人建议

百度竞价冷门产品相对抖音等比较内卷的平台来讲，门槛比较低，而且不存在与平台对抗封号，平台改规则等风险，相对竞争比较小，适合新人小白入场，项目也比较稳定长久。

项目门槛虽然低，但是对执行力要求比较高，心态也要好。不建议想一夜暴富，三天打鱼两天晒网的人来做。

建议前期多选点品再开始测试，这样能增加拿到正反馈的速度和概率，好多人都是短期看不到效果就放弃了。

这套打法不只限于虚拟教程，新人可以先从冷门产品入手，后期可以尝试去做高复购高利润的私域产品。
