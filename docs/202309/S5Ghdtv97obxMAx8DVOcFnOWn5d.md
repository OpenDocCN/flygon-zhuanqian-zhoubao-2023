# 二、什么是小红书笔记带货

可能有些圈友会问，什么是小红书笔记带货？没听说过啊！

# 三、小红书笔记带货的优势

它相对于之前的小红书无货源模式来说，它不用你去交押金。

不用你去开店，不用你去进货。不用你去垫货款，不用你去请客服，更不用你去发货，这就是小红书笔记带货独特的优势。

而且它也更加的灵活多变。你今天想带A产品，明天也可以带B产品，后天也可以带C产品。这是无货源做不到的

# 四、小红书笔记带货的弊端

毕竟没有哪个项目是完美的，当然它的劣势在于它的 门槛

首先我认为它最难的一个门槛，就是 账号要满足1000个粉丝以上 。因为单单这点，对于很多新人来说，是很难做得到的。

包括我在这上面也吃过了很多亏，甚至还被骗了300块，这个待会我再给你们说

第二个门槛。就是要 实名

第三个门槛，是这个 账号无任何的历史违规记录 ，记得，是无任何历史违规记录！！

其实后面这两点都还比较好解决，唯独就是在短时间内能得到一个千粉账号，是非常困难。

不过没关系，这些在后面我都会一步步的教你去攻破它，学完之后，感觉真的很简单

## 1.怎么解决千粉账号？

相信这个问题，也是很多人都想知道的问题，毕竟你想用一个新号，在短时间内瞬间涨到千粉，这是非常困难的

当时我也考虑想通过买号的形式，来解决这个难题。

然后我在淘宝上问了很多家，基本上呢，都是一元一个fen，也就是说，如果我要把粉丝shua到1000个的话，那么我的这个账号成本就要花1000块钱了

因为我做项目有一个核心的理念，就是一定要低成本，如果这个项目实在是用太高的成本去做的话，那么我是绝对不会去做的。

当时放弃了这个淘宝shua粉的玩法后，我又去尝试过用搞笑素材，擦边素材，用网红明星素材。

甚至用鸡汤素材去实操，全部都以失败告终，因此还被封了2个号....

想要在小红书上短时间内积累到1000粉的这个门槛，是非常非常困难的，于是我又想到了另外一种方法，当时我也不知道这个方法能不能行得通

那就是直接买别人的账号来使用，我也想过买卖账号是违反平台规则的，如果一旦被抓到，那账号可能就会被永封不得申诉了

想了想当时没其他的办法了，最后还是决定想去冒这个风险试试.....

当时我在小红书上面搜索：“小红书闲置”“小红书号”等这些关键词，然后去他们的每条笔记下面去留言，话术说：“谁有千粉号，我收，我高价收！”，然后第二天就引来了20+的人，私信说要卖号给我

当时我一看，哇那种感觉很兴奋，像是看到了希望一样。

我觉得只要价钱能低于500以下的，我都能接受了，记得当时有个人是中介，他就是专门倒卖账号的。

我问他，这个千粉号卖多少？他跟我说是一个300，我当时犹豫了一下，很想买的。但是后面又有一个倒卖账号的人出的是230一个。

嗯...当时我在想，既然价钱都不同，那还能不能再降降？后面我又继续用了半天的时间去找。

去留言找各种的小红书闲置账号，然后我看到留言区里面，很多人都说120出此号

然后我看他们的账号也正好是千粉的。我在想，既然这个价钱一降再降，那还有没有可能，再降得更低呢？

于是我就有了这种好奇心，打算用更低的数字来试探下他们的底线是多少

当时跟他们聊天挺好玩的，我给你们模拟下当时的私信场景

没想到真有一个人给我成交了。我当时真的是非常的是兴奋啊！！

因为从最开始1000元的shua粉，到后面心理价降到500，再从500降到300，再从300降到120，再从120降到50，我觉得我应该要学会知足了，后面我就用这个账号，开始了小红书笔记带货的搞钱之路

在这里我想跟大家说一下，就是你们去买账号的时候，一定要找第三方交易平台，千万不能你加了他微信后，你就提前把钱转给他了，在这之前因此我还被骗了300块，这个就不说了。

所以一定要记得，加微信后，要先走第三方平台去做交易，比如像转转，闲鱼这种平台。

然后交易之前，必须要跟他说明一点，就是此账号如果三天内有任何的违规记录，那这个账号我就要退回全款的

如果三天后有任何的违规记录，这个就不关你的事了，如果他能同意这一点，那我们就交易，如果说不同意的话，那我们就取消交易。

因为官方明确有一条门槛要求，那就是不能有任何的历史违规记录，比如禁言，导流，包括你发布一些平台禁止的违禁词等等。

总之有过历史违规记录的，都不行

因为这些违规记录是没办法查询得到的，所以只能通过你去试用这个账号后，你才能感受得到的。

这一点是非常重要，希望各位圈友们不要去踩这种坑。真的要切记！

ok，当我们在第三方平台下单付款后，我就让他在账号那修改换绑成我的号码。换绑成功后，他那边就会自动掉线了，我这边就马上登陆账号。

登陆上之后发现没什么异常，我就开始不动它了，然后在这一两天之内，啥也不要做，因为这个账号异地登录之后，如果马上进行频繁的修改资料或发布笔记的话，你的异常行为，会引起平台系统监控的。

什么是异地登陆呢？

简单给你们科普下，就是上一秒这个账号还在北京登陆的，下一秒就直接到上海登陆了，这种异常行为任何平台都会给你监控起来的

所以我在这个时候，我也不敢去动他，静静等待它在我这台手机上躺两天。

两天过后，我才开始慢慢的去操作，比如像修改资料，把之前的笔记都删除等等。

当账号的问题搞定了之后呢，我就开始去选品了，当时我也不知道要选什么样的产品，也不知道什么样的产品会比较好卖。

按照了这个思路，我继续思考着，既然小红书里女生占比很大，而且都是一二线城市的，要不...我搞个女生成长类的书试试？

既然已经确定了品类是什么，那接下来呢，就是要去找同行的对标账号了。

然后我就搜索了“成长图书”这个关键词，点击最热。找找看有哪些是挂了链接的笔记。

## 2.为什么我只找挂链接的笔记呢？

这是个非常重要的小细节啊，因为在这个关键词下，有很多都是单纯分享图书类的笔记，没有挂任何链接。

那就意味着，在同等条件下，没挂链接的肯定比挂了链接的流量更好。

因为它是不带有任何商业属性的，但如果带了商业属性，那它的性质就不一样了。流量肯定会有所影响

所以我的笔记一开始，就是奔着商业属性去的，这样目标也就非常的清晰了。

就是要找带有链接的热门爆款笔记，来作为我的参考对象。

我找出了十个对标账号来参考，分别参考了他们的封面，标题和正文，然后顺便看看这十个对标账号里，他们用的是图片，还是视频来发的。

然后就可以开始光明正大的“抄”了

# 六、注意事项

首先第一点，你刚开始发布笔记的时候，是没有任何流量。

这是很正常的，因为你这个账号在异地登陆的情况下，这种异常行为会被平台系统给检测到的，所以会给你1-3天的隐性限流。

流量会很低，我记得刚开始发第一篇笔记的时候，小眼睛只有39

那要怎么解决这个问题呢？

很简单的，我连续发三天笔记后，这个浏览量的问题才得到解决。这点不用担心，多发就没事了

那么第二个问题呢，就是你们要发布笔记的时候，不要在短时间内，频繁的去发布。

比如你刚发完上一篇笔记对吧，那么你下一篇笔记的发布时间，至少是相隔一个小时以上的。

因为你在短时间内频繁的去发布笔记，很容易被系统识别到异常行为，一旦被平台判为营销号的话，哪怕后面你的内容再优质，账号流量也是起不来的

第三点，当你发布图文笔记的时候，发现没什么流量了，我可以简单的教你一招把流量给拉起来。

就是可以把你创作好的图片，建成一个音乐卡点视频。配上温馨的背景音乐。这样的流量会更高

而且现在小红书平台会把更多的流量，倾斜给到短视频这边。如果你有一点点的剪辑基础能力，不妨可以去试试。

第四点呢，是我废了十个小号，才总结出的一个道理，这条至少值1000块吧。

注意听哦，就是你们用小号去留言别人卖号的笔记时，一定要非常注意一点，就是不能出现有“卖号”，“买号”，“交易”等这种关键词，因为买卖账号是平台明令禁止，也是打击最严的一块

因此我也被封了有十个号吧，现在我那十个号的手机号码至今都没办法要回来

而且账号被封是永久封，记住，是永久封！！

那我当时怎么缓解这个问题的呢？

当我被封完第二个账号后，就意识到这个问题真的很严重，如果再不优化话术，再多的手机号码都不够用，于是我便重新调整了一遍话术，改为：“xhs千fen，私我”

这个话术是当时测试了很多遍，都没被封的其中一个

这样发出去后，如果他们能懂啥意思就好，不懂的话，我也不去做过多的解释。就怕有人来点个举报，那就麻烦大了，小心驶得万年船啊！

第五点呢，就是去找千粉账号的时候，千万千万不能去发，带有任何买卖账号内容的笔记。

如果你一旦发了，被平台系统识别到，或被别人举报，那你这个账号就直接被永封了，痛苦的是，你连申诉的机会都没办法申诉

当时我有两个账号，也是因为发布了买卖账号的笔记后，被搞到永封的，我也尝试过给官方薯队长发邮件，连发了有好几条吧，对方看了我的账号都不受理。

所以我就知道了这个事情的严重性。你们千万可别继续踩这个坑啊！！

第六点，不要去跟中间商买号！

因为当时我不太懂这里面的规则嘛，也急于想要把这个项目给跑通吧。然后我就去跟了一个中间商去买了一个账号，他说要300，先转再给账号我。

当时我半信半疑的转完给他后，他就直接把我给拉黑了！真的，这点让我感到智商有被侮辱到！虽然钱不多，但真的有点被气到

所以在这里，我真心希望你们不要去跟任何的中间商做交易，因为他们实在真的太坑了，坑了我300块...300块啊！都够我吃好多碗螺蛳粉了...

那可能你会问，我做这个小红书笔记带货，一天能赚多少钱呢？哈哈，终于来到你们最关心的问题了

我当时一天单个账号，得到的佣金也就是50~80之间吧。真的不多，想爆赚的朋友，在这可以劝退了。但可以当副业去做做。或者打包成项目教人去做都行的

我们当时矩阵了十个号，做了快一个月，最后十个账号结算下来，也就得了一万八这样。

我不建议你们把这个当成主业去做，可以利用下班后的业余时间赚点零花。

除非你现在全职在做自媒体，手头上正好也有很多手机和号码，那么你批量去矩阵的话，一天赚个五百八百的，问题也不大

但跟大家说的是，这里面的风险成本还是比较高的，因为在你找账号的时候，一不小心会面临封号的问题，其他的就没什么了
