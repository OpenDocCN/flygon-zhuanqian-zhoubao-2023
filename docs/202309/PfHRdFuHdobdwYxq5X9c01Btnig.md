# 一：开通优选联盟

## 1：开视频号小店

视频号小店-【开店】指引 按照教程填资料就可以了。

视频号小店类型分为「官方旗舰店」「旗舰店」「卖场旗舰店」「专卖店」「专营店」「普通企业店」，建议开通「普通企业店」，「普通企业店」不需要品牌授权，大部分类目都可以上架售卖，性价比最高。

## 2：开通优选联盟

微信视频号「优选联盟」准入及清退规则

「官方旗舰店」「旗舰店」「卖场旗舰店」「专卖店」「专营店」「普通企业店」商家需同时满足下列所有准入条件才可准入；暂时没有视频号店铺评分数据的新商家只需满足下列准入条件（1）即可准入。

准入条件：

（1）视频号小店处于正常经营状态；

（2）商家的视频号店铺评分>=4.2

（3）商家近30天商责纠纷率<=0.15%

现在视频号小店开通优选联盟没有门槛了，新店铺没有评分的可以直接开通优选联盟，老店铺需要店铺评分>=4.2

商家近30天商责纠纷率<=0.15%。

注意：跟抖音不一样，视频号小店，个体工商户类型商家暂时不予准入视频号优选联盟。

## 3：店铺进入优质商品池

现在视频号小店进优选联盟后，只能被4.2分以上的达人带自己小店的链接。

没有评分的新达人以及评分低于4.2的达人只能带优选联盟【优质商品池】内的链接，怎么进优质商品池呢？

优选联盟达人带货管理规则

![](img/5599ce2f66f338f1abbcbc7de5e16dda.png)

可以看到，官方规则要求【视频号店铺评分>=4.2分】（一般30单评价可以出评分）、【公域结算订单大于100单】（要通过公域流量成交的订单，确认收货7天后结算完成才能满足），这两条都可以通过刷单来完成。

#### 刷单流程：

店铺上架一个五六块钱的链接，联系好鱼塘（官方要求公域结算订单大于100单，最好做150单以上，防止部分订单被系统稽查掉），用店铺主账号绑定的视频号开播（新店主账号绑定的视频号才能挂店铺链接，主账号的订单才会被统计到结算订单中），开播前先让鱼塘搜索主账号的视频号，开播后通过直播广场进去，直播间会在鱼塘号的直播广场首页，进直播间后购买，这样就会被计入到公域流量订单中。发货可以发礼品，等物流信息签收后，让鱼塘及时确认签收，及时给好评，再等7天结算后就可以满足【视频号店铺评分>=4.2分】【公域结算订单大于100单】

# 二、精细化选品玩法

在视频号小店上，铺货玩法基本上没什么用，因为达人最主要看重的是商品的价格和佣金，价格决定达人好不好卖，佣金决定了达人能拿多少钱，直接铺货的链接可能已经有人以更高的佣金更低的售价在优选联盟上了，所有铺了链接也不会有人带。我们要把重心放在选品上，选品定生死。

## 2、那么有什么技巧可以帮助我们快一点达成合作呢？

每一个能卖的出货的达人，每天会需要对接很多厂家，手机一直都是很多人主动联系的状态，所以如果你想在这些厂家和品牌方中获取达人的关注，那么就要有技巧地去联系达人。

很多人加到达人以后，都是发“你好”，然后就等着对方的回复。结果日复一日都没有得到对方的回应。我们跟达人合作，一定要体现出自己的专业性，打招呼简单明了，突出事由。参考话术：“你好，我是xx品牌创始人，这是我们品牌的主推产品，可寄样”同时将产品的介绍以ppt或者是货盘的方式发送给达人

如果你还想着等对方问你要手卡再给的话，效率会慢很多。即使你现在发出去了，对方没有第一时间回复，后续他们也会看到，不至于遗忘。

1：提醒样品到了，但是语气亲切点，没上链接的提醒尽快上链接

### （2）如何提高达人回复率

1：从达人的利益需求角度去讲：价格性价比高，佣金可观，产品需求大，发货速度快，质量好服务有保障。把最突出的的亮点一定要让人看到

2：从他最想带的、最适合他的品入手，作为切入口

### （3）如何判断达人是否匹配所推产品

1：粉丝画像和目标人群对比

2：客单价是否和产品定位匹配等

3：产品热度周期和直播频次

4：达人对这类产品的了解程度，过往类似产品的销售数据
