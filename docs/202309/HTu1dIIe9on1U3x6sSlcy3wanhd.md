# 38岁中年妇女，面临降薪、裁员，我是如何依靠ChatGPT走上自救之路的

大家好，我是菜菜，85后，2胎妈妈，目前在一家互联网教育公司做内容运营。按照大多数女性的路径，我这种年纪、这种需要牵扯很多精力的身份，应该在事业上已经到头了，把自己的家庭经营好就是一种修为。然而，我是个不甘心平庸的人，今年2月，ChatGPT的风潮在中国席卷而来，我懵里懵懂地加入了这一浪潮，最终利用自身的特长——内容创作与ChatGPT的结合，成为了一名ChatGPT内容创作指导老师，目前也是三节课平台的特邀讲师，累积已帮助3000+学员提升内容创作的质量和效率。

# 01 种下焦虑之根：被嫌弃的幼年

其实能有这样的突破，我反而要感谢我不甘于落后的焦虑感，这种感觉其实是刻在我骨子里无法抹去的烙印。为什么会这样呢？有这要从我的小时候的经历说起。我爸爸是家中的长子，自然我也是爷爷奶奶家的长孙，在那个年代，老人们还是传统的重男轻女思想，我被生下来的那一刻，据说我的奶奶是十分失望和遗憾的，我记得我奶奶多次和我说过：要是你是个男孩就好了。

我有很长一段时间都是跟着爷爷奶奶，家里除了有我们仨，还有尚未结婚的叔叔和我的表妹（我姑姑的孩子，我姑姑是爸爸的亲妹妹），表妹从小的性格就大大咧咧、活泼调皮，她扎着两个可爱的羊角辫，脚上永远蹬着蹭亮的皮鞋，穿着可爱的花裙子，像一只蝴蝶般，无论到哪里，都吸引着长辈们的目光，受夸奖的也永远是她，引得长辈们连连欢笑的也永远是她，这也让我十分羡慕和嫉妒。那时的我，怯懦而自卑，因为我时常穿着大人不要的、被改小的旧衣服，我已经很多次和大人表示，我不想再穿那样的衣服了，但是在奶奶家的那几年里，我只记得唯一一次穿过新衣服，就是我二年级成为少先队员的那天。

![](img/17699f392ee534e42774b0ee5db95510.png)

为了能和妹妹“争宠”，我开始琢磨如何能讨得大人的喜欢，可能在那个时候，就爱上了“竞争”，基于我的奶奶多次提及“你要是个男孩就好了”，并且夸奖我妹妹像男孩调皮，于是我慢慢也开始模仿男孩，慢慢地就不喜欢穿裙子，不喜欢留长发，在行为举动上也刻意模仿男孩的野性和洒脱，以此希望获得家中长辈的关注。可是有一次弄巧成拙了，不但没有得到好评，反而让我的心灵蒙上了一层阴影，让我愈发变得渴望得到外界的认可。有一次我和妹妹打闹，我模仿电视剧里大侠的“英勇”，用夹煤炭的火钳敲在妹妹的头上，她大哭起来，头上也起了一个包，我惊慌失措，奶奶知道后，说了一句让我心碎且窒息的话：“你还是一个小孩子，怎么心这么毒啊！”在后续很长的一段日子里，奶奶多次在家人面前提起这个事，反复说我“心好毒”，也许当时只是大人间的闲聊，但让我长久都伴随着一种羞耻感，我讨厌自己，为什么是个女孩，为什么妹妹随便做个动作，奶奶和叔叔都那么喜欢，而我无论做什么，都得不到他们的认可。我成年后，有一天看到那部著名的日本电影——《被嫌弃的松子的一生》，我把自己关在房间里，崩溃大哭了3个小时。

# 02 意外惊喜：发现写作天赋

这种希望得到外界认可的渴求，在我小学三年级的时候，得到了充分释放。那时候已经从奶奶家出来了，来到了城里的父母身边，妈妈给我买了很多课外书，我爱上了看书，喜欢看童话故事、小学生作文、历史故事……我爱那些文字，在我的身体里，似乎也有着一股蠢蠢欲动的力量，在冥冥之中召唤着我，让我把那些情感、那些想法写出来。那时候，很多孩子都苦于如何写作文，但我能不费吹灰之力，就能在十分钟之内写好一篇作文，我总有表达不完的话，家里买了一瓶酱油、在路上看到一朵盛开的小花……我都能写出一篇优秀作文。语文老师给我的极佳评语、批改优美文字的那些红圈圈、每次上作文课，老师都会当众表扬我，频繁地念我的作文，参加市里的小学生作文比赛，能轻松拿到一等奖……这些都让我有一种前所未有的满足和自信。也正是这个时候，看书和写作成为了我对外展示美丽的利器，并像基因一般，深深地植入我的身体。

# 03 敏感的少年期：改变了我一生轨迹的男同学

虽然依靠天赋，我有了自信，可来到青春期，我又对自己产生了怀疑。青春期胃口好、饭量大，于是脸盘子日益圆润起来，同时没有人教我如何洗护脸蛋，我脸上长了难看的痘痘。再加上没有注重体型维护，有了驼背、脖子前期不好的体态，班上有一些男同学给我取了一系列难听、带有侮辱性的外号。每次下课，我从他们身边经过时，他们就会站在走廊上阴阳怪气地笑我。有一次，因为某位男同学的事件，改变了我的人生轨迹。注意，这不是一个爱情故事。

我还记得那是高二的一天，我和我的男同桌在一旁有说有笑，班上经常笑我的那名男生，又像往常一样和其他男生阴阳怪气地笑我，甚至还说了一些对我来说，是十分伤自尊的话，我决定以其人之道还治其人之身，第二天，我写了一封信，扔在了那名男生的抽屉里，我模仿了他平时羞辱我的话语，添加了对他外貌的描述，把他骂了一顿，他看了之后十分生气，当下揪着我的衣领，要打我，而我也不甘示弱，我骂了他，他先对我动手，然后和他打在了一起。当然，我是个女生，我是打不过他的，我觉得我的脸丢尽了，小时候那种强烈的羞耻感顿上心头，我脑海里像放电影一般：小时候被“嫌弃”，长大后被同学羞辱，我不能再这样下去了，我发誓我要出人头地，我不能再被人欺负和笑话。我记得我当时捂着被打红的脸，我一字一句、铿锵有力、并用最凶狠的语气和他说：XXX，你给我永远记住今天，你今天横没什么了不起，我正式向你发起挑战，到时候高考成绩出来，我看咱们俩谁是彻底的失败者。

自从那天之后，我就像变了个人一样，不再参加好朋友的课后活动、把最爱看的漫画束之高阁，开始没日没夜的学习，我发誓我要用一年时间，冲上班级的前十名（当时班上近50人，我在班上排三十名左右）。于是，白天，我开始认真听课，每一个上课不懂的问题，我都会在下课去找老师问清楚，给老师打扰到什么程度呢？，每次下课，我都会第一个跑到老师办公室，以致于我的很多任课老师看到我，又好气又好笑：“怎么你又来了，我这刚下完课，水都没喝。”晚上，我就拼命地在家中做题、背各种知识点，常常努力到凌晨三点，实在困了，我就拿锥子扎自己的腿让自己清醒。我让我妈听我背课文，及时纠正我哪里背得不对，由于时间漫长、过程极其枯燥，我妈第二天又要上班，常常求着我说：“我给你十块钱，你去买点好吃的，别让我给你背了，好吗？”

有时候确实题目做不出来，很痛苦，我就写日记，在日记本里反复地告诫自己：我一定要考上大学，我一定要赢过那位男同学。我曾经无数次地幻想，高考放榜的那天，我要写一封信，当众读出来，在全班同学面前取笑他，我要去他家楼下，堵着他，笑他是个失败者。在很多女孩的日记里，通常都是在描写自己暗恋的对象，我却一直在写当时我最恨的人，现在想起来，也让我觉得哭笑不得。

就是靠着这样的自我强逼和“变态”的动力，我居然喜欢上了学习，我极其享受那种攻克一个又一个知识点，成绩飞速上升的感觉。

后来，我如愿地考上了本科，我的老师都惊叹，我是个奇迹，居然通过一年时间，就恶补上来了，同时也惋惜：以我的学习天赋，如果我能在高一的时候就努力，那么现在应该考的不是二本，而是妥妥的985、211了。

没错，你们猜的没错，那位男同学没有考上，读的是大专还是三本，我已经不记得了，至于在最后的班会上，我有没有羞辱他，我也不记得了，当我实现了自己的承诺，我突然释怀了，我对他没有了恨意，反而有一种发自内心的感谢，如果不是他的欺负，我也不会自强自息，获得敲门砖，开启后面的精彩人生。

# 04 做自媒体、高光的12年

## 1.四年的传统媒体生涯

因为渴望得到聚光灯般的认可，以及喜欢写作，很自然的，我大学读的专业和这两点息息相关——广播电视新闻学，大学毕业后，我就参加了长沙电视台招聘考试，并顺利地进入了电视台工作，一呆就是四年。体制内复杂的人际关系和条条框框的限制，总是让我不能畅快淋漓地发挥自己的想法，那四年，我虽然也获得了一些认可，但过得并不开心，甚至有些憋屈。而这时，传统媒体也逐渐进入衰退期，新浪微博顺势而起，我那时每天都会逛微博，身为记者，也经常会接触到一些明星资源，我会利用这些资源，在微博上和用户互动，或者写一些感悟和心得。

## 2.初入自媒体行业

偶然得知新浪在湖南建站，招第一批员工，于是我又报名参加了招聘考试，由于多年的媒体经验，我对于内容和用户有非常敏锐的洞察力，凭着优异的成绩和独特的网感，我顺利进入了新浪湖南，也就是从这里正式开启了我12年的自媒体生涯。 那时候，我真的是好多朋友艳羡的对象，能到处去吃喝玩乐，还能近距离接触大牌明星。

## 3.被人民日报转载

从微博出来后，我进了一家不到50人的电商公司，当时之所以想进那家公司，是因为想跟着大佬学东西，面试我的领导，是大家所熟知的RPA知名导师@阿愚呱呱，他刚从深圳回来，当时听着他在深圳如何通过做一个网站，把一家珠宝公司从0做到营收几百万，我惊呆了，原来互联网可以有这种骚操作，当下也不管公司的实力和开出的工资是不是符合，下定决心要进入这家公司，跟着大佬学本事。

后来证明我是对的，在阿愚呱呱的鼓励下，我开始利用自己之前做媒体、做微博的经验，组建起新媒体团队，开始做公众号。公司的业务是卖海外保健品，如果在公众号直接宣传保健品，是有风险的，于是我们走曲线救国的路线，做营养健康知识。当时公众号刚刚兴起，关于营养健康领域，一般会有2种不同类型的内容，一是家族群里经常会有长辈转发的各种耸人听闻的健康保健知识：x种吃了必得癌症的食物、还不快扔掉家里的这种食物，误吃你会后悔一辈子……或者是医生的专业科普，写的每个字我都认识，但是就是看不懂。难道就不能有专业的、但是又容易下口的营养健康知识吗？

直到我接触到“混子曰”这个公众号，用好玩有趣的段子+漫画讲生动地讲解各种生涩知识，我一拍大腿，这就是我要找的，于是我就带着新媒体团队，做了一个漫画科普营养健康的IP，当时我们还引进了专业的医学编辑，她负责专业科普知识，我和另一个文案小伙伴符合把晦涩的知识翻译成大白话，并加上各种网络搞笑梗，再请我们的设计师伙伴，画成漫画。一推出来短短几个月，就积累了几千粉丝，我凭借自己之前在传统媒体、互联网媒体的链接能力，联系各种大V公众号进行投稿转载，渐渐地，我们的内容被人民日报、科普中国、共产党员、人卫健康等大号转载：

## 4.初步试水课程研发+培训

可是好景不长，虽然我们的公众号一直做得有声有色，但那时公司的经营状况不行了，从隔三差五的迟发工资，再到后来直接发不出了，而当时我们的账号还远达不到能变现养活自己的程度，后来生活所迫，我离开了公司，跳槽去了一家大型医院，团队也逐渐解散了。

其实去这家医院也是做了妥协的，我家距离这家医院有25公里之远，每天上下班要在路上耗费近2个小时，而且我不会开车，搭的是同事的车，每天来回车费也要大几十，一个月下来，是笔不小的开支，再加上这家医院是单休，让我十分不习惯。因为在以往的工作经历里，我从来没有干过单休的工作。可是当时被生活所迫，又羡慕这家大型医院的光环，我还是硬着头皮去了。

在这家医院，我再一次做了突破，干了一个之前想都不敢想的事——培训医生，帮助他们打造自己的自媒体IP。我和团队伙伴依照我们做漫画IP的路径，做了一整套医生IP培训课程。第一期培训结束没多久，就有3位医生顺利出圈，多次拿到了今日头条青云奖，而在但是，医院还没有开业，我们就在网络上为医生积攒了一波人气，这让团队十分受鼓舞。

## 5.课程制作+分享变得轻车熟路

后来由于实在是路费成本太高、身心疲惫，干了一年后，我辞职了，去了一家TOB管理SaaS公司，在这里一干就干到现在，到今年已经是第四个年头了。

在这家TOB公司，我又一次向自己的能力边界发起了挑战，由于服务的是培训机构，所需要的内容要更加专业，而我之前写的内容都是给普通受众看的，如何能顺利过渡，战胜困难呢？于是我开始恶补各种教培行业知识和软件知识。在学习的过程中，我发现2点，第一我以往所学习的互联网运营知识与它们是可以结合的；第二，要宣传软件，一定要结合客户实际的工作场景去撰写内容，这样才能打动客户。可是作为一个从来没有从事过教培行业的人来说，我如何获得这些场景呢？去和销售、客服沟通，也只能管中窥豹，我只有真正去接触用户，才能获得这些场景。于是，我又走了一个曲线救国的路线，在行业的垂直平台上，去发布互联网运营结合教培行业的干货，以此吸引精准用户加到我的微信，我们双方达成互惠互利的合作关系，我来给他们做互联网运营的知识分享和指导，他们来给我提供与工具能结合的实际工作场景，就这样，写着写着，我从写单篇到开始写连载专栏，我写成了平台的专栏作家，还时常被邀请去一些付费社群如运营研究社、野生运营分享干货知识。

![](img/7fa55c6172866559525bd1ead14feff3.png)

基于我的内容含金量很高，有很多用户加了我，向我取经。正是因为在医院给医生做培训、在平台分享知识这两段“为人师”的经历，让我找到了那种在“聚光灯”下的感觉，也为我后来从事ChatGPT教学工作奠定了坚实的基础。

在这家TOB公司，我也主动学习短视频和直播，利用业余时间，为公司打造抖音号，录制了一系列视频，以及自己研发教培干货课程，在其中融入工具的介绍，连续40天，每天晚上直播2小时，通过那段时间的锻炼，不但给公司带来了不少0成本的线索，并成功转化，同时还克服了自己在镜头面前的恐惧感，谈吐自然，分享时轻松自如。

本来一切都还顺利，但2021年双减政策出台，给了我们公司以及我个人不小的打击，减员、降薪、行业里的一片哀嚎，再加上我年过35，已经到了职场瓶颈，一度我非常焦虑，整晚睡不着。

# 05 为我量身打造的职业：ChatGPT内容训练讲师

2023年2月，我在生财有术第一次知道了ChatGPT，我从焦虑转为亢奋。因为在我以往的思考中，我就在想，能不能有一种工具或者程序，它是无需使用者有编程基础，只要与它对话，它就能按照我们文字的意思，去自己建造一个程序，以方便地代替我们完成各种工作，我突然发现，居然有人把我的想法做成了现实，于是我率先去淘宝找人帮我注册了账号，开启ChatGPT的使用。

我发现，我好像与它特别熟悉，凭借着多年的媒体从业经历和内容写作经历，我似乎天生就知道该如何和它对话，来指挥它帮我完成各种任务，在我过往的工作经历中，那些积累下来的、成熟的文案底层方法论，也能结合ChatGPT完成，于是我开始在朋友圈分享我的各种ChatGPT训练案例。

![](img/778f55a4c5442606b3aa5dfca964443d.png)

在收获了朋友圈很多好友的关注后，我渐渐发现，很多人对于如何用上ChatGPT是有较大门槛的，于是我批发了一些账号，便在抖音边做直播边卖账号，和其他同行不一样，他们只是展示自己如何和ChatGPT对话的，而我是做直播连线，直接帮助用户解决一个又一个问题，向他们展示如何通过向ChatGPT提问，再耐心调教，最终得到自己想要的内容，大家都觉得很神奇，直播间的人越来越多，也引得不少同行来围观，他们还在我的评论区打广告，让我哭笑不得。就这样，我前期靠着卖账号，赚了几千元。

后来，随着卖账号的人越来越多，竞争激烈之余，也带来了封号的风险隐患，于是我思考该转型了，我并不擅长解决账号、技术问题，但是如何和ChatGPT对话，如何融入文案的生产方法论，让它帮我们创作出惊艳的内容，这是我的强项，而且，在前段时间的直播答疑中，我洞察到当时大多数人用不好ChatGPT，是根本不懂它的原理，只会直抒胸臆地表达，导致ChatGPT只能靠盲猜输出，于是，我决定开始做ChatGPT内容创作教学，我自己也总结了一下，我做教学有如下优势：

1.我有多年的采访经验，我擅长提问题，同时，我能根据对方的回答，洞察到他可能对于我问题的理解有什么偏差，接下来我该如何调整，这些正好能完全适配在和ChatGPT的对话中；

2.我通过多年的自媒体经验，掌握了大量的内容生产方法论，我可以把这些方法论教给ChatGPT，让它做我的助手，去写出和我人工产出时，质量差不多的内容，而这个技能，对于自媒体初学者，或者是完全没有内容创作经验、但又想做引流变现的人来说，是非常非常重要的；

3.我通过前面几份工作的积累，有人脉，也有能力，人脉在于我认识了许多社群群主，他们擅长做引流，而进入他们社群的学员是有学习需求的；在能力方面，我的课程研发能力、讲课能力、答疑能力也、社群运营能力也都还不错，基本上可以一个人活成一个团队。
