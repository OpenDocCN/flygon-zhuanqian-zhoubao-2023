# 二手手机行业前景

手机，我相信大家都不会陌生。

手机作为当今密不可分的工具，人的吃穿住行都需要依靠手机。

那么这个行业前景怎么样？

经常和同行聊天，会听到一句话。

大概意思就是生意越来越难做了，二手手机不如以前赚钱了。

不可否认的是，目前的二手手机行业利润当然是大不如前。

毕竟确实也过了红利期。

可能大家对此没有概念，我给大家加深下概念。

二手手机前几年，利润率是在70%左右，现在多数15-30%这样。

当然也有办法提高利润率，主打的就是一个信息屏障以及增值服务。

那么为什么即使利润率如此低，我还是坚持这个行业，并且深耕。

主要的原因我从三个点给大家分析。看完后大家别忘记点赞评论，谢谢支持！

### 第一点：为什么人会买二手手机产品

买二手产品的会是那些人群？

1.刚需人群

2.中低产阶级

3.工作室设备

4.个人置换

5.学生上缴、备用

6......我就不一一说下去了，分类下去就太多了。

那么这些人都会有一个核心点“省钱”。

不管任意以上哪一条踢出来都是为了省钱而进行的动作。

从疫情开始那几年，我的二手手机生意就暴涨，很多实体行业都倒闭了。

原因还是因为，我不做实体。

而我又是怎么做到生意暴涨，等后面我给大家说说，大家就能理解了。

### 第二点：为什么人会卖二手手机产品

说这一点之前，大家可以思考下。

卖手机需要什么？

我给大家列举一下。【排除一些知识积累】

1.流动资金

2.人

没了？对没了，对于冰山上的必备条件就是这些。

而对于冰山下的，比如懂手机、会维修、有更好的供货源等等，那些是需要时间去开采需要熟人去带着熟悉的东西。

而这些东西，只需要用时间去填补，所以这就是为什么这么多人可以去做卖二手手机这个行业。

同样也是为什么，我一直认为我们这个行业不太具备太复杂的知识储备与需求。

单从一个项目来说，二手手机这个行业。

可以低投产、可以高投产，但是他的利润低，环节简单。

无非就是低买高卖，不限专业技术、不限年龄、不限知识。

他的闭环也很清楚。

那么对于这类高资金、低利润的产业他的壁垒又在哪里？

感兴趣的可以评论区讨论。

当然总结上面的信息咱们可以得出，做二手手机并不难。

与此同时是否仅仅只是入行并不难？大家思考下。

玩任何项目的时候，不去看别人赚了多少钱，而只是看这个项目逻辑是否成立即可。

因为经常就像我说的，别人赚多少钱跟我毛关系没有。

因为我做不到，但是同行业的人，我能在闲鱼一个月赚10几万，他们做不了。

但是这和项目本身是没有关系的，操作差异是平台和人而已。

### 第三点：二手产品的行业使命是什么

为什么这一点我没有去说，二手手机产品的使命是什么。

却是说的二手产品的行业是什么？

首先，大家不用局限于手机，二手产品其实目前也是属于上升期间。

对于这个不太算热门的类目，二手产品的推进其实也和国家政策息息相关。

首先第一点，根据北京环丁环保大数据研究院的核算，一部二手手机重新流入市场，可以减少68.44kg的碳排放，

相当于一辆燃油乘用车行驶390公里的碳排放量。

废物利用，减少碳排放，为全球变暖缓解做出自己的一份力。

这句话不仅仅是口号，也是目前很多国家都在推进的事情。

像前几年，我就经常刷到视频，国家鼓励大家坐公交车鼓励大家少开车。

减碳行动一直都在，不过年轻人的关注点。更多的是在满足自我快乐。

那么我们做二手行业的使命是什么？

顿号这边说下自己的看法。

作为二手行业从事者之一，我觉得我对于社会的使命感，就如我们行业的巨头爱回收曾经说过的一句话。

万物新生，让二手手机没有闲置。

归根结底的，还是将一个废物利用，减碳落地。

我相信在未来，关于环境、绿色、健康、二手行业会是一些热门小赛道，包括我自己最近也在分析。提前布局

当然玩法就不和现在一样了。

那么对于我们这类小商家来说，其实可能大部分同行也不太关心世界的发展。

更多的就是，低买高卖就是，但是顿号的想法不一样。

今年顿号看了不少项目，特别是实体这块，二手行业还有很大的上升空间。

可能后期我的打算也是深入了解，不止是数码的二手还有其他的。

关注上游关注生态环境，或许你会有新的思路。

比如我最近已经在推出的，一些圈友已经在和我合作的投资合作服务。

0元租机，合分利益。

或者直接租机，减少大家做项目的损耗，加强大家循环使用二手手机的概念，而不是去买新机。

促进政府的减碳落地。

也是看到了这一点所以和团队人员商讨后，开始推进这一块。

当然租机、售卖手机、回收手机正常进行，毕竟我们的位置就是为各位社群群友提供服务。

完善服务充当好在其中的角色就是。

# 为何二手手机成为大运会最受欢迎“单品”

说这个之前，我给大家说下当初疫情三年，为什么我们公司的二手手机业务出售，不仅没亏钱。

甚至赚得比疫情没来之前多。

首先第一点，以大运会转转供给上万台手机给运动员为例。

那么我告诉大家，在重庆疫情的时候，我们也曾经提供过二手手机给本地做核酸检测手机。

直到2023年疫情结束，这些机子被悉数退回。

而这只是一部分，在疫情期间，因为大家都无聊做项目的自然也多了。

而市场三天两头风控，但是由于我们的工作室离市场是比较远的，所以就重庆疫情最厉害的时候，我还能出去溜达。

每天正常给客户发货，也导致了，那阵子。自己一个人在工作室每天发400多台手机，人都发傻了。

手机这个产品，其实大家更应该把他当做一个工具来看。

一句老话说的好：挖金矿中，谁最赚钱？卖铲子的吧。

所以有时候，我更多的觉得。我不是卖的一个产品，而是在卖铲子在卖工具。

一个和我们生活息息相关密不可分的东西。

# 入行壁垒

那么对于这个，高投产、低利润的行业。他们项目逻辑又如此简单，那么它的壁垒在哪里？

这个答案，或许每个人心中不一样。

但是我觉得。是“时间”。

就比如目前我认为自己比较拿的出手的能力有

1.我有2年以上在天猫某类目排名前10的运营经验

2.我有8年在闲鱼独自自学成才做二手手机的经验

3.我有8年的数码经验

这些经验都是我用时间去积累的壁垒，也是别人打不破的壁垒。

除开这些，就还有这8年来我从1W块成本赚到现在库存的100W，3000+手机的压款了。

但是这些都是时间成本，并且我踩中了闲鱼的红利。

# 给同行的一些心里话

不管你是准备做二手行业，还是已经是二手行业。

对于当今互联网飞速发展，越来越多的人接受使用二手产品，在顿号眼中。

二手行业空间还是非常大的，利用互联网+二手单品其实很多类目可以通用营销模板。

我相信在未来，二手行业会有一些小的单品爆类。那么这些爆类就等你我去探索了。

如果你对二手行业感兴趣也可以和我唠唠。

最后，在给予准备放弃二手产业的同行们说一句话。

很多答案就在眼前，我们不是不知道，而是不敢接触。

可能你现在会觉得利润很低，但是往长远的看，或许你会发现不同的观点。
