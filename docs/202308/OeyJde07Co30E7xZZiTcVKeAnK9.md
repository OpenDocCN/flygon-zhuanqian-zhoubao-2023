# 教程

首先，登录网站：免费的，无需魔法和付费

https://rightbrain.art?inviteCode=PKEUUFMC

![](img/5ebf2159ab099cd3af56882a122eba69.png)

![](img/4c835cf90f73000e8ffbee548834ee58.png)

在这里输入“云天”

![](img/83b98c9f265f2698fae8ae9d50026b11.png)

点击小和尚图片

![](img/ef63154d8e54b39a62f34eee0f5d427f.png)

选择9:16,4张，普通画质就可以了。高清和超清只能做2张图片，效率慢。

我们这里选择普通画质，然后后面我们用高清修复软件变成高清即可。

![](img/1a5838a450440b1fc19cc2c0f663c328.png)

在下面输入我们的描述语

![](img/c8faf3ea9a692d9f7fcd80eaaffec9f4.png)

![](img/be9c593d6249448e21a6b41528a89d10.png)

最后生成了这样四幅图

![](img/b9c6b59a3fdb9261d4869fde28c1286f.png)

我们选择合适的图片下下载之后，利用高清修复软件或者网站把图片变成高清。这里有一个网站

https://arc.tencent.com/zh/ai-demos/faceRestoration
