# 一、项目介绍 

这个项目的原理就是通过二次原创其他平台的视频经过我们的去重玩法上传到小红书平台 

积累到 1000 粉丝，最后通过渠道去接广变现 

品牌方会直接提供文案，博主要做的只是按照要求把笔记上传即可 

也不用咱们自己进行创作，还有一点就是广告类型是一口价，也就是说只要发布了就是那些钱 

不存在看后期数据结算那种情况，有的广告商看你数据好了还会给你发红包 

经过我们的市场调研以及两个月的实操，接商单项目的市场比较蓝海
